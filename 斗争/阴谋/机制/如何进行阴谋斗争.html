# 如何进行阴谋斗争

本文翻译自craneclan的文章running an intrigue一部分，由于阴谋斗争是五轮传奇五版TRPG中最为复杂的斗争形式，规则书介绍的也十分依赖GM的自由发挥，对于没有l5r跑团经验的GM很难落地。因此craneclan提供了一系列如何进行阴谋斗争的实际操作方式。

# 何时需要阴谋斗争

阴谋斗争可以被认为是一种“社交战斗”，就像战斗中有很多对手需要角色去打败，阴谋斗争中有很多目标需要角色去击败。就像战斗中对抗一个强力的敌人是十分少见且很难在游戏中平衡，一个只针对单个困难目标的阴谋斗争也应该是少见的。

总的来看，阴谋斗争的目标是下述一项或多项：

  • 说服一个NPC
  • 了解信息或者研究目标
  • 败坏NPC的名声
  • 阴谋外发生某些事情时，转移NPC的注意

阴谋斗争可以是调查为主也可以是纯社交的。然而，所有阴谋都需要以某种方式被反对，应该有一个派系或者团体阻止玩家正在进行的事情。如果阴谋斗争失败了，也应该有所风险。

例如：KAORI, TOSHIKI, KOTORI, TSOLMON和KYONO知道了一个血语者邪教控制了大名领土边缘的一个村庄。他们需要军队来击败邪教徒，因此他们前往大名的城堡。

GM决定创造一个阴谋斗争来说服大名派出支援。

阴谋可能会遭到已经渗透到朝堂的血语者反对，还有OTOMO MORI的反对，因为他希望在他造访期间，大名可以专心为他提供礼貌和尊严的访问体验。

# 步骤1：每个玩家设立一个目标

阴谋斗争总体上应该被分解为若干目标。每个玩家应该在阴谋开始时先设定一个目标。

一开始，GM应该预先设计好一系列目标，玩家可以每人从中挑选一个。最终，目标可以完全由玩家设定，不过这需要玩家对游戏系统和GM游戏风格十分了解，且有时间去讨论准备阴谋时再使用。

如果有玩家比其他玩家等级显著的低，或者某一个目标十分困难，设定比PC少一两个目标都是没问题的，或者在必要时可以让PC的目标数量翻倍也是可以的。可以相应的修改目标声势。

一旦玩家们在阴谋斗争中非常有经验的合作过，那么就有可能建立一个大型的阴谋任务。这需要专业性来平衡，并且应该留作单独一次遭遇。

目标示例：

  • 说服NPC
  • 败坏NPC名声
  • 阻止NPC干涉
  • 发现一个地方的信息
  • 激怒NPC
  • 强迫NPC自取其辱
  • 威吓NPC
  • 召集人群
  • 赞美某人（包括你自己）
  • 强迫NPC承认一个秘密
  • 说服一群人
  • 一群人中找到关键人士
  • 发现NPC的相关信息
  • 贬低某人
  • 在环境中散布秘密

例如：GM决定两个目标应当为：1、说服大名在村庄中存在着血语者威胁；2、确认可能回去报信的朝堂中的血语者间谍。

如果必要的话，多个目标可以对抗同一个人。多个玩家也可以有相类似的目标。

例如：GM决定增加一个额外目标：3、说服大名在帝国军团到达他的领土前迅速行动。

目标可以用来修正其他目标的难度。一个主要目标可能有很高的声势要求来完成这个目标，不过如果第二目标先完成了，主要目标的声势要求会降低。

例如：GM决定OTOMO MORI会尽可能让大名分心，这让玩家更难说服大名。因此他增加了额外目标：4、其他玩家辩论时，阻止OTOMO MORI让大名分心。

如果玩家愿意，玩家可以选择来发展他们自己的目标来代替。

例如：TSOLMON决定利用这次机会败坏他的私人敌人，SHIKA ODA，他恰好是大名朝堂的一员。在GM的允许下，他添加了如下目标：

5、羞辱SHIKA ODA

如果玩家希望在阴谋斗争中贬低或者反对其他玩家，或者设置一个隐藏的目标，他可以设置隐藏目标，特别是涉及PVP的情况下。

例如：KYONO决定赢得OTOMO MORI的个人喜爱作为个人目标，不过他不想让其他玩家知道这一点。在GM的允许下，他添加了如下目标：

6、给OTOMO MORI留下深刻印象

阴谋不需要被特定的NPC反对，不过这是可行的。这些NPC也应该拥有自己的目标。

例如：GM决定，在这次冲突中，玩家会被SHIKA ODA和血语者间谍反对，他设定了如下目标：

SHIKA ODA：贬低TSOLMON

间谍：不被抓到的情况下，贬低或者反制TOSHIKI 或者KAORI，或者逃离朝堂

# 步骤2：决定阴谋在何种情况下结束

GM必须决定阴谋斗争何时结束。这将决定PC有多少时间来达成自己的目标。阴谋斗争可以有多种结束方式，先达到的结束方式将会终结整个阴谋斗争。

可能的结束方式：

  • 主要目标被达成（通常是最高声势点的目标）。
  • 在若干轮后。
  • 当场景的的一个角色脱面
  • 当对立角色达成他们的目标

例如：GM决定这次阴谋将会在以下两个目标中任意一个达到后结束：

  1. 五个回合后
  2. 目标1和2均完成后一回合
  3. 场景内的角色脱面后一回合

# 步骤3：决定如果没有达到目标会发生什么

有些目标有着非常严重的后果，有些则可能没有那么严重的后果。对于每个目标，GM都应该决定目标没完成会发生什么。目标也可以修正其他目标需要达成的声势点，及时完成一个目标可以让一个困难的目标简单些。

后果示例：

  • 后续增加对手的人数
  • 提高声势点
  • 失去突然性
  • 某个敌人逃脱
  • 错过线索
  • 失去载誉值
  • 失去荣耀值
  • 遭受债务
  • 失去某种资源
  • 获得额外矛盾
  • 暴露信息给你的对手
  • 一个负面的传闻

例如：GM决定了如下失败后果：

  • 说服大名认可血语者威胁：主要目标，阴谋斗争失败。玩家只能独自面对血语者的威胁或者找到其他盟友。
  • 确认朝堂中的血语者间谍：在玩家到达之前血语者领导者已经逃离
  • 说服大名迅速行动：对村庄总攻时，村庄的防御已经准备完毕。
  • 阻止OTOMO MORI：增加目标1声势点4点。
  • 羞辱SHIKA ODA：与SHIKA ODA对抗，没有效果，除非SHIKA ODA达成他的目的。
  • 给OTOMO MORI留下深刻印象：无

# 步骤4：决定目标的声势值

根据经验，目标声势值就是达到目标的专注值。这个值可以被修正来让难度调整。

  • 非常难：+4声势
  • 难：+2声势
  • 简单：-1声势
  • 非常简单：-2声势

达成声望的行动难度取决于玩家选择的行动类型。一般来看，目标的警戒值是社交行动检定的难度。这个难度可以使用目标的风度来修正。对于以一群人为目标，或者隐藏的信息，可以使用隐藏着的角色的警戒值，或者GM来设置一个合适的检定难度

例如：GM设置了如下声势值：

  • 说服大名认可血语者威胁：大名的专注值+6（基于MORI的干涉）
  • 确认朝堂中的血语者间谍：间谍的专注+2
  • 说服大名迅速行动：大名的专注值
  • 阻止OTOMO MORI：MORI的专注值
  • 羞辱SHIKA ODA：与SHIKA ODA对抗，大致等于ODA的镇定值
  • 给OTOMO MORI留下深刻印象：MORI的专注值

GM可以为玩家的对手目标设定声势值，或者简单的让他们来降低玩家的声势值。

例如：

  • SHIKA ODA：羞辱TSOLMON：TSOLMON两倍的荣耀值
  • 间谍：从朝堂上五个回合内离开。

# 步骤5：让玩家选择目标

目标不是被分配的，每个玩家必须基于自己的修辞和其他能力选择目标中的一个。

一个玩家应该只选择一个目标，除非玩家已经选择了一个秘密目标且希望表现为选择一个非秘密目标。如果这种情况出现了，那么选择的非秘密目标被列在那里但不会被推进。

例如：从目标列表中：

  • TOSHIKI选择1：说服大名认可血语者威胁
  • KAORI选择4：说服大名迅速行动
  • KOTORI选择2：确认朝堂中的血语者间谍
  • TSOLMON选择5：羞辱SHIKA ODA
  • KYONO表面上选择3：阻止OTOMO MORI，但实际上他选择自动失败这个目标转而秘密的选择6：给OTOMO MORI留下深刻印象

注意：在冲突斗争中，每轮结束时，玩家可以选择在某一目标完成前放弃它，或者与其他玩家交易目标。如果一个目标被放弃了，那么它将会被标记丢失，除非另一个玩家重新选择了它。如果一个目标丢失了，那么其失败的后果将在阴谋斗争结束时生效。

# 步骤6：开始阴谋斗争

一旦每个玩家完成了目标的选择，那么掷主动性来开始阴谋斗争。

修辞可以引导玩家在阴谋斗争中以更好的方式完成目标，或者允许角色在朝着主要目标努力的同时执行次要目标或者额外任务。这可以让玩家赢得声势或者得到可能降低声势目标的方法建议。

阴谋斗争中获得的机会可以为同一场景内为其他目标努力的角色提供增益。

例如：TSOLMON拥有修辞：捕食弱者。玩家决定他们准备通过迫使ODA在公众前脱面来羞辱ODA。他们会使用其他修辞，例如煽风点火，来增加ODA的冲突直到ODA妥协。

SHIKA ODA拥有修辞：朝堂之道（怀疑是写错了，目前没有这个修辞，可能是朝堂低语，rank1风属性修辞）。GM决定使用它来在人群中羞辱TSOLMON。如果他成功，GM将会给予风+额外成功数的声势来完成羞辱TSOLMON的目标。

# 步骤7：完成目标

一旦玩家成功完成了目标，他们可以选择协助其他玩家完成他们的目标，只要斗争冲突尚未结束。这允许技巧精湛的朝臣玩家协助没有那么擅长阴谋的玩家完成目标，或者允许幸运的玩家在完成他们自己的目标后协助完成最复杂目标。这也允许被放弃或者秘密目标被恢复。

例如：阴谋正在顺利的执行。

  • TOSHIKI对于达到了主要目标有些难度（1）说服大名认可血语者的威胁），因为声势值被OTOMO MORI的影响而+4。
  • KAORI完成了她的目标（2）说服大名快速行动）。她的下个回合，她选择协助TOSHIKI推进目标1）说服大名认可血语者的威胁。
  • KOTORI尚未发现朝堂上的间谍，不过她在使用她对于文化的知识以及共情来审视朝堂的出席者，来判断来者是否有人举止可疑。
  • TSOLMON让ODA成功妥协，并且使用捕食弱者来让KAORI和TOSHIKI有更多的时间来说服大名。他正在努力阻止ODA散布谣言。
  • KYONO完成了他的目标（6）给OTOMO MORI留下深刻印象），然后发现TOSHIKI在处理他的目标有难度，因此他选择推进目标（4）阻止OTOMO MORI），希望他可以降低TOSHIKI面对的难度。

当阴谋的结束标准完成时，阴谋将会以一个合适的方式结束：朝堂将以胜利或者尴尬结束。接下来将会发挥和总结目标失败的后果。

例如：KYONO利用GM基于之前成功的目标（利用他与OTOMO MORI建立了新的联系）给予的额外成功，也成功的完成了阻止OTOMO MORI的目标。

这让说服大名的难度降低了4。

在KAORI的帮助下，TOSHIKI可以完成说服大名目标。

TSOLMON成功利用了捕食弱者，不过ODA没有成功脱面。他们失去了荣耀并快速离开了场景。

因为ODA在他离开前完成了谣言的散播，他们对TSOLMON的口碑造成了一定的伤害。

KOTORI并没有在间谍离开前发现间谍。当玩家们和大名的军队到达血语者的村庄时，他们会发现血语者领袖已经在玩家到来前离开了，他会把灾难散播到其他地方，他们会在其他地方等着玩家改天再战。
