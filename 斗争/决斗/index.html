# 决斗

# 妥协

每个回合开始时，参与决斗的一个角色可以选择认输，荣誉的退出竞赛。绿岩的法律规定那些认输的人输掉了全部的战斗意图和目的，并且需要履行他们失败的条件——如果决斗前约定好的话，决斗者失去自己的性命也可以是条件。

然而，法律规定允许胜利者以武士道仁的名义放弃部分或者全部败者的条件：例如一名战士可以要求被他击败的敌人砍掉他的顶髻而不是砍掉他的头，这意味着他必须在修道院隐退，但可以保住自己的性命。饶恕敌人的性命或者其他让步是明智的，让决斗者可以保持社会和谐而不会显得自己软弱。一名认为自己无法获胜的决斗者可能会选择投降来保全自己的性命，而不会让自己的家族陷入报复的循环。

# 决斗中死亡的礼节

杀死对手是某些决斗的目标，对于另一些决斗来说完全不可接受，主要基于实际情况以及事先约定。

对于第一击和无法行动的决斗，杀死对手最好的情况是一种可悲的意外，最坏的情况可能被视作谋杀。

对于第一滴血的决斗，杀死对手是需要解释的，不过不属于严格不荣誉的情况。

显然的，杀死对手是致死决斗的目标。放过对方是极大的无理，除非执行的十分巧妙。

# 常见的决斗形式

仅仅达到决斗目标并不是总可以取得胜利；两个角色可能同一回合达成目标，或者一个角色达成目标但成功的一击被裁判取消资格。这就是仪式化战斗的变化莫测，因为没有一个凡人裁判可以在刀剑的碰撞中完美的感知到每一击。

  • 居合决斗（第一击或者第一滴血）：居合决斗是高度仪式化的，决斗者穿着仪式服装，只有武士刀和胁差允许使用，每名决斗者都需要在开始是将剑收入鞘中，不早于第一回合将其拔出剑鞘。当目标时第一击时，决斗目标是给对方施加致命一击。当目标为第一滴血时，决斗目标是给对手造成程度5或以上的致命一击。

  • 居合决斗（致死）：致死决斗时，决斗目标是杀死对方或者给对方施加濒死状态。

  • 陪练（无法行动）：练习决斗可以让战士们磨练他们的技巧，增长他们的老茧让他们可以在战场上表现更好。武者和和尚经常利用这种决斗练习（和尚更喜欢徒手格斗）。战士喜欢使用训练武器或者钝器进行这种决斗，决斗目标是给对手施加无法行动状态。

  • 战士的决斗（击败）：许多希望证明自己剑技的浪人们喜欢更粗鲁的决斗，以期望某个领主看到自己的技巧从而登用自己。战士的决斗使用任何手头的武器进行的，几乎没有规定。决斗的目标是给对手造成无意识状态、杀死对手、造成濒死状态、或者让对手屈服。

  • 他流试合（无法行动）： 修验者是稀少且有价值的，因此他们很少被允许进行刀剑决斗。然而他们是一群拥有强大力量的骄傲的人们，他们的争吵就能撼动大地。他流试合决斗主要以祈愿为主。决斗中的物理打击被某些氏族超自然领袖允许，例如巨蟹氏族或者毒蝎氏族，在另一些氏族则被拒绝，例如凤凰和仙鹤氏族。决斗目标是给对手造成无法行动状态，或者展示自己的绝对优势来让对手投降。

# 决斗中的特殊规则

# 主动性

决斗中的主动性检定是难度为1的冥想检定。

每个角色的主动值基于他们主动性检定的结果。不过每轮中，在阶段2进行回合时，角色可以参加凝视来调整主动性。

凝视：

随着决斗中紧张氛围的加剧，即便最为坚韧的武士也会感到压力。战士们的感官被提升了，即便碎石的嘎嘎声或者蝉鸣都变得如同雷鸣一般。这就是凝视，每轮阶段2进行回合开始时的特殊步骤。

在凝视中，每个角色将会受到等于轮数的冲突（第一轮获得1，第二轮获得2，第三轮获得3，以此类推）。

此外，在受到冲突后，每个没有妥协的角色都可以出价额外的冲突来增加他的主动性。出价过程中，每位参与角色秘密的选择0到专注值中的一个数值。然后，参与出价的角色同时展示他们的数值，并承受如此数量的冲突，自己的主动性增加如此数值。

# 执行回合

轮：

每个回合，每个角色按主动性顺序执行回合。高主动性角色先执行回合，低主动性的角色次之。对于主动性打平的情况，低载誉值的角色在双方主动性一致的情况下先行动。在本轮结束时，每个角色主动性恢复到凝视前的数值。

一场决斗没有固定的持续轮数。每轮结束时，要不然某个角色达成了决斗目标，那么决斗结束并执行阶段3结算。否则会到阶段2执行回合并继续执行一个新轮。

终结一击：

随着决斗人员开始行动，他们的防御可能会有缺陷，让他们的敌人可以有机会以一个良好的攻击结束战斗。

当你的对手在决斗中第一次妥协或者脱面，你可以立刻执行终结一击。这是个特殊行动，可以在你的回合外执行。

执行终结一击的角色选择一个攻击行动，中断目标的任何行为（包括结算他们自己的检定，因此终结一击可以在检定的步骤6触发，在【冲突】结算后【机会】结算前），一个终结一击甚至可以中断另一个终结一击。

如果角色执行终结一击成功，代替造成伤害，终结一击造成一次致命一击，程度等于两倍武器致命度或者攻击使用的致命度，加上攻击的额外成功数。任何其他攻击效果都如常结算。

如果两个角色同时出发终结一击（例如在凝视中强制获得冲突），低载誉值的角色先完整的执行终结一击，接下来另一个角色执行终结一击（如果他还活着）。如果决斗目标通过终结一击达到，决斗立刻结束。否则，决斗将在其中断的地方继续进行（例如继续结算被中断的检定）

架势：

决斗中，架势反映了角色的物理姿势和决斗整体策略。

玩家被鼓励描述他的角色如何使用指定架势，例如风架势下灵活佯攻，或者空架势下一动不动的等待。

移动和距离：

机制上讲，决斗中处理距离十分简单：开始时，角色之间的距离为2。决斗中，每个角色都被视为在任何对方效果的范围内。决斗结束时，两个角色再一次会到距离2。

玩家可以自由描述角色的移动方式，通过换位、佯攻和出乎意料的侧滑。

# 决斗行动

  • 平静呼吸

描述：斗争期间，你可以在呼气前深吸一口气，从而汲取你的内在力量

激活：作为一个支援行动，你深吸一口气平静自己恢复耐力

效果：如果你的冲突值大于你镇定值的一半，你移除一点冲突。如果你的疲乏值大于等于你耐力值的一半，你移除一点疲乏。

  • 集中

描述：你让虚空接管你的感官来让你周边的世界慢了下来。除了那一刻，一切都随着你本能的寻求胜利而消失。你的思绪穿过了永恒的危险可能，以及你的对手的锋利的钢铁，达到你想要的结果。

激活：作为一个空架势的支援行动，你可以把能量集中在内在，在脑海内设想你的行动并寻求完美的时机来接受它。你使用集中的时候必须声明一个技能。

效果：扔至多等于你选择的技能等级数量的【技能骰】，并可以保留任意数量的骰子。如果你如此做，你在本场景内下一次执行选择的技能检定或者使用集中行动，在掷骰后，可以使用任意数量的保留骰替换掷骰结果。你不能保留超过你目标技能等级的骰子数量。

  • 预测

描述：你巧妙的移动，通过换位或者发出没有发生过的打击的声音来吸引对手注意。通过预测敌人的反应，你目标通过思绪来获得胜利，迫使对手承受决定性打击或者提前攻击。

激活：作为一个攻击和密谋行动，你可以秘密的选择风、土、水、火并记录下来。

效果：下一次你的对手选择架势时，你可以展示你的选择，如果和对手选择的架势相同，你的对手承受4点冲突，并且必须选择一个不同的架势。这个效果持续到你的下个回合结束。

  • 准备物品

描述：你准备、装填一个物品或者武器。

激活：作为一个支援行动，你可以使用此行动。

效果：准备一个物品来使用，将武器准备在一只你选择的持握手中，或者装填一个物品

  • 打击

描述：你对单一对手进行一次攻击

激活：作为一个使用一把准备好武器的攻击行动，你可以执行一次难度为2的武术检定，使用对应武器的合适技能，目标一个武器距离内的单一角色。

效果：如果你成功，你对目标造成等于武器的基础伤害加上你的额外成功。

新机会效果：

【机会】【机会】：如果你成功，你可以对你的目标造成一次致命一击，程度为你武器的致命度。

  • 独特行动

描述：你可以执行一次机制上或者叙事上检定（章节3：技能）

激活：作为一个行动，你执行一个满足GM叙述要求的技能检定。

效果：如果成功，你可以执行其叙事效果，执行其符合示例的单个行动效果，或者执行GM认为合适的其他任务。

# 结算决斗

任一轮结束时，决斗中任一角色达成决斗目标则决斗结束。

部分决斗的结果十分明显，一名武士死亡另一名武士站立，剑上染了红色的鲜血，大部分决斗都更加有歧义且混乱。决斗结果最终落入一个或者更多裁判的判罚，来决定打击的合法性，谁先被击中，以及打击的有效性。

在阶段3结算时，每个参与者根据下表上描述的行为计算分数。分数差的大小决定了决斗结果：

  • 0分数差——没有胜者：决斗是平局，可能是双方同时击中对方（也可能不是）。角色可能再次决斗，或者退下照顾伤口。
  • 1分数差——勉强胜利：胜利者抓住了微弱的胜利优势。
  • 2分数差——明确胜利：胜利者毫无疑问的取得了胜利。
  • 3-6分数差——绝对胜利：胜利者清楚的展示了对决斗的掌握。如果胜利者受到荣耀奖励，则奖励值+3.
  • 7+分数差——没有竞赛：胜利者与其说是获胜，不如说这从来就不是一场竞赛。如果胜利者受到荣耀奖励，则奖励值+5。
