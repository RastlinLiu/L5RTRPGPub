# 通常机会示例

# 任意机会

  • 【机会】：如果你（此次检定）失败了，确定你尝试的检定最容易的通过方式（包括轮和技能）
  • 【机会】+：每花费一点机会，则移除1点此检定中获得的冲突
  • 【机会】【机会】：为下一个尝试类似检定的角色提供援助

# 风机会

  • 【机会】：了解本场景内的另一个角色的举止（如果他是NPC）以及当前的冲突值
  • 【机会】+：巧妙的行动以吸引尽可能小的注意力，花费更多的【机会】会让行动更加巧妙。
  • 【机会】【机会】：注意到场景内角色的有趣细节，例如优势或者劣势。在GM的决策下，你可以为场景内NPC添加新的细节。

# 土机会

  • 【机会】：通过你的存在让其他角色安心，允许他移除2点冲突
  • 【机会】+：谨慎的行事以尽可能降低任务可能失败或者其他情况产生的危险，花费更多【机会】可以让尝试更加安全。
  • 【机会】【机会】：突然想起一件不直接相关于任务的重要信息。在GM的决策下，你可以建立一个小型的准备行动，例如带来了一件普通的有用物品。

# 火机会

  • 【机会】：通过你的存在激怒其他角色，让他获取2点冲突
  • 【机会】+：用华丽的方式执行任务，吸引他人的注意，花费更多的机会会让你更加引人注目
  • 【机会】【机会】：注意到周围不直接相关于任务的事物的缺失或者不在合适的位置。在GM的决策下，你可以建立一个缺席，例如在屋外没有鞋暗示了屋内没有别人。

# 水机会

  • 【机会】：移除自己的2点冲突
  • 【机会】+：用高效的方式执行任务，更快速的完成任务或者任务完成时还有补给剩余，花费更多的机会可以让完成任务的时间更短或者存留下更多补给。
  • 【机会】【机会】：发现周围环境一个和任务不直接相关的有趣物理细节。在GM的决策下，你可以建立一片地形或者一个周围的普通物体

# 空机会

  • 【机会】：选择一个空轮以外的轮，你的下一次检定如果使用该轮，则该检定难度减一
  • 【机会】+：如果场景内存在超自然干扰，那么你可以通过后脊发凉、突然的沉默等现象发现，花费更多的机会可以让你更加精准的定位超自然干扰的位置
  • 【机会】【机会】：获得对宇宙本质或者自己内心的超自然洞察力。在GM的决策下，你可以建立一个和这个情景相关但是此前未被揭示的事实。

# 战斗和武术机会示例

# 风机会

  • 【机会】：为你的下次武术检定增加一个保留的【轮】【机会】骰
  • 【机会】+：移动行动检定中，你每以此法花费一个机会，就可以将一格距离调整为垂直方向（既在移动3的情况下，花费2个【机会】可以变成垂直移动2+水平移动1）
  • 【机会】【机会】：你下一个回合开始前，提升以你为目标的武术【远程】检定难度2点

# 土机会

  • 【机会】：在移动行动中，忽略一个你选择的地形特质
  • 【机会】+：你的下一个回合开始前，你每以此法花费一个机会，降低你下一次承受致命一击程度1点。
  • 【机会】【机会】：直到你下个回合结束，你检定时可以忽略一个你的劣势

# 火机会

  • 【机会】：选择场景内的另一名角色，在你的下个回合开始前，如果他的检定没有选择你作为目标（之一），则增加他们的检定难度1点
  • 【机会】+：攻击检定中，你每以此法花费一个机会，则让目标在你的下个回合开始前的下一次抵抗致命一击坚定的难度加一
  • 【机会】【机会】：直到你的下个回合开始，其他角色选择你作为攻击或密谋行动的目标（之一）时，必须承受两点冲突

# 水机会

  • 【机会】：移除1点疲乏
  • 【机会】+：攻击检定中，你每以此法花费一个机会，则可以忽略1点目标的物理抵抗
  • 【机会】【机会】：移动距离1

# 空机会

  • 【机会】：你下个回合结束前的下一次攻击行动检定中，忽略一个你选择的地形特质
  • 【机会】+：支援检定中，你每以此法花费一个机会，则你的主动值增加1
  • 【机会】【机会】：你的下个回合结束前，忽略一个你现在遭受的状态

# 主动性检定机会示例

# 风机会

  • 【机会】：主动性检定中，获取一个敌手的弱点。了解该敌手的一个劣势

# 土机会

  • 【机会】：主动性检定中，选择另一个角色的你知道的劣势，该角色在主动性检定中不应用此劣势

# 火机会

  • 【机会】：主动性检定中，当你被意外袭击的时候，使用你的专注代替你的警戒用来进行主动性检定

# 水机会

  • 【机会】：主动性检定中，获取场景内的全部地形特质

# 空机会：

  • 【机会】：主动性检定中，感受到场景内是否存在异界生物。

# 其他技能机会示例

# 风机会

  • 工匠【机会】：如果你成功，为你正在翻新的一件物品增加华丽或者灵活特质
  • 学识【机会】：了解你正在研究的物品的制造者或者使用者的一些情况（例如GM可以选择一个他的优势或者劣势，其相关于这件物品）
  • 社交【机会】：了解场景内的一名角色其载誉、荣耀以及地位属性与自己的关系（大于、小于或者等于）
  • 贸易【机会】：说服一名买者为你正在售卖的物品支付额外10%的价格

# 土机会

  • 工匠【机会】：如果你成功，为你正在修复的一件物品增加耐用特质
  • 学识【机会】：记起一个你可以研究或者温习你试图回忆起的内容的地点
  • 社交【机会】：选择一个其他角色，直到本场景结束前，他的下一个社交检定难度加一
  • 贸易【机会】：选择一个其他角色，直到本场景结束前，他的下一个同种检定难度减一

# 火机会

  • 工匠【机会】：如果你成功，为你正在制作的物品增加一个拷贝
  • 学识【机会】：推断场景内或更广阔情况下某个角色的动机或愿望
  • 社交【机会】：选择一个其他角色，直到本场景结束前，他的下一个社交检定难度减一
  • 贸易【机会】：不寻常的灵感来袭，为你另一个技能的检定增加一个保留的【轮】【机会】骰

# 水机会

  • 工匠【机会】：为你下一个工匠技能的检定增加一个保留的【轮】【机会】骰，直到本次游戏流程结束前
  • 学识【机会】：发现你正在识别的事物的独特之处、可识别的品质、风格和能力
  • 社交【机会】：为你下一个社交技能的检定增加一个保留的【轮】【机会】骰，直到本场景结束前
  • 贸易【机会】：说服一名卖者为你正在购买的物品额外打10%的折扣

# 空机会

  • 工匠【机会】：降低你使用正在自省的物品进行的下一次检定的难度1点
  • 学识【机会】：通过直觉判断你目前探究的过程中是否可以了解到任何有价值的东西
  • 社交【机会】：辨识场景内另一个角色的目标
  • 贸易【机会】：降低你对环境（以及你造成的物理痕迹）到最小

# 休整活动机会示例

# 风机会

  • 【机会】+：每以此法花费一个机会，了解一个你的伙伴的情况（例如他选择的优势或劣势）。每个单独的休整场景只能了解每个伙伴各一个情况
  • 【机会】【机会】：不让一个或更多角色发觉你执行了你的休整活动

# 土机会

  • 【机会】+：你每以此法花费一个机会，你团队内的其他角色可以移除一点冲突或者疲乏
  • 【机会】【机会】：从你的行动中回忆起微小但重要的细节，你后续可以不通过检定回忆起这个记忆

# 火机会

  • 【机会】+：你每以此法花费一个机会，你可以在你团队中一个角色的下一个休整活动检定中进行支援
  • 【机会】【机会】：通过行动激励你团队内的其他人，他可以额外执行一个休整活动（最多两个）

# 水机会

  • 【机会】+：你每以此法花费一个机会，你可以移除一点冲突或者疲乏
  • 【机会】【机会】：在你的休整活动中结交一个新朋友

# 空机会

  • 【机会】+：你每以此法花费一个机会，保留一个被弃掉的骰子，至多为你做检定的技能等级数量。将保留的骰子作为保留骰子加入下一次同技能的检定中
  • 【机会】【机会】：在休整活动过程中，对未来可能发生的事情产生一个简单的预期
