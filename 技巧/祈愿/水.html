# 水祈愿

水祈愿可以给予修验者速度和快速反射，同样也提供了治愈伤害和平静发热头脑的能力

# core

# 水之钵——rank1

毁坏的船桨（先祖）惠比寿的拐杖（福神）术士之杖（外国）

激活：作为一个支援行动，你可以执行一次难度为1的宗教（水）检定，来从附近的池塘、溪流或者空气中的蒸汽召唤出一把杖

效果：如果你成功，你可以召唤一把水做的杖。它拥有罗网特性。每轮结束时，你可以把该武器转化为另一把不同的近战武器。武器持续到它离开了你的掌握，或者场景结束，那时它将飞溅到周围的池塘。

新机会效果：

水【机会】：你可以召唤任意的远程或近战武器

水【机会】：你可以立刻以你召唤的武器进行一次打击行动

水【机会】：当你以此武器进行攻击行动时，最大或最小射程视为增加或减少1

# 四季之舞——rank2

季节变化之盐（炼金）克里科的真诚祝愿（先祖）旋转涡芽之轮（福神）

激活：作为一个移动和支援行动，你可以执行一次难度为3的宗教（水）检定，目标为一个距离1-3的地点，其中包含水源（例如池塘，河流，雪，冰，甚至于雾）

效果：如果你成功，你可以召唤出季节的精髓来冰冻、解冻或者蒸发任意数量位于0-1的水。如果你将水变为冰，那么它变成固体，地点变为危险地形。如果你将水变成蒸汽，那么它变成雾，地点变成模糊地形。生物之内的水不会受到影响，因为神灵不希望通过荒诞的行为毁了精致的场景。

新机会效果：

水【机会】+：你每以此法花费一个机会，就可以多选择一个额外地点。

水【机会】+：你每以此法花费一个机会，就可以选择一个在目标地点内的角色，使其被冰包裹住。每个被选择的角色都必须通过难度为3的体能检定（土1，火4）抵抗，否则将会承受定身和迷惑效果。

# 水灵领域——rank1

井内的窗户（先祖）众河终将归一（元素）花园池塘的间谍（外国）

激活：作为一个移动和密谋行动，你可以执行一次难度为2的宗教（水）检定，目标为距离0-1的一个含有水体的地点

效果：如果你成功，你在涟漪中占卜。目标水体将会展示你选择第二片水体及其周围的景象，就好像观察者在水体之下一样。第二篇水体必须在目标地点距离0-6内，且你必须对特定的水体有具体的了解才能如此使用。

新机会效果：

水【机会】：除去你能看见其他水体的信息，你还能听见周围的噪声（非常沉闷的声音）。

水【机会】：只有你能看到景象，其他人在水体上无法看到任何内容

水【机会】：你看见的景象可以是距离你任意远的地点的水体（但你依然需要对其熟悉）

水【机会】【机会】：你可以使用云，浓雾，雪或者冰面来作为此技巧的水体

水【机会】【机会】【机会】：除去你可以看见其他水体的信息，你和至多等于你水轮数值的角色可以穿过这个临时的传送门到达另外的地点。使用此技巧的旅途永远是单向的，而且你们到达时一定是湿透的

# 持续变化的波浪——rank5

变形灵药（炼金）优雅的祝福（福神）无数形态（外国）

激活：作为一个支援行动，你可以执行一次难度为4的宗教（水）检定，目标为自身。

效果：如果你成功，你强化你自身，将自己变形为一个体型为2的自然动物。在此形态期间，你使用该形态的轮值，衍生属性，能力，以及体力或社交类型的优势，但使用你自己的技能等级，以及精神及超自然类型优势。这个效果持续到你选择结束，或者你获得无意识状态为止，此时你会转变为你的原有形态。在你变回原状时，不要移除你在之前形态上获得的疲乏或者冲突；如果这导致你的疲乏值超过了你的耐力值（或者你的冲突超过了你的妥协值）但你在之前形态上没有超过，你承受一次程度为5的致命一击（对于冲突的情况，你立刻变成妥协状态）

新机会效果：

水【机会】：如果你成功，你可以把你的身体变成不同的有生命物种的身体。

水【机会】：如果你成功，你可以把自己的身体转换成异界存在，例如超自然野兽

水【机会】+：你每以此法花费一个机会，你可以选择变身成体型大1或者小1的存在（至少为体型0至多为体型6）

# 潮汐之手——rank3

命脉踪迹（炼金）生驹与阿古渡的跌宕起伏（先祖）水灵的命令（福神）

激活：作为一个移动行动，你可以执行一次难度为5的宗教（水）检定，目标为你自己及至多等于你水轮数值的距离1-4的其他角色。

效果：如果你成功，你召唤目标们。你可以让这些目标互换位置。注意到最后，每个目标的初始位置都必须被一个目标占据，而且每个目标都必须占据一个位置。如果你成功，每个目标获得2点冲突。

新机会效果：

水【机会】+：如果你成功，你每以此法花费一个机会，你可以为一个目标选择一个新的架势

# 水龙之心——rank2

蒸汽的善意（炼金）寿老人身边行走（福神）抵抗标记（外国）

激活：作为一个激活行动，你可以执行一次难度为3的宗教（水）检定，目标为距离0-2的一个角色。

效果：如果你成功，你可以通过水的灵性流动来增强目标。目标获得等于你水轮数值的超自然抗性，并且在每轮结束时移除一点疲乏值。这个效果持续一轮，以及每个额外成功额外一轮。每个目标在此场景结束前都不能再被水龙之心祈愿影响。

新机会效果：

水【机会】【机会】：当目标成功通过检定抵抗气功、邪术以及祈愿时，使用被抵抗技巧的角色必须执行同样的检定来抵抗效果，否则其自身将会遭受效果。

# 稻荷的祝福——rank1

宫的礼物（先祖）水神灵的盛宴（元素）沙漠中的绿洲（外国）

激活：作为一个支援行动，你可以执行一次难度为2的宗教（水）检定，目标为距离为1的一个地点

效果：如果你成功，你在目标地点召唤出一顿饭，可以喂饱至多等于你水轮数值+额外成功的人数。

新机会效果：

水【机会】：这顿饭被特别的强化了。任何一个在休息时间内分享了这顿饭的角色会移除等于你水轮数值的疲乏值

水【机会】+：你每以此法花费一个机会，这顿饭就可以额外喂饱至多你水轮数值+额外成功的人数

# 通往内心平静之路——rank1

寿老人之吻（炼金）超验朝子（先祖）温泉之触（外国）

激活：作为一个支援行动，你可以执行一次难度为2的宗教（水）检定，目标你自己或者另一个距离0-2的距离

效果：如果你成功，你弥补你的目标来舒缓肌肉。目标移除等于你水轮+额外成功数值的疲乏值。每个目标都不能在此场景结束前再次受到通往内心平静之路祈愿的效果。

新机会效果：

水【机会】：选择如下一个状态：流血、眩目、迷惑、或者轻伤状态，移除每个目标的该状态。

水【机会】：你的目标同样移除等于你额外成功数的冲突。

# 潘库的回响——rank1

阿贺娑的浩瀚智慧（先祖）水神灵的谣言（元素）钢的低语（外国）

激活：作为一个密谋行动，你可以执行一次难度为2的宗教（水）检定，目标一个距离0-1的物品。

效果：如果你成功，你可以占卜来探查该物体具有的全部属性，包括它的名字、制造者、世俗属性、魔法属性以及被封印的技巧。（参见未眠之物）

新机会效果：

水【机会】：获得物品起源的大致知识，例如在哪里铸造的，携带最长时间的家族，或者类似的信息。

水【机会】：你可以目标一个角色而非一个物品。如果你成功，你可以鉴定一个该人物拥有的祈愿、气功、邪术、超自然能力或者物品，包括它的名字以及常规特质。

# 水，崛起

酒的祈愿（炼金）水之守护（先祖）伊索拉的侍从（福神）

激活：作为一个支援行动，你可以执行一次难度为6的宗教（水）检定，目标为一个距离0-3的地点，其中含有大量的水。

效果：如果你成功，你在召唤一个显灵水元素来为你而战。它会出现在目标水体上并听从你的命令两轮，以及每个额外成功额外一轮，它在你每轮结束后立刻执行它的回合。当此效果结束，它会四分五裂，或者如果它被激发了特别的愤怒，那么它会开始遵循自己的指令。如果它开始遵循自己的指令行动，它除非被召唤的修验者激怒，否则不会攻击召唤的修验者。

新机会效果：

水【机会】：显灵水元素试图吞噬它其中的任何人；每个这样的角色必须通过难度为4的体能检定（土2，水5）抵抗，否则将会开始窒息。作为一个行动，抵抗失败的角色可以进行一次难度为3的体能检定（土1，水4）来逃离水牢。

水【机会】：在水元素到达时，水元素可以移动至多等于你水轮数值的距离。

水【机会】+：你每以此法花费一个机会，可以让显灵水元素分裂为一个更小的显灵水元素。每个显灵水元素的耐力值等于12除以以此法花费的机会数（向上取整）

# 踏浪而行——rank2

软木特性的灵药（炼金）破坏之路（先祖）流水花园行走（福神）

激活：作为一次支援行动，你可以执行一次难度为2的宗教（水）检定，目标为一个距离0-2的角色

效果：如果你成功，你召唤一阵浮力流来扫荡目标周边。即便他们不能游泳，目标也能控制自己在水上或者水中自由移动。目标在水中会忽略任何地形的负面效果，并且不会在水下窒息。这个效果持续到此场景结束。

# 海啸打击——rank3

瓶中的海洋风暴（炼金）水天之箭（福神）召唤波涛（外国）

激活：作为一个攻击行动，你可以执行一次难度为4的宗教（水）检定，目标为至多等于你水轮数值的距离你0-3的角色。你必须有水源来施展此技巧。

效果：如果你成功，你通过浪花打击你的目标。每个目标承受等于你水轮数值+额外成功数值的超自然伤害。每个目标都必须通过难度为3的体能检定（土1，火4）抵抗，你可以移动每个抵抗失败的角色等于他失败差额的距离（可以向任意方向）

新机会效果：

水【机会】：每个抵抗失败的角色也要承受流血状态。

# 水灵的拥抱——rank4

水滴中的海洋（炼金）追寻道寺的脚步（先祖）水神灵之吻（元素）

激活：作为一个攻击行动，你可以执行一次难度为4的宗教（水）检定，目标为一个1-3的角色。

效果：如果你成功，你用海水填满目标肺部来打击目标。目标承受等于你水轮+额外成功的两倍的超自然伤害。如果目标的疲乏因此而超过他的耐力值，目标会开始窒息，持续时间为等于你水轮+额外成功的轮数，或者通过一个支援行动来放过目标

# 交感能量——rank2

新条的同情（先祖）慷慨的水灵（福神）沙漠花粉（外国）

激活：作为一个支援行动，你可以执行一次难度为2的宗教（水）检定来扩散一个祈愿的持续效果，目标为至多等于你水轮数值个数的距离0-1的角色。

效果：如果你成功，你可以通过已经在一个目标上生效的一个超自然力量或者祈愿的持续效果强化。每个其他目标都被视为也被此效果影响。这个效果持续到原有效果在原有目标上失效

新机会效果：

水【机会】【机会】：你可以选择一个目标承受的状态来代替祈愿持续效果

# 汹涌波浪——rank1

歌国的水性坐骑（先祖）粘土之船（炼金）伊索拉的旋转之舞（福神）

激活：作为一个移动行动，你可以执行一次难度为2的宗教（水）检定，目标为一个距离0-1的角色。此祈愿必须在周围有大量水体的情况下才能进行。

效果：如果你在小规模冲突中执行成功此祈愿，你召唤激流来推动目标。目标可以立刻移动距离1，以及你每个额外成功额外移动距离1。如果你在休息场景或者叙事场景中执行成功此祈愿，你召唤出温和的水流来让目标跨越或在水流中移动的速度翻倍（例如游泳或者坐船）。这个效果持续到此场景结束。
