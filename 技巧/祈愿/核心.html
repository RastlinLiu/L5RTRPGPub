# 核心

只有学派列表中具有祈愿技巧的角色才能购买祈愿技巧。

祈愿有着一系列新的方式来花费【机会】，gm是这些新的使用机会的方式是否该使用的最终裁决者。

# 合适的供奉

修验者享有和神灵之间的特殊关系，甚至这种关系高于其他祭祀。虽然理论上任何人都可以接触到神灵并获取它们的祝福，而所有祭祀都会进行此方面技艺的训练，但只有修验者掌握了解开这些力量巅峰的秘密。修验者学派投入了相当多的资源来保护它们的秘密，并且它们粉碎了一切试图窃取它们神圣学识的人。

即便是最有强大的祈愿，其核心也是信仰交换力量的简单交易。尽管物质上的供奉对于灵魂之物有些奇怪，但这些胜礼是十分有效的。训练有素的修验者可以不提供这些供奉，但这会使执行祈愿变得更加困难，因为修验者的魂魄必须代替供奉来承担负担。

在一次祈愿过程中提供物质供奉的修验者可以至多重置3个空白面的骰子。物质供奉将会以此方式被消耗掉。

# 强求祈愿

神灵有时候会被说服去执行特殊的事情。这被称作强求祈愿，允许角色执行他们通常无法执行的祈愿。

每个场景一次，学过一个或更多祈愿的角色可以选择一个他们未学习的祈愿（甚至他们不需要满足该祈愿的前提条件），并为神明提供供奉来换取此力量。供奉必须是神明感兴趣的东西，并且对该角色有一定价值。该物品即便不是独特的，也应该至少是稀有的。随着神明潜逃，该物品永久性的遗失了。

如果gm决定神灵对供奉感到满意，角色可以执行这个技巧相关的行动。然而，检定的难度增加1点，以及该技巧的先决条件每超过该角色当前学派等级一点再增加1点难度。该角色可以如常传导此祈愿，不过他只能执行此行动一次。

# 传导

祈求神灵是一个漫长的过程；即便可能角色很着急，但有些精魂却不能急于求成。耐心是与神灵交流的关键，以免神灵反过来对修验者感到不耐烦。 说辞、手势，或者承担仪式这些激活祈愿的实际行动，只会发生在让自己与力量调和从而让魔法变得可能之后。传导是一种让角色延迟完成祈愿的方式，允许角色将一次祈愿中的结果带到同一个属性的另一次祈愿中。

选择传导：当执行一次祈愿检定时，在第五部选择保留骰子时，角色可以选择传导任意数量的保留骰子，代替继续结算此检定，角色将这些传导的骰子进行保留，确保记录好每个保留骰子的结果。这次检定结束，玩家不再结算任意骰子上的结果或效果，包括成功以及失败

使用传导骰：在角色的下个回合内，如果他执行了一个相同元素的祈愿，他可以使用传导骰。在第三步建立骰池并掷骰子中，角色你可以少掷若干【轮】骰和【技能】骰，并将传导骰加到这次检定结果中。（将他们设置为当它们传导时的结果）

传导的限制：在冲突中，角色可以传导任意数量次。冲突外角色可以传导一次并加倍祈愿耗费的时间。

打断：如果一个具有传导骰的角色执行了不属于同一个元素的祈愿的行动时，所有的传导骰子都被丢掉。如果一个角色在传导中承受了眩目、沉默、昏迷状态，所有的保留骰子都会丢掉。

检定：由于祈愿依赖于呼唤指定的神灵，每个祈愿都要求有合适的方式进行。因此，祈愿的检定通常都需要使用指定轮而非交给gm按照玩家的叙述来进行解释。

# 精神反冲

修验者拥有着凡人无法企及的力量，他们接受了广泛的训练来避免与神灵互动中不激怒它们。然而，即便是最有经验的修验者，在向神灵求助时也可能面临意外的结果。此外，如果修验者的灵魂处于动荡之中，他们召唤的神灵也会同样混乱。一个精神不稳定的修验者在向湖泊祈求缓和水流时可能会召唤出一个破坏性的漩涡。

当一名修验者在一次祈检定中保留的骰子产生了三个或更多的冲突时，他将会承受精神反冲。他承受3点疲乏值，作为天然的力量过度透支了他的身体的表现，而且他在场景结束前无法使用同一元素进行祈愿，作为他的行为触怒了神灵从而神灵退散的表现。此外，他还会以不同元素承受如下效果：

风：gm选择至多两个在射程内的角色作为目标（或者gm选择一名玩家去选择）

火：该技巧目标射程内的每个角色。在gm的裁决下，0-2范围内的地形着火了，变为危险地形

地：该角色0到3范围内的地形获得不平衡特性，修验者的每个冲突会让地形保持一天不平衡特性

水：该角色0到1范围内地形变为泥潭并具有卷入特性

# 准备好的祈愿

通过卷轴或其他合适的材料，修验者可以将祈愿绑定到可消耗的容器中，制造出准备好的祈愿来让他们或其他人可以随后立刻启用。

通常情况下，一个修验者只能拥有一个准备好的祈愿；如果他准备了另一个，那么神灵的祝福就会从第一个容器中消散。任何持有准备好的祈愿的角色可以按照列出的方法激活它。

准备祈愿是一个休息行为，需要准备检定、使用的技能和材料。检定使用祈愿指定轮和难度。在进行检定后，玩家应该将结果记录下来。

护卫：护卫是用神秘符号标记的羊皮纸。

准备技能：作文

材料：书法套装，一袋羊皮纸

激活：角色可以将护卫放在任意可以附上的表面，他们将守卫上写上一个条件，指定什么情况释放它。当护卫的条件满足时，护卫在它所在的位置激活。

# 祈愿的名字和关键字

祈愿的名字：每个祈愿都有一系列别名列在名称下面。许多学派可以通过不同的方式来达成祈愿，这也是为什么不同传统的修验者可以达成同一祈愿的原因。

祈愿效果关键字：每个祈愿效果中都包含了一个或更多的斜体关键字，这些关键字提醒了祈愿的作用，一些能力也引用了这些关键字。

增强：改变或加强目标的祈愿

弥补：恢复目标的祈愿

净化：驱逐邪恶精魄的的祈愿

占卜：揭示信息或者展示其他时间和地点的一瞥的祈愿

打击：造成伤害的祈愿

召唤：创造些什么的祈愿
