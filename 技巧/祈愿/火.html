# 火祈愿

火之神灵都是凶猛且直接的，火祈愿也是如此的凶猛。这些祈祷是修验者可以召唤的最具有破坏力的一些祈愿

# 光辉护甲——rank2

觉醒火焰之油（炼金）大野窝的护甲（先祖）伊芙利特之袍（外国）

激活：作为一个支援行动，你可以执行一次难度为2的宗教（火）检定，目标为距离0-1的一套护甲。

效果：如果成功，你增强目标护甲，将其笼罩在刺眼的光芒和灼热的温度中（这个效果奇迹般的不影响穿戴者）。当一个位于穿戴该护甲者0-1的角色开始他的回合时，他们承受灼烧和眩目状态。这个效果持续一轮，以及你每个额外成功额外一轮。

新机会效果：

火【机会】：该护甲获得军备特质

火【机会】【机会】：此效果持续到此场景结束

# 刺痛之钢——rank1

不屈金属的安慰（炼金）觉醒的刀刃之焰（元素）兴国的熔炉（福神）

激活：作为一个支援行动，你可以执行一次难度为2的宗教（火）检定，目标为距离0-1的一件武器

效果：如果你成功，你通过耀眼的能量增强此武器。将该武器的基础伤害增加你火轮的数值。这个效果持续一轮，以及你每个额外成功额外一轮。

新机会效果：

火【机会】：在你进行此行动后，如果你增强的武器已经准备好，你可以立刻用此武器执行一个打击行动。

火【机会】+：如果你成功，你每以此法花费一个机会，你可以为你的武器致命性增加1点

火【机会】【机会】：此效果持续到此场景结束。

# 火龙之息——rank3

鬼之酒（炼金）富坚的炙热吐息（先祖）沙漠之风的标识（外国）

激活：作为一个攻击行动，你可以执行一次难度为4的宗教（火）检定，目标为一个距离0-3的角色。

效果：如果你成功，你呼出灼热之焰以打击目标及其周围的角色。目标承受等于你火轮数值+双倍额外成功的超自然伤害，并承受灼烧状态。每个距离目标0-1的角色都必须通过难度为3的体能检定（风4，水1）抵抗，否则将会承受等于你火轮数值的超自然伤害。

# 熄灭——rank1

激活：作为一个支援行动，你可以执行一次难度为3的宗教（火）检定，目标为至多三个距离0-3的世俗火源。

效果：如果你成功，你召唤出火神灵的力量吸入自身。熄灭每个目标火焰，每个以此法熄灭的火源都可以移除你的一点疲乏值。

新机会效果：

火【机会】【机会】：如果你成功，你也可以终止这片区域的全部来自其他火祈愿的持续效果，以及火焰为基础的超自然效果。

# 福禄寿的机智——rank1

木津的智慧（先祖）点燃心智之烛（元素）学王的颂歌（外国）

激活：作为一个支援行动，你可以执行一次难度为1的宗教（火）检定，目标自身。

效果：如果你成功，你可以通过福神的恩惠来增强目标的机智和狡诈。当其执行检定时，目标可以增加一个保留的被设置为【机会】【冲突】的【轮骰】加入结果中。这个效果持续一轮，以及你每个额外成功额外一轮。

新机会效果：

火【机会】：你可以目标一个距离你0-2的其他角色。

# 大野窝之怒——rank3

天火之球（炼金）宝丘的最后一战（先祖）收割火种（元素）

激活：作为一个攻击行动，你可以执行一次难度为5的宗教（火）检定，目标为距离0-4的一个角色。这个技巧只能在户外使用。如果天气本身就是暴雨，降低其难度一点

效果：如果你成功，你呼唤闪电来打击你的目标。目标承受等于你的火轮+三倍额外成功的超自然伤害。如果目标因此伤害而无法行动，他承受一次程度为8的致命一击。

新机会效果：

火【机会】：目标及每个距离其0-2的角色都必须通过一次难度为3的冥想检定（风4，水1）抵抗，否则将会承受眩目状态，以及等于你火轮数值三倍的冲突。

火【机会】+：你每以此法花费一个机会，增加此行动造成的致命一击程度2点。

# 火焰刀——rank1

炼火为钢（炼金）短暂回响（先祖）大野窝之剑（福神）

激活：作为一次支援行动，你可以进行一次难度为1的宗教（火）检定来在你手中召唤火焰之剑。

效果：如果你成功，你召唤并准备好一把火焰做的武士刀。它拥有军备特质，由此武器造成的伤害是超自然伤害而非物理伤害。武士刀持续到它离开了你的掌握，或者场景结束，那时它将会变为灰烬或者爆发为一阵火花。

新机会效果：

火【机会】：你可以召唤任意类型的近战武器。

火【机会】：你可以立刻使用你召唤的武器进行一次打击行动。

火【机会】：如果你成功，你可以召唤并准备好一对武器而非一把单独的武器。

火【机会】：在你以此武器进行一次攻击行动后，每个目标必须通过难度为3的体能检定（风4，水1）抵抗，否则将会承受燃烧状态。

# 松之战嚎——rank1

拥抱火焰（元素）战争阴霾（外国）布袋的诅咒（福神）

激活：作为一次密谋行动，你可以执行宗教（火）检定，目标至多等于你火轮数值个距离0-2的角色。这个检定的难度等于你目标中最高的警戒值。

效果：如果你成功，你召唤一个只有目标能感知到的地狱般的场景，将他们推向狂怒。目标承受1点冲突，以及你每个额外成功1点冲突，并且必须通过难度为3的冥想检定（风4，水1）来抵抗，否则立刻承受暴怒状态。

# 贪婪蜂群——rank3

朝比奈的遗忘狂怒（先祖）起舞的火球（元素）稻荷的狐火（福神）

激活：作为一个支援行动，你可以执行一次难度为3的宗教（火）检定，目标为一个距离为0-1，包含可燃物体的地点。物体将会被火吞噬。

效果：如果你成功，你召唤一定数量的燃烧微粒，数量等于你的火轮+被吞噬的物体的体型。这些微粒会跟随着你，定期向周围的物体点燃。当你在一次攻击行动中成功，你可以花费至多等于你学派等级的微粒来为你的行动增加额外成功，每个花费的微粒增加一个额外成功。微粒持续到此场景结束。如果你成功，你每进行一次火祈愿，你召唤出一个额外的微粒。这个效果持续到此场景结束。

新机会效果：

火【机会】：火焰微粒只会在你命令他们这么做时才会点燃周围的物体，而不是随意点燃。

火【机会】：当你防御伤害时，你可以花费至多等于你火轮数值个微粒来降低承受的伤害，每花费一个微粒会降低1点伤害。

# 火，崛起

哭泣化身（炼金）火之守卫（先祖）涡芽的侍女（福神）

激活：作为一个支援行动，你可以执行一次难度为6的宗教（火）检定，目标为一个距离0-3包含可燃物的地点。

效果：如果你成功，你召唤显灵火神灵。它吞噬了可燃物，并出现在目标地点。听从你的命令两轮，以及每个额外成功额外一轮。它在你每轮结束后立刻执行它的回合。当此效果结束，它会四分五裂，或者如果它被激发了特别的愤怒，那么它会开始遵循自己的指令。如果它开始遵循自己的指令行动，它除非被召唤的修验者激怒，否则不会攻击召唤的修验者

新机会效果：

火【机会】：火神灵在一个灼热的火柱中迸发而生。每个距离目标地点0-2的角色都必须通过难度为4的体能检定（风5，水2），否则将会承受灼烧和眩目状态。

火【机会】+：你每以此法花费一个机会，火神灵将会吞噬额外一个可燃物，每个花费机会同样可以扩展吞噬的范围1。为火神灵增加吞噬的可燃物体型总和数值的耐力值。

火【机会】：当火神灵的疲乏值超过了它的耐力值，它将被引爆，成为一道灼烧天地的火柱。每个距离0-3的角色都必须通过难度为5的体能检定（风6，水3）抵抗，否则将会承受等于你两倍火轮数值的超自然伤害，并承受灼烧和眩目状态。

# 净化之火——rank1

火龙的膏油（炼金）伊泽的仁慈狂怒（先祖）大野窝的平静（福神）

激活：作为一个攻击行动，你可以执行一次难度为1的宗教（火）检定，目标为每个距离0-2的角色。

效果：如果你成功，你以咆哮的火焰打击每个目标。每个目标承受眩目状态，且必须通过难度为3的体能检定（风4，水1）抵抗，否则将会承受等于你火轮数值+他们失败差额的超自然伤害，并获得灼烧状态。

# 火墙——rank2

被点燃的灰烬粉末（炼金）大野窝的追逐（福神）火宫殿之墙（外国）

激活：作为一个攻击和支援行动，你可以执行一次难度为4的宗教（火）检定，目标为两个距离2-4的地点

效果：如果你成功，你在两个地点间召唤出一道火墙，其厚度为1，高度为3；这片区域被视为危险区域。这个效果持续一轮，以及每个额外成功额外一轮。每个站在火墙爆发地方的角色都必须通过难度为2的体能检定（风3，水1）抵抗，否则将会承受等于你的火轮数值的超自然伤害，并获得灼烧状态

新机会效果：

火【机会】：火墙也被视为如下一种特质：卷入或模糊

新机会效果：

火【机会】【机会】：这个效果持续到此场景结束。
