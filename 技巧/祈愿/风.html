# 风祈愿

风之神灵是变化无常的存在，没有谁比它更灵活或敏锐。它们可以赋予轻盈的脚步、灵动的心灵，以及增强或遮蔽视线的能力

# 祝福之风——rank1

瓶装尘暴（炼金）多门的启示（福神）世界之柱（外国）

激活：作为一个支援行动，你可以执行一个难度为2的目标0到3范围内地点的宗教（风）检定。

效果：如果你成功，你在目标位置召唤一阵旋风。任何距离目标地点0-2范围内的角色在面对攻击行动时都视为处在模糊地形。这个效果持续一轮，加上你每个额外成功持续一轮。

新机会效果：

风【机会】：这个效果持续到场景结束。

# 以月亮大人之光——rank1

木津纪之眼（先祖）散影之风（元素）揭示鬼魂世界（外国）

激活：作为一个密谋行动，你可以执行一个难度为2的宗教（风）检定，目标为距离你0-2的地方。

效果：如果你成功，你占卜每个目标区域内的隐蔽角色和隐藏物体（包括秘密隔间、陷阱门、以及隐藏的武器），占卜的效果通过只有你能看见的虚幻而发光的轮廓来揭示。这个效果只能揭示那些通过世俗手段掩盖的物体和人。

新机会效果：

风【机会】+：如果你成功，你每以此法花费一个机会，就可以至多揭示一个魔法方式隐藏的物体。

风【机会】【机会】+：你每以此法花费两个机会，就可以选择一个距离为0-1的角色，被选择的角色也可以看见物体。

# 唤风——rank2

鹪鹩的温柔之翼（炼金），道寺之步（先祖），弁天的微笑（福神）

激活：作为一个移动行动，你可以执行一个目标为自身的难度为4的宗教（风）检定

效果：如果成功，你以一阵风增强你的移动并载着你扶摇而上。除去你可以在垂直方向可以自由移动，你在飞行过程中忽略地形效果。这个效果持续到场景结束。

新机会效果：

风【机会】【机会】+：你每以此法花费两个机会，你可以携带一名范围0-2内自愿的角色一起飞行。

# 夜之斗篷——rank1

窃光之烟（炼金）北风之幕（福神）行走于暗影世界（外国）

激活：作为一个密谋和支援行动，你可以执行一个宗教（风）检定，目标为一个0-1的角色或物体，难度为目标的体型。

效果：如果你成功，你为你的目标增强一个幻觉，使其肉眼不可见。目标物体依然物理性质上依然处于该处，且可被接触，或者任何除去视觉以外的方式感知。这个效果持续一轮，以及每个额外成功额外持续一轮。

新机会效果：

风【机会】【机会】+：你每以此法花费两个机会，可以选择额外一个体系等于或小于的目标

# 狐魂的虚假领域——rank4

生驹回想传说（先祖）迷失于伊泽之森（福神）窥视镜头（外国）

激活：作为一个密谋行动，你可以执行一次难度为5的宗教（风）检定，目标为一个范围0-4内的区域。

效果：如果成功，你于目标区域召唤幻影地形，延展至多等于你风轮数值的范围。一个未发现幻象虚假本质的角色必须通过难度为4的求生检定（地5，火2）来在一定距离发现其虚假本质。由于幻象没有实体，任何接触都会揭示其谎言本质（尽管幻象还存在于那里）。这个效果持续等于你风轮+额外成功的轮数（或者在一个叙事场景或者休息场景）

新机会效果：

风【机会】：这个效果持续到场景结束。

风【机会】+：你每以此法花费一个机会，幻象地形获得一个特性：危险，卷入和模糊。

# 紧握风龙——rank3

史卡金之跃（先祖）事故树之旅（福神）风的摇篮（外国）

激活：作为一个攻击或支援行动，你可以执行一次难度为4的宗教（风）检定，目标距离3-5的一个角色。

效果：如果成功，你召唤一阵烈风，移动等于你风轮+你的额外成功距离（至多距离为6），忽略其中任意地形。你可以选择将目标轻轻放在地上（如果你选择了支援行动），或者狠狠的扔到地上（如果你选择了攻击行动）。当你狠狠的将目标扔到地上时，目标必须进行难度为4的体能检定（土5，火2）否则需要承受距离3的高度坠落效果。

新机会效果：

风【机会】+：你每以此法花费一个机会，此技巧的最远距离增加1

风【机会】【机会】+：你每以此法花费两个机会，增加坠落的高度1

# 风之面具——rank2

沾上移动羽毛（炼金）全矛之面（先祖）萨达箱的玩笑（福神）

激活：作为一个密谋和支援行动，你可以执行一个难度为3的宗教（风）检定，目标于自身

效果：如果成功，你通过幻象增强你自身让你看起来像其他人。观察者必须通过难度为4的情绪检定（土5，火2）来注意到你的外貌上有些不对。如果观察者成功，他们可以使用【机会】来专门辨识你。这个效果持续到本场景结束。

新机会效果：

风【机会】：你可以改变你的嗓音和声音来让你听起来像幻象的角色。观察者必须通过难度为4的情绪检定（土5，火2）来注意到你的声音上有些不对。如果观察者成功，他们可以使用【机会】来专门辨识你。

风【机会】【机会】+：你每以此法花费两个机会，那么你可以让效果持续额外一个场景。

荣誉和祈愿：有些祈愿可能会允许或者鼓励pc进行不荣誉的行为。例如风之面具，一个角色可以冒充比自己地位更高的角色，而风之秘密则允许角色毫不费力的偷听他们的领主。gm应该提醒玩家角色违反武士道行为依然会降低或可能降低载誉，即便这些行为时通过神秘力量来达成目标的。

# 自然之触——rank1

新条的低语（先祖）与麒麟并驾齐驱（福神）幽谷王之声（外国）

激活：作为一个支援行动，你可以执行一个难度为1的宗教（风）检定，目标为自身。

效果：如果成功，你可以增强自己与自然中的动物交谈的能力（不包含异界生物或者被污染的生物）。动物能懂得你的讲话（以他们自身的能力来做到这一点——人类的政治关系对于马来说没有任何意义，因此不论你向它解释多少次也没有意义），不过这并不能迫使他们服从你。这个效果持续到本场景结束。

新机会效果：

风【机会】：直到本场景结束，你可以听懂动物间的交谈

风【机会】：如果你成功，动物会因为你的礼貌对你自动产生好感，可以满足一个不会让他们陷入危险的要求，而不需要通过食物进行贿赂或者进行后续的说服检定。

风【机会】【机会】：如果你成功，你可以通过此法与异界生物交流，只要它有话想跟你说。

# 风，崛起——rank4

尘暴化身（炼金）风之守护（先祖）召唤风巨灵（外国）

激活：作为一个支援行动，你可以执行一次难度为6的宗教（风）检定，目标为距离3-5的一个地点。

效果：如果成功，你召唤一个显灵风元素为你而战。它出现在目标的位置，听从你的命令两轮，再加上你的额外成功数数量的轮数，每轮在你行动之后立刻行动。当此效果结束，它会离开，或者如果它被激发了特别的愤怒，那么它会开始遵循自己的指令。如果它开始遵循自己的指令行动，它除非被召唤的修验者激怒，否则不会攻击召唤的修验者。

新机会效果：

风【机会】：显灵风元素在飓风中到来，将附近的所有东西都扔到一边。每个距离目标地点0-2的角色都必须通过难度为4的体能检定（土5，火2），否则将会承受等于你风轮数值的超自然伤害，并被推远2距离。

# 风中秘密——rank2

激活：作为一个密谋行动，你可以执行一次难度为3的宗教（风）检定，目标为距离为0的一个地点。

效果：如果你成功，你通过密语的风来占卜，偷听选择地点的内容。风神灵可以偷听到等于你风轮数值范围内任何说过的话。这个效果持续24小时，直到你选择一个新的地点去偷听。在效果持续期间，你可以花费一个行动听到当前被偷听地点说的话。

新机会效果：

风【机会】：除去你可以听到偷听地点的声音外，你还可以闻到该地点的味道。

风【机会】：如果另一个角色发现了你的祈愿，这个效果立刻结束，防止他们确定是你的偷听行为

风【机会】【机会】+：你每以此法花费两个机会，效果持续额外24个小时。

# 记忆的象征——rank1

阿贺娑的谜语（先祖）风神的诡计（元素）惠比寿的玩笑（福神）

激活：作为一个密谋行动，你可以执行一个难度为2的宗教（风）检定，目标为一个距离0-1的地点。

效果：如果成功，你在目标位置召唤一个无生命物体的幻象。幻象的体型至多为你的额外成功数（起始为0，或者一个小型的手握物体）。这个物体看起来像是真的，但并不实际存在，而且不能被使用。任何面对此物的角色都必须通过难度为4的工匠、锻造或者设计检定来使用学者技能组的技能发现幻象的本质。幻象持续到本场景结束

新机会效果：

风【机会】：物体可以是一个生物或者人（尽管它不能移动）

风【机会】+：你每以此法花费一个机会，就可以在范围内召唤一个额外的幻象物体

风【机会】【机会】：你召唤的幻象物体拥有质量，并且可以被当作真实物体使用（尽管他们还是无生命的，也不能通过他们自身移动）。他们在效果结束后依然会消失。

# 蒸汽梦魇——rank3

富坚的轻语（先祖）你内心的敌人（元素）业力之轮的阴影（福神）

激活：作为一个攻击行动，你可以执行一次难度为4的宗教（风）检定，目标一个距离为2-3的角色。

效果：如果你成功，你召唤一个你目标最大恐惧的幻象。你的目标必须通过一个难度为4的冥想检定（土5，火2）来看穿幻影。如果他们失败，他们承受等于你风轮+你的额外成功的冲突值，此外如果他们因此妥协，他们必须立刻揭面。如果他们在幻象存续期间揭面，他们将专注于幻象，攻击它，逃离它，或者对它大发雷霆（而不是对你或者对其他人）。幻象持续等于你风轮数值的轮数。

新机会效果：

风【机会】：如果目标在冥想检定中失败，他们同时还承受迷惑状态

风【机会】：幻象持续到此场景结束。

# 风神之怒——rank5

瓶中台风（炼金）歌国惊跑（先祖）风龙之怒（元素）

激活：作为一个攻击行动，你可以执行一次难度为5的宗教（风）检定，目标为范围2-4的每个角色。

新机会效果：

风【机会】：你可以通过行动来终止飓风。

风【机会】+：你每以此效果花费一个机会，每个在你回合结束时体能检定失败的目标将会被扔出1距离。每个被扔出的角色落地时会遭受等于他们移动距离的落地距离效果。

风【机会】【机会】：当你在飓风中，你可以纵向移动，忽略地形效果。

# 风之枪——rank1

风枪之杖（炼金）水灵之矛（福神）巨灵之戟（外国）

激活：作为一个支援行动，你可以执行一次难度为1的宗教（风）检定来召唤一把由旋风制成的长矛入手。

新机会效果：

风【机会】：召唤的武器可以是任何远程或近战武器。

风【机会】：在你进行此行动后，你可以立刻使用你召唤的武器进行一次守卫行动

风【机会】【机会】：你可以召唤一对武器而非一把武器。
