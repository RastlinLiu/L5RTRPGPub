# 通用招式

# core

# 心中之战——rank3

激活：当你在一次使用空轮的决斗主动性检测中，你可以以如下方法使用【机会】：

空【机会】【机会】：你声明两个轮，你的对手必须从这两个轮中选择一个。当你的对手在自己的首轮决斗中选择架势时不能选择他们选择的轮。

空【机会】+：选择一个技巧类型，每花费一个【机会】，则了解一个你的对手在此类型下的知晓的技巧

# 风息式——rank4

激活：当你进行一次武术【近战、远程或徒手】检定时，你可以用如下方式使用【机会】：

风【机会】+：你行动的目标之一必须以难度3的体能检定（土4，火1）抵抗，否则会遭受迷惑状态，并承受检定差额的疲乏值。你每花费1个【机会】就提升需要抵抗的难度值1点。

# 破浪式——rank4

激活：当你进行一次武术【近战、远程或徒手】（水）时，你可以用如下方式使用【机会】：

水【机会】+：你行动的目标之一必须以难度3的体能检定（土1，火4）抵抗，否则会遭受流血状态，并承受检定差额的疲乏值。你每花费1个【机会】就提升需要抵抗的难度值1点。

# 新月式——rank2

激活：当你执行守护行动时，你可以用如下方法使用【机会】：

【机会】：在一个距离1或2范围的角色对你或你守卫的角色执行一次攻击行动时，你可以通过一把准备好的武术武器【近战或徒手】进行一次目标该角色的打击行动（如果他在你的武器攻击范围内）。这个效果持续到你的下个回合开始，或者你以此法执行了一次打击行动。

# 红叶击——rank3

激活：作为使用一把准备好的武器的一次攻击行动，你可以进行一次与该武器相对应技能的难度4的武术（土）检定，目标为一名你武器范围内的角色。

效果：如果你成功，那么你的目标承受等于你土轮加上你的额外成功数量的物理伤害。选择一把你的目标准备好的武器，目标必须抵抗难度4的体能检定（风2，水5），否则将会失去所选的武器，该武器朝着你选择的方向至多远离范围3。

新机会效果：

土【机会】：如果你缴械了你的对手，你可以控制该武器而非击飞它

# 失世式——rank4

激活：当你进行一次武术【近战、远程或徒手】（火）时，你可以用如下方式使用【机会】：

火【机会】+：你行动的目标之一必须以难度3的体能检定（风4，水1）抵抗，否则会遭受眩目状态，并承受检定差额的疲乏值。你每花费1个【机会】就提升需要抵抗的难度值1点。

# 流水击——rank3

激活：作为使用一把准备好的武器的一次攻击和移动行动，你可以进行一次与该武器相对应技能的难度4的武术（水）检定，目标为一名距离0-2范围内的角色（无视你的武器本身的距离限制）

效果：如果你成功，那么你的目标承受等于你水轮的物理伤害并承受流血状态。目标必须抵抗难度为4的体能检定（土2，火5），如果失败了则将他们承受的物理伤害加上你的武器的基础伤害值。

新机会效果：

水【机会】：移除自身的眩目、迷惑、定身、倒地状态

# 穿心击——rank3

激活：作为使用一把准备好的武器的一次攻击和移动行动，你可以进行一次与该武器相对应技能的难度4的武术（火）检定，目标为一名你武器范围内的角色。

效果：如果你成功，那么你的目标承受等同于你的武器致命值+你的额外成功数程度的致命一击。抵抗此致命一击的体能检定难度为3（风4水1）而非通常难度。如果你失败了，你承受迷惑状态。

新机会效果：

火【机会】：结算此效果前，至多向你的目标移动1范围。

# 铁山式——rank4

当你进行一次武术【近战、远程或徒手】（土）时，你可以用如下方式使用【机会】：

土【机会】+：你行动的目标之一必须以难度3的体能检定（风1，水4）抵抗，否则会遭受倒地状态，并承受检定差额的疲乏值。你每花费1个【机会】就提升需要抵抗的难度值1点。

# 樋田大人之握——巨蟹rank2

激活：作为一个攻击行动，你可以目标一个距离0～1范围内的体型3或更大的生物进行一次体能（空）检定。检定的难度等于目标体型（至少为1）。

效果：如果你成功，你的目标遭受定身状态。如果你成功，你对所有执行攻击行动目标你的目标的角色算作提供协助。这个效果持续到你的下个回合开始。

新机会效果：

空【机会】：你的目标必须通过体能检定进行抵抗，否则需要承受等于体型的物理伤害（无视抵抗）以及倒地状态。这个检定的难度等于你的学派等级。

# 柴大人的勇气——凤凰rank2

激活：每次游戏过程一次，作为一个使用准备好的武器的支援行动，你可以目标任意数量在你的武器范围之内的友方角色来做一个难度为2的战术（空）检定。

效果：如果你成功，其他角色如果可以选择你为攻击或者密谋行动的目标的话，那么不能选择你的目标作为目标。如果该行动选择多个目标，则你必须作为第一个被选择的目标。该效果持续到等于你学派等级的回合数。

# 翱翔切削——rank1

激活：作为使用一把准备好的单手武器的一次攻击行动，你可以进行一次与该武器相对应技能的难度2的武术检定，目标为一名距离你2-3的角色。

效果：你将你的武器向你的目标投掷。如果你成功，你对你的目标造成等于你的武器基础伤害+你的额外成功数量的物理伤害。如果目标防御了这次伤害，那么武器在距离目标你选择的范围内落地，如果目标承受了致命一击，那么武器则陷入目标体内。如果你失败了，武器将会飞向目标的方向，距离为此技巧的最大距离

新机会效果：

【机会】+：每以此法花费一个机会，此技巧的距离最大值加一

【机会】：降低对目标的下一次进攻检定难度1点，直到你的下一个回合结束。

# 斩断灵魂——rank5

激活：作为一个支援行动，你可以进行一次难度为4的冥想（空）检定来让你的刀剑和你荣耀的祖先之灵协调一致。

效果：如果你成功，选择一个准备好的武器，它获得神圣品质，当你使用它来对外界的存在造成致命一击时，在致命性上增加你的额外成功数。这个效果持续到你的下个回合结束。

新机会效果：

空【机会】+：你每以此法花费一个机会，此效果持续时间增加一个回合

空【机会】【机会】：他界的存在无法对你执行进攻行动直到你的下一个回合开始

空【机会】【机会】：你可以用你准备好的武器立刻执行一个打击行动

# 如风击——rank1

激活：当你进行一次武术【近战、远程或徒手】（风）检定时，你可以用如下方式使用【机会】：

风【机会】+：保留一个你掷出的骰子，若你每以此法花费两个【机会】则可以额外保留一个骰子。当你在你的下个回合结束前进行相同技能检定时，你可以少掷若干【轮】骰和【技能】骰，并将保留的骰子加到这次检定中。这些骰子算作掷出的骰子，且将保留时的结果加到结果中。

# 如土击——rank1

激活：当你进行一次武术【近战、远程或徒手】（土）检定时，你可以用如下方式使用【机会】：

土【机会】+：每以此法花费一个机会，你的物理抵抗视为原有结果加一，持续到你的下个回合开始。

# 如火击——rank1

激活：当你进行一次武术【近战、远程或徒手】（火）检定时，你可以用如下方式使用【机会】：

火【机会】+：你的目标下一次遭受致命一击时，你每以此法花费一个机会，该致命一击程度加一。该效果持续到你的下个回合结束。

# 如空击——rank5

激活：当你进行一次武术【近战、远程或徒手】（空）检定时，你可以用如下方式使用【机会】：

空【机会】：立刻切换到一个不同的架势

空【机会】：如果你失败，那么你可以执行另一个行动。这个行动必须是你这回合没有执行过的行动。你每回合只能激活这个效果一次。

# 如水击——rank1

激活：当你进行一次武术【近战、远程或徒手】（水）检定时，你可以用如下方式使用【机会】：

水【机会】+：每以此法花费一个机会，你的目标物理抵抗视为原有结果减一，持续到你的下个回合结束。

# 战术评估——rank2

激活：当你进行主动性检定时，你可以用如下方法使用【机会】：

风【机会】：选择一个在冲突中的敌人。了解一个他的优势（他选择）以及他的物理或超自然抵抗（你选择）

土【机会】：选择一个在冲突中的角色，在他的第一个回合结束前，不需要应用他的劣势到检定中。

水【机会】：选择在此场景内的一个地形，直到你的首个回合开始，对于目标你和你队友的攻击行动检定，它视作具有模糊特性

火【机会】：选择一个在冲突中的角色以及一个你知道的他的优势，直到他的首个回合结束，他可以将该优势用于所有他的检定

# 战士决心——rank1

激活：每个场景一次，作为一个支援行动，你可以花费一个虚空点来恢复

效果：移除等于你的载誉等级的疲乏值。
