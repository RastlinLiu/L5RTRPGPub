# 远程招式

# core

# 鹰的精准——rank1

激活：当你执行一次武术【远程】检定时，你可以用如下方式使用【机会】：

【机会】+：你每以此法花费一个机会，可以为你的武器射程增加1点，至多为距离6

# 怒雹式——rank2

激活：当你执行一次武术【远程】检定时，你可以用如下方式使用【机会】：

【机会】+：你每以此法花费一个机会，则可以选择一个距离0到2内的其他角色，每个你选择的角色承受你的武器基本伤害数的冲突。

# 钉于扇上——rank5

激活：作为使用一把准备好的远程武器的一次攻击和移动行动，你可以执行一次武术【远程】检定，目标为你的武器射程。检定的难度为目标的警戒值加上攻击的距离

效果：如果你成功，目标承受等于你武器基础伤害值两倍+你的额外成功数的伤害。

新机会效果：

【机会】：如果目标因为此次伤害获得致残状态，则他承受一次致命一击，程度为两倍的武器致命值+额外成功。
