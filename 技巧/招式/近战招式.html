# 近战招式

# core

# 盘蛇式——rank2

激活：当你使用罗网属性武器进行武术【近战】或武术【徒手】检定时，你可以用如下方式使用【机会】：

【机会】+：选择一个目标准备好的武器，那个武器不能用作攻击行动，这个效果持续到你的下个回合结束。

【机会】【机会】+：每花费两个机会可以让一个目标获得定身状态。

# 居合斩：横刀术——rank2

激活：作为一个使用一把入鞘的锐锋武器的攻击和移动行动，你可以执行一次目标为距离1到2的难度为2的武术【近战】检定。

效果：你抽出并准备好入鞘的的武器，单手持握该武器。如果成功，你对目标造成你武器致命性+额外成功数的物理伤害。

新机会效果：

【机会】：准备好另一把入鞘的锐锋武器。

# 居合斩：拔刀术——rank2

激活：作为一个使用一把入鞘的锐锋武器的攻击行动，你可以执行一次目标为距离0到1的难度为目标警戒值的武术【近战】检定。

效果：你抽出并准备好入鞘的的武器，单手持握该武器。如果成功，你对目标造成你武器致命性的物理伤害。如果你的目标妥协了，那么他无法防御此伤害。

新机会效果：

【机会】：准备好另一把入鞘的锐锋武器。

# 铁林式——rank2

激活：当你使用长柄武器执行一次武术【近战】检定时，你可以用如下方式使用【机会】：

【机会】+：在你武器范围内的区域被视为危险地形，你每以此法花费一个机会可以影响一个目标。当被你影响的角色试图向你移动时，他们必须执行一个难度为4的体能检定（风2，水5），如果他们失败他们不能前进。这个效果持续到你的下个回合开始。

# 开手式——rank2

激活：当你使用罗网属性武器进行武术【徒手】检定时，你可以用如下方式使用【机会】：

【机会】【机会】+：你每以此法花费两个机会可以选择一个目标，该目标必须切换至不同于你选择的架势。体型3或以上的目标可以通过难度为3的体能检定来忽略此效果。

# 冲崩式——rank2

激活：当你使用钝器武器进行武术【近战】检定时，你可以用如下方式使用【机会】：

【机会】：如果你失败的差值在2点或以内，你的目标承受等于你体能技能的等级的物理伤害。

【机会】：如果你成功且你的对象处于倒地状态，你造成的伤害增加你体能技能等级。

# 转剑式——rank2

激活：当你执行一次武术【近战】或武术【徒手】检定时，如果你有第二把准备好的武器但没有用于此次攻击，你可以用如下方式使用【机会】：

【机会】+：你必须花费等同于你的目标的警戒值的机会数量来激活此技巧。你使用你第二把准备好的武器对你的一个目标进行行动，使其承受等同于第二把准备好武器基础伤害值的物理伤害。如果你的目标存在眩目状态，增加你使用的技能等级数值作为物理伤害。

# 霹雳击——rank3

激活：作为使用一把准备好的武器的一次攻击行动，你可以执行一次武术【近战】或武术【徒手】的难度3的风检定，目标每个你武器范围内的角色

效果：如果成功，每个目标承受你武器的基础伤害数值的伤害，并且需要进行一次难度3的体能检定（土4，火1）抵抗，否则需要与你相反的方向远离2距离。

新机会效果：

风【机会】+：你每以此法花费一个机会，抵抗效果的检定难度加一。

# 隐吓式——rank2

激活：当你使用单手武器执行一个武术【近战】或武术【徒手】攻击检定时，你可以用如下方式使用【机会】：

【机会】+：如果你成功，选择一个未察觉或迷惑状态的目标，该目标承受致命一击，程度等于你的武器致命值加上你以此法花费的第一个以后的机会总额。

# CoS

# 撑杆跳——rank3

激活：作为一个使用一柄准备好的长柄武器的攻击和移动行动，你可以执行一次难度为3的风或水属性的武术（近战）检定，目标为一个距离3-4的角色。

效果：你可以向你的目标移动距离2，忽略你越过的所有地形效果。gm允许的情况下，还可以跨过不可通行地形，例如小裂谷或者矮墙。

如果你成功，你对目标造成物理伤害，伤害值为武器的基础伤害加上额外成功。

如果你低于难度的数值为2或以上，你会遭受倒地状态，并承受2点疲乏和2点冲突

# 绊腿击——rank1

激活：作为一个使用一柄准备好的长柄武器的攻击行动，你可以执行一次难度为2的武术【近战】检定，目标一个距离1-2的角色。

效果：如果你成功，目标遭遇倒地状态。

新机会效果：

【机会】：如果成功，目标遭受2点疲乏和2点冲突。

【机会】+：如果目标的警戒低于或等于你以此法花费的机会，目标遭受迷惑状态。
