# 典礼

典礼只能在休息场景及其他根据gm判断的场景中执行。只有学派列表中具有典礼技巧的角色才能购买典礼技巧。

# core

# 清洁仪式——rank1

激活：作为一个休整活动，你可以执行一次难度为3的宗教（空）检定，目标为至多五名角色，在超自然污染在目标身上扩散开前移除它们。注意此技巧无法移除已经拥有的阴影之地污染劣势，不过它可以防止角色暴露在超自然污染后被污染扎根。

效果：如果成功，移除目标的折磨状态

新机会效果：

空【机会】：如果一个或更多目标是异界存在，你在完成这个典礼后你就会知道这一点——不过你不知道哪个目标是异界存在

空【机会】+：你每以此法花费一个机会，可以选择一个额外的角色。

空【机会】+：你每以此法花费一个机会，你可以选择一片等于花费机会数范围的区域。移除这片地区的亵渎特质。如果这片区域是非常严重的不洁区域，曾经长时间属于邪恶的领域，那么亵渎特质会在一天以及你每个额外成功一天后回到这片区域。

# 与灵交流——rank1

激活：作为一个休整活动或者支援行动，你可以执行一个难度为1使用风、土、火、水或空的宗教检定，向附近对应元素的灵体寻求帮助（如果使用空则是向已逝的灵体，例如先祖之魂或者其他非元素的超自然存在）。

效果：如果成功，选择并获得一个下列恩赐：

感受神灵：侦测等于你选择的轮数+额外成功数距离范围内的全部灵体以及使用他们的修验者们。

超自然知识：了解灵体们知道的这片区域的一个事实

元素礼物：神灵会向你提供少量的对应元素。例如，向水神灵祈祷可能会造成一场小雨，也可能导致你出现在一道溪流中间，甚至于发现地下水。相反的是，向火神灵祈祷可能会带来一片小火花，而对风神灵祈祷会带来一阵风，对地神灵祈祷会找到一块大石头，或者在荒地中找到一片可耕种的土地。对先祖的祈祷可能会让祈祷者在需要的时候找到先祖的东西。

新机会效果：

【机会】：降低你下一次与此元素对应的神灵或超自然力量互动的检定难度减一。

【机会】+：如果你成功，你每以此法花费一个机会，就可以获得一个额外恩赐，每次激活每种恩惠只能获得一次

# 预测——rank1

激活：作为一个休整活动，你可以执行一次难度为2的宗教（空）检定，目标为一个角色来看到他们未来的一瞥

效果：如果你成功，选择一个下列征兆作为目标角色的近期未来：

苍龙预兆：目标在本场游戏中下一次做一个工匠技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

黑龟预兆：目标在本场游戏中下一次做一个学者技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

白虎预兆：目标在本场游戏中下一次做一个武术技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

朱鸟预兆：目标在本场游戏中下一次做一个社交技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

牛郎织女预兆：目标在本场游戏中下一次做一个交易技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

如果你失败，gm来选择以上的一个预兆代替。目标直到下一场游戏前无法接受另一次预测

新机会效果：

【机会】：如果你失败，在第一个预兆之后你会接受额外一个预兆。在gm选择以外你也选择一个如上的预兆

# 茶会——rank2

激活：每场游戏一次，作为一个使用茶具的休整活动，你可以执行一次难度为2的表演（空）检定，目标为你自己和至多等于你文化技能等级的其他角色，这些角色也参加茶会。至多一名参加角色可以协助，如常提供效果。

效果：每个目标移除3点疲乏值，以及你每个额外成功额外移除一点疲乏

新机会效果：

【机会】：每个目标的下次检定难度降低一点

【机会】+：每个目标移除1点疲乏，以及你每花费两点机会额外移除一点疲乏

【机会】【机会】：如果你成功，每个目标获得一个虚空点

# 门栏屏障——rank1

激活：作为一个休整活动，你可以通过风、土、火、水、空来做一次难度为2的宗教检定，目标为距离0-1的一个门栏、大门或者其他入口

效果：如果你成功， 门栏会阻止特定类型的实体，基于你选择的轮类型。这个效果持续一个小时，以及每个额外成功额外持续一个小时。当对应类型的实体试图进入时，它必须通过难度为3的冥想（空）检定抵抗，否则将会被超自然屏障驱逐。不管它成功还是失败，你都会立刻知道它试图进入的意图。基于轮类型的特定实体类型如下：

风：灵体异界存在，例如灵或者鬼魂

土：活着的被污染存在

火：肉体形态的不死生物

水：野生动物

空：人类

新机会效果：

【机会】：你可以保护至多等于你额外成功数范围的区域。

【机会】：如果在过去二十四小时内，有任何你试图封锁门栏的生物通过了门栏，你会意识到他们的通过。

# shadowlands

# 钢之祝福——工匠rank2

激活：作为休整活动，你可以执行一次难度为2的锻造检定，可以使用任何轮来维护你的武器和护甲。

效果：如果你成功，基于你检定使用的轮，将下列效果其中之一应用于你维护的武器和护甲。

风：增加武器的致命度1点

土：移除物品的损坏特质，如果该物品没有损坏特质，那么获得耐用特质

火：物品获得逻辑上可以获得的特质。GM是以此法获得特质的最终裁判者，并且GM应该在PC决定是否执行此休整活动前，告知PC其可以通过此法获得的特质。

水：增加武器的基础伤害1点，减少其致命度1点，或者增加护甲的超自然抵抗1点，减少其物理抵抗1点。

空：降低下一个通过武器进行的检定1点

以上效果只能有一个应用在一个物品上。任何以此方法添加的效果持续到下一个使用该物品的场景结束

# 制造式神——修验者rank2

激活：作为使用体型0或以下神圣容器的休整行动，你可以执行一次空轮的工匠检定来制作一个式神，检定难度由你选择（最小难度为1）。

效果：如果成功，你可以选择一系列等级至多等于检定难度的祈愿。将这些被选择的祈愿记录下来并给式神起一个独有的名字。式神存在期间，被选择的祈愿被封印在式神其中——你不能使用这些祈愿，但式神可以。你同时只能拥有一个式神。当式神的物理形态被摧毁，它的精髓将会回到你身上；你可以为它通过制造式神的方法制造一个新的物理形态，不过它可以保留记忆和个性。

式神具有智能，可以与你交谈，并且基本上会服从你的命令。你可以通过一个行动来执行你封印与式神中的任何祈愿，使用你的创作技能和式神的轮值进行技能检定。

新机会效果：

【机会】：选择一个你的优势和一个你的劣势，把它们加在式神上。

【机会】：作为以制造物体或者材料的形象出现的代替，式神可以以你选择的同体型的形象出现。

【机会】+：你每以此法花费一个机会，该式神体型增加1

【机会】+：选择一个技能，你以此法花费的机会数，式神在此技能上拥有对应数值。这个效果可以启动多次，每次选择不同的技能。

【机会】+：你每以此法花费一个机会，式神的耐力和镇静都会增加2

# 式神详情：

爪牙 军事1 政治1

风1 土1 火1 空1 水1

耐力4 镇静4 专注2 警戒1

举止：凶猛 火+2 土-2

工匠1 武术1 学问1 社交1 贸易1

优势：不知疲倦 武术，物理

劣势：反抗 武术，心智

擅长的武器和装备：

切口：距离0，伤害1，致命2

已装备：无

能力：

灌注力量：封印在式神的祈愿越强大，其相关的轮值越高。如果式神被封印的同属性祈愿有多个，那么该轮值取这些祈愿的最高等级（如一个属性只有一个祈愿，则该祈愿等级为该轮等级）。

例如：一个式神被封印了1个rank 2风祈愿，1个rank 1土祈愿，1个rank 3水祈愿，那么该式神的轮属性为风2 土1 火1 空1 水3（如果该式神再被封印一个rank3的风祈愿，那么该轮属性为风3 土1 火1 空1 水3）。

活祈愿：每个场景一次，作为一个行动，式神可以执行一个被封印的祈愿而不需要检定，这个祈愿被视为有2个额外成功，以及【机会】【机会】。

限制行动：在斗争中，如果式神和它的创造者都还在，式神不会执行自己的回合。在修验者的回合，它可以至多移动1距离。作为一个支援行动，创造者可以让式神执行协助行动。

式神的动物形态：式神通常使用纸或者其他常规材料创造，不过它也可以用泥土、金属、液体或者一小团火炎制成。它们通常是以小鬼魂的人形出现，不过也可以按照修验者的意愿以动物形态出现，根据制作式神的祈愿属性，一些常见的形态如下：

  • 风：鸟，蝴蝶，蝙蝠，飞蛾
  • 水：鱼，青蛙，蜥蜴
  • 火：蜻蜓，萤火虫，蜈蚣
  • 土：老鼠，蜘蛛

# 正式茶会——rank5

激活：每次游戏流程一次，做为使用茶具（核心书248页）的修正行动，你可以执行一次难度为5的表演（空）检定，目标为你自己和至多等于你文化等级数的其他角色，这些其他角色是参与者。至多一个参与者可以协助，提供通常的协助好处。

效果：每个参与者移除6+额外成功数的冲突。

如果你成功，每个参与者获得1点虚空点。

新机会效果：

【机会】【机会】：如果你成功，每个参与者下次检定难度-2。 【机会】+：每个参与者移除2点疲乏，以及你每以此法花费一个机会，再移除1点疲乏 【机会】【机会】：如果你成功，每个比你地位高的参与者都能让你获得1荣耀值。
