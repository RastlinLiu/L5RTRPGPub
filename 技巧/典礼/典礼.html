# 典礼

典礼只能在休息场景及其他根据gm判断的场景中执行。只有学派列表中具有典礼技巧的角色才能购买典礼技巧。

# core

# 清洁仪式——rank1

激活：作为一个休息活动，你可以执行一次难度为3的宗教（空）检定，目标为至多五名角色，在超自然污染在目标身上扩散开前移除它们。注意此技巧无法移除已经拥有的阴影之地污染劣势，不过它可以防止角色暴露在超自然污染后被污染扎根。

效果：如果成功，移除目标的折磨状态

新机会效果：

空【机会】：如果一个或更多目标是异界存在，你在完成这个典礼后你就会知道这一点——不过你不知道哪个目标是异界存在

空【机会】+：你每以此法花费一个机会，可以选择一个额外的角色。

空【机会】+：你每以此法花费一个机会，你可以选择一片等于花费机会数范围的区域。移除这片地区的亵渎特质。如果这片区域是非常严重的不洁区域，曾经长时间属于邪恶的领域，那么亵渎特质会在一天以及你每个额外成功一天后回到这片区域。

# 与灵交流——rank1

激活：作为一个休息活动或者支援行动，你可以执行一个难度为1使用风、土、火、水或空的宗教检定，向附近对应元素的灵体寻求帮助（如果使用空则是向已逝的灵体，例如先祖之魂或者其他非元素的超自然存在）。

效果：如果成功，选择并获得一个下列恩赐：

感受神灵：侦测等于你选择的轮数+额外成功数距离范围内的全部灵体以及使用他们的修验者们。

超自然知识：了解灵体们知道的这片区域的一个事实

元素礼物：神灵会向你提供少量的对应元素。例如，向水神灵祈祷可能会造成一场小雨，也可能导致你出现在一道溪流中间，甚至于发现地下水。相反的是，向火神灵祈祷可能会带来一片小火花，而对风神灵祈祷会带来一阵风，对地神灵祈祷会找到一块大石头，或者在荒地中找到一片可耕种的土地。对先祖的祈祷可能会让祈祷者在需要的时候找到先祖的东西。

新机会效果：

【机会】：降低你下一次与此元素对应的神灵或超自然力量互动的检定难度减一。

【机会】+：如果你成功，你每以此法花费一个机会，就可以获得一个额外恩赐，每次激活每种恩惠只能获得一次

# 预测——rank1

激活：作为一个休息活动，你可以执行一次难度为2的宗教（空）检定，目标为一个角色来看到他们未来的一瞥

效果：如果你成功，选择一个下列征兆作为目标角色的近期未来：

苍龙预兆：目标在本场游戏中下一次做一个工匠技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

黑龟预兆：目标在本场游戏中下一次做一个学者技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

白虎预兆：目标在本场游戏中下一次做一个武术技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

朱鸟预兆：目标在本场游戏中下一次做一个社交技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

牛郎织女预兆：目标在本场游戏中下一次做一个交易技能组的检定时，可以增加一个保留的【轮骰】，结果被设定为【机会】【冲突】

如果你失败，gm来选择以上的一个预兆代替。目标直到下一场游戏前无法接受另一次预测

新机会效果：

【机会】：如果你失败，在第一个预兆之后你会接受额外一个预兆。在gm选择以外你也选择一个如上的预兆

# 茶会——rank2

激活：每场游戏一次，作为一个使用茶具的休息活动，你可以执行一次难度为2的表演（空）检定，目标为你自己和至多等于你文化技能等级的其他角色，这些角色也参加茶会。至多一名参加角色可以协助，如常提供效果。

效果：每个目标移除3点疲乏值，以及你每个额外成功额外移除一点疲乏

新机会效果：

【机会】：每个目标的下次检定难度降低一点

【机会】+：每个目标移除1点疲乏，以及你每花费两点机会额外移除一点疲乏

【机会】【机会】：如果你成功，每个目标获得一个虚空点

# 门栏屏障——rank1

激活：作为一个休息活动，你可以通过风、土、火、水、空来做一次难度为2的宗教检定，目标为距离0-1的一个门栏、大门或者其他入口

效果：如果你成功， 门栏会阻止特定类型的实体，基于你选择的轮类型。这个效果持续一个小时，以及每个额外成功额外持续一个小时。当对应类型的实体试图进入时，它必须通过难度为3的冥想（空）检定抵抗，否则将会被超自然屏障驱逐。不管它成功还是失败，你都会立刻知道它试图进入的意图。基于轮类型的特定实体类型如下：

风：灵体异界存在，例如灵或者鬼魂

土：活着的被污染存在

火：肉体形态的不死生物

水：野生动物

空：人类

新机会效果：

【机会】：你可以保护至多等于你额外成功数范围的区域。

【机会】：如果在过去二十四小时内，有任何你试图封锁门栏的生物通过了门栏，你会意识到他们的通过。
