# core

# 潜伏——rank1

激活：当你执行一个使用风属性的主动性或者攻击检定时，你可以以如下方式使用机会：

【机会】：如果你处在模糊地形内，人群中，或者其他隐蔽视线的地方，一个警戒值低于你的诡计技能等级的角色失去对你的视线

# 致命之刺——rank2

激活：作为一个攻击行动，你可以使用一剂毒药并执行一次难度为3的武术【风】检定，使用一把准备好的隐蔽属性的武器目标一个在此武器范围内的角色。

效果：如果你成功，将该毒药应用于你的武器。然后，目标承受物理伤害，数值等于你的武器致命值加上你的额外成功数

新机会效果：

风【机会】：移动距离1

风【机会】+：如果你成功，目标承受一次程度等于2加上你每以此法花费的机会数的致命一击

# 毒云——rank3

激活：作为一个攻击行动，你可以使用任意两剂单独的毒药并执行一次难度为2的医疗【风】检定，目标为每个距离0-2的其他角色。

效果：如果你成功，每个目标承受3点物理伤害，忽略他们的抵抗。这个伤害是由你带毒的武器带来的。

新机会效果：

风【机会】：移动距离1

风【机会】：直到本场景结束，距离0-2的区域被视为模糊和危险地形

风【机会】【机会】+：你每以此法花费两个机会，就有一个目标承受程度为2的致命一击

# 寂静一击——rank4

激活：作为一个攻击行动，你可以执行一次难度为4的武术【风】检定，使用一个准备好的武器目标一个武器范围内的角色

效果：如果你成功，目标承受物理伤害，数值等于武器的基础伤害并承受无声状态

如果你成功，距离目标范围0-2以外的角色不会注意到自己已经被下手了，此效果持续一加上额外成功数的轮数

新机会效果：

风【机会】+：如果你成功，目标将承受致命一击，程度等于你的武器的致命值，加上你以此法花费第一个以外的【机会】数。

# CoS

# 巧妙的不在场证明——rank3

激活：当你在休整场景使用诡计检定监视或者执行其他秘密行动时，你可以以如下方法花费【机会】：

风或水【机会】+：如果你成功，你被视作执行成功了一次修正场景的创作或设计检定，来改进（风）或者调整（水）一件物品，你每个以此方法花费的额外机会都能让这次改进/调整物品的检定获得一个额外机会。

# 狡猾的干扰——rank2

激活：作为在决斗或者交锋中的一个密谋行动，你可以执行一个难度为2的诡计检定，目标为一个在决斗或者交锋中的角色。

效果：执行这个行动算作欺骗。任何在观察或者参与决斗/交锋且警戒值大于等于2的角色可以识破这次欺骗。

如果你成功，在决斗或交锋的下一个回合，参与其中的角色交换他们的主动值。

新机会效果：

【机会】+：你每以此法花费一个机会，识破这次欺骗的最低警戒值加1

欺骗是不荣誉的！

记得欺骗直接违反了武士道准则，你的GM可以要求你丧失载誉值来执行这类技巧——例如狡猾的干扰。如果你被发现了欺骗，你或者你的盟友可能会输掉决斗，并可能因为你的行动遭受更大的惩罚。

# 欺诈之击——rank1

激活：当你在决斗或交锋中进行一次武术检定时，你可以用如下方式使用【机会】：

【机会】+：每个警戒值低于1+你以此法花费的机会数的观察者都认为你的攻击不会有意致命的。

# 如同鬼影——rank2

激活：作为一个移动和密谋动作，你可以执行一次风或水属性的难度为2的体能检定。

效果：如果你成功，你可以移动距离1，以及每两个额外成功额外距离1。这次移动忽略你穿过的全部地形效果。你也可以纵向移动或者跨越裂隙，只要这些移动按GM评判可以通过跳跃达到即可。

新机会效果：

【机会】【机会】：你的脚步声十分轻微，这次移动是完全寂静的。

# 切风踢——rank3

激活：作为一个使用一柄准备好的长柄武器攻击和移动行动，你可以执行一次难度为3的风属性武术【徒手】检定，使用徒手踢击数值，目标一个距离1-2的角色。

效果：如果你成功，目标承受物理伤害，数值等于你的徒手踢击基础伤害加上额外成功。

如果你成功，目标承受迷惑状态。

新机会效果：

风【机会】：结算成功或失败前，移动距离1

风【机会】【机会】：一个目标承受倒地状态

风【机会】【机会】+：你每以此法花费两个机会，可以多选择一个目标

# 寂静终结——rank3

激活：作为一个使用一个拥有落网特性的准备好的近战或者徒手武器的攻击和移动行动，你可以执行一次难度为3的武术【近战或徒手】检定，目标为一个在武器范围内的，体型0-2的活着的角色。如果你上个回合对同一目标成功的执行了同一动作，那么检定难度降低2，最低为1

效果：如果你成功，目标承受定身和无声状态，将目标拉至与你距离0。如果目标选择执行攻击行动，那么他只能使用武术【徒手】检定来执行目标为你的行动，并且目标难度增加1

新机会效果：

【机会】：如果你成功且目标是一个爪牙，目标承受无意识状态

【机会】+：如果你成功，你每以此法花费一个机会，目标将会承受2点疲乏以及2点冲突。如果目标因此效果承受了超过其耐力值的疲乏，那么目标将会承受无法行动状态之余，还承受无意识状态（正常情况疲乏值超过耐力值只会承受无法行动状态）。

【机会】【机会】：如果你成功，你和你的目标一起移动距离1，最终还是保持两者之间距离0

# 静止死亡——rank5

激活：当你执行一次使用忍术技巧的检定中，你可以用如下方式使用【机会】：

【机会】+：你每以此法花费一个机会，目标抵抗下一次致命一击的难度会增加1。此效果持续到你的下个回合结束。

# 沉或浮——rank2

激活：当你执行一次水或土属性的武术检定时，你可以用如下方式使用【机会】：

水或土【机会】+：从你身上移除存在的迷惑或者倒地状态

水或土【机会】：如果本检定会让你承受迷惑或者倒地状态，那么你不会承受这两种状态（任一或全部）

# 你的属于我——rank2

激活：作为一次使用准备好的且没有笨重特性的武器的移动和攻击行动，你可以执行一次难度为3的火属性武术【近战】检定，目标为一个距离你0-1的角色。

效果：如果你成功，目标承受等于你武器基础伤害的物理伤害。

如果你成功且你的目标处于炫目状态，你可以从你的目标身上偷到一个物品；这个物品的大小和位置需要叙事上合理，并且取决于GM的允许。如果你选择偷取准备好的武器，目标可以执行难度为3的体能检定（风5，水2）来抵抗。

新机会效果：

火【机会】：目标承受炫目状态

火【机会】+：移动距离1，以及你每以此法花费额外两个【机会】可以再移动1

火【机会】【机会】+：你每以此法花费两个【机会】，你可以额外偷走一件物品。
