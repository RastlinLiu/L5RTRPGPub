# core

# 死亡之触——rank4

激活：作为一个攻击和支援行动，你可以执行一次难度为3的武术【徒手】（空）检定来增强你的徒手打击。当你执行此行动时，你可以选择一个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你使用拳打或者脚踢来执行攻击行动时，目标无法防御该伤害，除非花费1点虚空点才可以防御该伤害。

爆发效果：如果你有三个或更多额外成功，该目标承受眩目和迷惑状态，并且必须通过难度为4的体能检定来抵抗，否则将承受昏迷状态。如果你有四个或者更多额外成功且目标是爪牙npc，那么他将被直接杀死。

# 平息元素——rank3

激活：作为一个密谋和支援行动，你可以执行一次难度为4的冥想检定（空）。当你执行此行动时，选择：风，土，火，水之一。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你被一个你选定的轮的检定选为目标时，你可以选择这个检定中的一个保留的骰子，将这个骰子设置为空白面。在此气功激活期间，当你被一个你选定的轮的检定选为目标时，你可以花费1虚空点来选择等于你冥想技能等级数量的额外保留骰子，将这些骰子设置为空白面。

爆发效果：如果你有两个或更多额外成功，0到3范围内的地形被视为失衡【元素】直到此场景结束。

# 接触空龙——rank5

激活：作为一个支援行动，你可以执行一个难度为4的冥想（空）检定来将宇宙之力吸入你的体内。

增强效果：如果你成功，此气功激活。在此气功激活期间，你可以将虚空检定中的【机会】视为任意元素花费。在此气功激活期间，你可以将风、土、火、水检定中的机会视为空检定中的机会花费。

爆发效果：如果你有两个或更多额外成功，你获得1虚空点。每个场景你只能以此法获得一个虚空点。

# 无锋剑道——rank5

激活：作为一个攻击和支援行动，你可以执行一个难度为3的冥想检定来强化你的徒手和即兴武器攻击。当你执行此行动时，你可以选择任意数量距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，你的徒手和即兴武器基础伤害和致命性被视为你的武术【徒手】+你的空轮数值。此外，它们也被视为具有持久和锐锋属性。

爆发效果：如果你有两个或更多额外成功，每个目标承受一次致命性等于你的空轮+武术【徒手】数值的致命一击。
