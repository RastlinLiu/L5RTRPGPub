# 风气功

# core

# 风之拳——rank1

激活：作为一个攻击和支援行动，你可以进行一次难度为1的武术【徒手】（风）检定来增强你的徒手攻击。当你进行此行动时，你可以选择一个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，你的徒手拳击和踢击距离视为你的风轮数值。

爆发效果：如果你有两个或更多额外成功，你可以对你的目标造成等于你风轮数值的物理伤害，你的目标必须通过难度为4的体能检定（土2，火5），否则会被向与你相反的方向推等于检定差额的距离。

# 腾云而上——rank2

激活：作为一个移动和激活行动，你可以做一个难度为2的冥想（风）检定来在一片云上移动

增强效果：在此气功激活期间，当你移动1或更多范围，你可以移动的范围+1，而且你可以选择其中一个距离为水平或者垂直方向。额外的，你可以忽略移动过程中的地形效果。在此气功激活期间，如果你坠落了，你视为从距离为1的高度坠落的。

爆发效果：如果你有一个或更多额外成功，你立刻移动一距离，此外你每有一个额外成功再移动一距离。你可以选择水平或者垂直方向进行此移动，与此同时，你移动时忽略负面地形效果。

# 伟大寂静——rank1

激活：作为一个密谋和支援行动，你可以激活此气功，你可以执行一个难度为1的宗教（风）检定来制造一片围绕着你的寂静区域。

增强效果：如果你成功，此气功激活。在此气功激活期间，增加1目标你的密谋行动+你的命令技能等级的难度。

爆发效果：如果你有两个或更多额外成功，范围0-3的角色承受无声状态且无法听到声音。这个效果持续到你的下个回合开始。

# 垂柳之道——rank3

激活：作为一个移动和激活行动，你可以做一个难度为3的冥想（风）检定来调整你的防御以适应你的对手的打击节奏，让你自身更难被击中。

强化效果：如果你成功，此气功激活。在此气功激活期间，当一个角色进行一次目标为你的攻击行动检定时，每有一个【大成功】被留下了，就降低你承受的任意伤害2点。

爆发效果：如果你有两个或更多额外成功，提升以你为目标的攻击行动难度，提升的难度等于你的冥想技能等级，直到你的下个回合开始。
