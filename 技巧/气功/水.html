# core

# 冻结命脉——rank2

激活：作为一个攻击和支援行动，你可以执行一次难度为2的武术【徒手】（水）检定来增强你的徒手打击。当你激活此技巧时，你可以选择一个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，在一个角色防御了你使用拳击或脚踢的攻击行动造成的伤害后，该角色承受定身效果，否则需要承受三点疲乏值

爆发效果：如果你有两个或更多额外成功，目标承受等于你的水轮数值的超自然伤害并忽略他的抵抗，同时承受定身状态。如果你有四个或更多额外成功，你的目标承受一次致命一击，程度等于你的水轮+医疗技能等级。

# 气功防御——rank1

激活：作为一个移动和支援行动，你可以执行一次难度为1的冥想（水）检定。当你激活此技巧时，你可以选择一个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你执行平静呼吸行动时，你可以移除两点疲乏而非一点。在此气功激活期间，你执行一个行动后，你可以花费一个虚空点来移除一个范围0-1内的另一个角色的疲乏，其值等于你的医疗技能等级。

爆发效果：如果你有两个或更多额外成功，你的目标移除等于你的医疗技能等级+额外成功数的疲乏，目标在下个场景前不能被再次以此法影响。

# 驾驭水龙——rank3

激活：作为一个移动和支援行动，你可以执行一个难度为3的冥想（水）检定来增强对超自然效果的抵抗。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你被一个祈愿、邪术或气功的检定选为目标时，增加该检定的难度，数值为你的冥想技能等级。按照gm的决定，在此气功激活时，你可以穿越超自然屏障或者守卫而不触发效果或通知其主人。

爆发效果：如果你有两个或更多额外成功，移除你的其他全部来自于超自然来源的持续效果（例如祈愿、邪术或气功）效果。如果你有四个或更多额外成功，移除范围0-2内全部角色的其他全部来自于超自然来源的持续效果（例如祈愿、邪术或气功）效果以代替。

# 水之拳——rank1

激活：作为一个攻击和支援行动，你可以执行一次难度为2的武术【徒手】（水）检定来增强你的徒手打击。当你执行此行动时，你可以选择一个距离0-3且与你共享一个接触表面的角色作为爆发效果目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，你使用拳击或者脚踢的徒手攻击检定忽略两倍于你的水轮的物理抵抗。在此气功激活期间，你可以通过你的目标接触的坚固平面来间接执行拳打或者脚踢的徒手攻击。只要你和你的目标都接触着同一个坚固表面（例如墙、地面、或者水体），互相之间距离0到你的水轮之间，那么你的目标被视为在你使用拳打或者脚踢的徒手攻击距离内。

爆发效果：如果你成功，你对你的目标造成等于你水轮数值的物理伤害。

# 海泡之道——rank1

激活：作为一个移动和支援行动，你可以执行一次难度为1的冥想（水）检定

增强效果：如果你成功，此气功激活。在此气功激活期间，你可以如同在平地行走一般行于水面之上。

爆发效果：如果你有两个或更多额外成功，移除你的定身和倒地状态。然后，你可以切换至任意架势并移动至多范围2
