# 火气功

# core

# 破势掌——rank1

激活：作为一个攻击和支援行动，你可以执行一次难度为1的武术【徒手】（火）检定来增强你的徒手打击。当你进行此行动时，你可以选择一个距离0-1的物体作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你成功执行一次针对目标的武术【徒手】检定时，选择位于目标位置的一件穿着的护甲或者一件准备好的武器，它除非目标承受2点疲乏，否则该物体获得损坏特性。

爆发效果：如果你有两个或更多额外成功，目标物体获得损坏特性，在gm的判断下，该效果也可以粉碎环境中的坚硬物体，例如木门、土墙或者树木。如果你有四个或者更多额外成功，被选择的物体获得毁灭特性，这个效果可以用来粉碎环境内的强化物体，例如铁门或者石墙。

# 传导火龙——rank2

激活：作为一个攻击和支援行动，你可以执行一次难度为2的冥想（火）检定来点燃你的气之火。当你进行此行动时，你可以选择每个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，你自动通过抵抗极端天气的全部检定，并且获得等于你火轮数值的额外成功。在此气功激活期间，如果你会受到物理伤害或超自然的火焰伤害（例如火属性的祈愿或气功），你可以花费1个虚空点来防止此伤害并移除此伤害数量的疲乏。

爆发效果：如果你有两个或更多额外成功，每个目标必须通过难度为4的体能检定（风5水2）或者承受等于你的火轮数值的物理伤害，以及燃烧状态。这片区域内的易燃物品将被点燃，在gm的判断下转变为危险地形。

# 火焰拳——rank1

激活：作为一个攻击和支援行动，你可以执行一次难度为1的武术【徒手】（火）检定来增强你的徒手打击。当你进行此行动时，你可以选择一个距离0-1的角色作为爆发效果的目标。

强化效果：如果你成功，此气功激活。在此气功激活期间，你的徒手拳击和踢击致命性为你的火轮数值，且它们的伤害类型为超自然伤害。

爆发效果：如果你有两个或更多额外成功，你的目标承受等于你的火轮数值的物理伤害，且目标必须通过难度为4的体能检定（风5，水2），否则会承受眩目状态。

# 身如铁砧——rank1

激活：作为一个移动和支援行动，你可以执行一次冥想（火）检定来准备你的防御。当你进行此行动时，你可以选择一个距离0-2的角色作为爆发效果的目标。

强化效果：如果你成功，此气功激活。在此气功激活期间，当你因一个攻击行动承受伤害时，攻击你的角色承受等于你火轮数值的超自然伤害并承受眩目状态。

爆发效果：如果你有两个或更多额外成功，在你防御目标造成的伤害前，将伤害数值降低你的火轮数值。如果你成功防御了伤害，目标用的一个武器承受损坏特性，而目标承受烧灼状态。

# 坠星之道——rank3

激活：作为一个攻击和支援行动，你可以执行一次难度为3的武术【徒手】（火）检定。当你激活此技巧时，你可以选择一个距离0-2的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，当你对一个眩目目标进行拳击或脚踢攻击时，将你的火轮数值增加到伤害上。

爆发效果：如果你有三个或更多额外成功，该目标承受等于你火轮+体能技能数值的超自然伤害。目标必须通过难度为5的体能检定（风6，水3）否则将会承受烧灼或眩目状态。
