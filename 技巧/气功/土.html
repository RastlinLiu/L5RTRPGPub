# core

# 净化灵魂——rank1

激活：作为一个支援行动，你可以执行一个难度1的宗教检定（土）来增强你的精神复原力。当你执行此行动时，你可以选择一个0-1距离的其他角色来作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，降低你全部抵抗中毒、疾病、邪术、污秽地形以及他界存在和感染者的效果检定难度，降低难度数值为你的土轮

爆发效果：如果你有两个或更多额外成功，移除你的对象一个如下状态：折磨、流血、眩目、迷惑、濒死、疲乏、沉醉。

# 大地无需眼睛——rank1

激活：作为一个支援行动，你可以激活此气功。当你如此做时，你可以做一个难度为1的冥想（土）检定来获得你周围的了解

增强效果：如果你成功，此气功激活。在此气功激活期间，你可以利用大地的震动来“看见”等于全方向上你土轮数值距离的范围。在此气功激活期间，将你的土轮数值加到你的警戒值上。

爆发效果：如果你有三个或更多额外成功，你立刻了解你土轮数值+你的额外成功数距离范围内所有着地的生物和物体。

# 土之拳——rank1

激活：作为一个攻击和支援行动，你可以进行一次难度为1的武术【徒手】（土）检定来增强你的徒手攻击。当你进行此行动时，你可以选择一个距离0-1的角色作为爆发效果的目标。

增强效果：在此气功激活期间，将你的徒手拳击和踢击基础伤害增加你的土轮数值。在此气功激活期间，你的物理抗性提升1点

爆发效果：如果你有两个或更多额外成功，你可以对你的目标造成等于你土轮数值的物理伤害，并且你的对象必须通过难度为4的体能检定（风2，水5），否则会承受倒地状态。

# 紧握地龙——rank3

激活：作为一个移动和支援行动，你可以进行一次难度为2的冥想（土）检定来保护你免受伤害。

增强效果：如果你成功，此气功激活。在此气功激活期间，降低你受到的来自物理来源的致命一击程度你土轮的数值。在此气功激活期间，在一个角色对你进行一个攻击行动后，你可以花费1点虚空点来让他承受等于你体能技能的疲乏值。

爆发效果：如果你有三个或更多额外成功，将你承受的全部物理伤害降低为1。任意角色对你造成物理伤害后，他们承受倒地状态。这个效果持续到你的下个回合结束。

# 地震之道——rank2

激活：作为一个攻击和支援行动，你可以进行一次难度为2的武术【徒手】（土）检定来控制你周围的大地。当你进行此行动时，你可以选择任意数量个距离0-1的角色作为爆发效果的目标。

增强效果：如果你成功，此气功激活。在此气功激活期间，在你进行一次进攻或支援行动后，或者防御伤害后，你可以选择一个其他距离0-2的角色，他必须通过难度4的体能检定（风2，水5），否则将会承受等于你土轮数值的物理伤害，还会承受倒地状态。

爆发效果：如果你有两个或更多额外成功，每个目标承受等于你土轮数值的物理伤害。每个倒地状态的目标承受你的土轮+你的额外成功数值的物理伤害，并承受定身状态以代替。
