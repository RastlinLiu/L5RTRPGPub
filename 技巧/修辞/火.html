# 火修辞

火修辞专注于最直接的挑衅，立刻引起狂热的反应。愤怒、喜悦、绝望和对正义的强烈渴望都是使用火修辞的人的工具。有了它，演讲者可以照亮不公正的现象，或者让人们的心因骄傲燃烧起来，但它也可能让人们没法看到更广泛的情况或者行为的后果。

# core

# 虚张声势——rank4

激活：作为一个密谋行动，你可以执行一次游戏（火）或者表演（火）检定，目标为场景内的任意数量角色，难度为目标中最高的警戒值。

效果：如果你成功，每个目标都会以你的荣耀比实际值高10或者低10，以及你每个额外成功再多10来行动（最高为100最低为0）。如果你做了什么事情会让你的荣耀受到质疑，任何角色可以通过难度为4的情绪检定（风5，水2）来确定你的实际荣耀值。这个效果持续到本场景结束。

新机会效果：

火【机会】：选择一个场景中的角色。了解他们的荣耀值比你的实际荣耀值高或者低

# 眩目表演——rank3

激活：当你执行一次工匠技能组检定（火），游戏检定（火），表演检定（火），你可以以如下方法使用【机会】：

火【机会】+：本场景内你下一次获得荣耀奖励时，获得的奖励值加一。如果这个场景内有更高地位的人，那么你每以此法花费一个机会额外增加一点奖励值。

# 煽风点火——rank2

激活：当你执行一次社交技能检定（火）目标为一个或者更多角色时，你可以以如下方法使用【机会】：

火【机会】+：你每以此法花费一个机会，可以让一个目标获得眩目状态

火【机会】【机会】+：你每以此法花费两个机会，可以让一个目标获得暴怒状态

# 闪电突袭——rank2

激活：每个场景一次，作为一次攻击和移动行动，你可以执行一次难度为3的命令（火）检定，目标为场景内的任意角色（或者在大规模战斗中的一个大队）

效果：如果你在一次小规模冲突中成功，每个目标增加等于你火轮+额外成功数的主动值，直到下一轮开始为止。 如果你在大规模战斗中成功，一个敌人领袖的大队将会承受等于你火轮+额外成功数的恐慌值

新机会效果：

火【机会】：在一次小规模冲突中，立刻执行一次打击行动

火【机会】：在一次大规模战斗中，立刻执行一次突袭行动，目标为敌人的大队领袖

# 振奋战呼——rank3

激活：作为一个支援行动，你可以执行一次难度为2的命令检定（火）或者表演检定（火），目标为这个场景内的任意数量角色（或者大规模战斗中的一个大队）

效果：如果你成功，目标会因为你的言辞而感到振奋和兴奋。在第五步：选择留下的骰子时，每个目标都会增加一个被设定为【机会】【冲突】的【轮骰】，这个效果会持续到你的下个回合结束

新机会效果：

火【机会】+：在小规模冲突中，你每以此法花费一个【机会】，场景内的每个敌人承受一点冲突。

火【机会】+：在大规模冲突中，你每以此法花费一个【机会】，敌人军队承受一点恐慌值。

# 灼伤伤口——rank5

激活：当你执行一次社交技能（火）检定，目标为一个或多个角色时，你可以以如下方法使用【机会】：

火【机会】+：你每以此法花费一个机会，就可以选择一个你的一个目标的已知劣势。这个劣势效果应用于所有该目标的检定，直到此场景结束。

# 敏感的分心——rank1

激活：当你执行一次社交技能（火）检定，目标为一个或多个角色时，你可以以如下方法使用【机会】：

火【机会】+：当与其他角色交互时，你每以此法花费一个机会，目标的警戒值视为少1；当与你交互时，你每以此法花费一个机会，目标的警戒值视为多1。这个效果持续到本场景结束。

# 燃烧的余烬——rank1

激活：当你执行一次社交技能（火）检定，目标为一个或多个角色时，你可以以如下方法使用【机会】：

火【机会】+：选择一个目标和他们已知的卓越优势，数量为你以此法花费的机会数。直到本场景结束，当此优势应用于检定，目标可以重置至多3个骰子（代替2个）

# 谎言中燃烧的真相——rank1

激活：当你执行一次社交技能（火）检定来评估一个角色的故事时，你可以以如下方法使用【机会】：

火【机会】：如果该角色的故事依赖单一的说法，你来决定这个说法是什么，以及你需要做什么来验证或者否认它。
