# 水修辞

水修辞专注于内心而非思想，号召听者的情绪来引导他们，并不由分说的被卷走。友情、羞耻和内疚，以及物质需求和基本欲望，都是水修辞的领域。使用这些技巧，一个熟练的朝臣可以把最为严峻的敌人以最小阻力的途径吸引到谈判桌上。

# 皆为玩笑——rank1

激活：当你执行一次商业（水）、礼节（水）或者表演（水）检定时，你可以以此法花费【机会】：

水【机会】+：如果你丧失载誉值作为检定的一部分来说一些粗鲁或者吸引你的目标的基本欲望（贪婪、性欲、懒惰等），你每以此法花费一个机会，可以重新获得一点载誉值（见37页的丧失社交属性）

# 振奋人心的到来——rank5

激活：作为一个密谋行动，你可以执行一次礼节（水）检定，目标为至多等于你水轮数值个角色。检定难度为目标中最高的警戒值。

效果：如果你成功，你可以优雅的加入或者离开谈话。你不需要丧失荣耀值或者载誉值来打断、突然离开或者抢夺他人的话题，无论参与者的地位如何

# 潮起潮落——rank3

激活：作为一个密谋行动或者支援行动，你可以执行一次礼节（水）、命令（水）或者游戏（水）检定，目标为一个角色。检定难度为目标的警戒值。

效果：如果你成功（且选择为密谋行动），选择一个目标的你知道的优势，无论何时应用它时，反转它，此效果持续到本场景结束。

如果你成功（且选择为支援行动），选择一个目标的你知道的劣势，无论何时应用它时，反转它，此效果持续到本场景结束。

新机会效果：

水【机会】：选择一个额外的优势或者劣势应用效果

水【机会】【机会】+：你每以此法花费两个机会，可以选择额外一个目标，其警戒值必须等于或者小于第一个目标。

# 皇室举止——rank4

激活：作为一个密谋行动，你可以执行一次命令（水）或者表演（水）检定，目标为场景内任意数量个角色，检定难度为你的目标中最高的警戒值。

效果：如果你成功，每个你的目标都将你视为地位高或低10点加上每个额外成功额外1点（至多为100点，至少为0点）。如果你做了任何可能会引起你的地位被怀疑的事情，任意角色可以通过难度为4的共情检定（土2，火5）来确定你的实际地位值。这个效果持续到本场景结束。

新机会效果：

水【机会】：选择一个场景内的角色，了解他的地位比你高还是低（与你的实际地位相比）

# 浅水区——rank1

激活：当你执行一次社交检定（水），目标一个角色时，你可以以此法花费【机会】：

水【机会】：了解目标渴望的一件物品或者一个世俗体验

水【机会】【机会】：了解目标的人情

# 狡猾的行动——rank2

激活：作为一个移动和支援行动，你可以执行一次难度为2的命令（水）检定，目标为场景内的任意数量友方角色（或者在大规模战斗中你的大队）。这个技巧只能被用于小规模战斗或者大规模战斗。

效果：如果你在小规模战斗中成功了，选择一个包含地形的位置，每个目标都可以立刻向该位置移动距离1，目标该地形内的目标角色的攻击检定将该地形视为模糊特性。这个效果持续到本场景结束。

如果你在大规模战斗中成功了，选择一片你能看到的地形，如果该地形没有被占据的话，你和你的大队可以立刻移动到该地形占据它，你在该地形内期间，目标你或者你的大队的攻击检定将该地形视为模糊特性。这个效果持续到本场景结束。

新机会效果：

水【机会】：每个目标都可以让目标地形的模糊效果更好；目标他们的攻击检定难度+2而非+1，这个效果持续到本场景结束。

# 贸易的支流——rank2

激活：每个场景一次，作为一个密谋和支援行动，你可以执行一次难度为1的商业（水）技能检定来找到一个你之前选定获取的物品。从叙事的角度看，你在最后的机会获取了这件礼物（或者安排了一名仆人去取得这件物品），不过你现在才将其展现出来。

效果：你获得了一件小于等于1加上你的额外成功数稀有度的物品。你必须仍然花费费用来获得它。

新机会效果：

水【机会】+：你每以此法花费一个机会，就可以降低你取得此物必须的花费中的四分之一（你至少要花费原花费的四分之一）。

# 欲望之井——rank1

激活：当你执行一次礼节（水）检定来将一件礼物提供给目标时，你可以以此法花费【机会】：

水【机会】：为了拒绝此礼物，目标必须丧失等于你水轮的荣耀值。此外，如果该礼物是目标所渴望的，目标还必须承受等于其稀有度的冲突。如果目标接受了此礼物，降低你下一次对该目标的社交技能检定难度，数值为你的礼物的稀有度（难度至少为1）
