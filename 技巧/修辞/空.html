# 空修辞

空修辞围绕着理解和表达宇宙的内在奥秘来帮助自己或其他人走向启迪、无私的目标或者其他世界的终结。

有些空修辞局限于指定的氏族，这列举在标题处。这一点意味着只有属于该氏族的角色可以学习此修辞。

# core

# 诸艺皆一——rank3

激活：作为一个支援行动，你可以执行一次难度为4的武术（空）、游戏（空）或者表演（空）检定来迅速吸收该技能所涉及的新活动的信息

效果：如果你成功，那么你立刻凭直觉了解如何去持有一把新类型的武器、进行一种新的游戏、使用一种新乐器，或者表演一个新的歌曲、舞蹈以及其他任务。你不会应用任何可能与你接触新事物相关的难度加值。

新机会效果：

空【机会】：选择一个轮。降低你的下一次应用该轮进行此类型的检定难度1点。这个效果持续到你的下一个回合结束（或者在一个叙事场景中持续到场景结束）

# 武士的命运——rank4

激活：作为一个行动，你可以执行一次难度为5的命令（空）检定，目标为任意数量的角色（或者在大规模战斗中目标你的大队）。

效果：如果你在小规模冲突中成功，每个目标都会都会忽略程度小于你的空轮值加上额外成功数的致命一击。这个效果持续到本场景结束（在受伤的时候生效）

如果你在大规模战斗中成功，你的部队视为拥有无畏军队能力，效果如下：

无畏军队：每轮结束时，你的部队移除等于你荣耀等级的恐慌值。这个效果持续到本场景结束。

# 朝臣的决心——rank1

激活：每个场景一次作为一个支援行动，你可以花费一个虚空点来恢复冲突

效果：移除等于你荣耀等级的冲突

# 道寺女士的裁决——仙鹤rank2

激活：每次游戏流程一次，你可以执行一次礼节（空）检定，目标至多等于你学派等级的角色数。难度等于你的目标中最高的地位（等级）

效果：如果你成功，你的目标不能对你执行攻击行动。这个效果持续一轮，以及每个额外成功额外一轮，或者直到你执行一次攻击行动。

新机会效果：

空【机会】+：你每以此法花费一个机会，你可以选择地位比你低的角色作为目标

空【机会】【机会】：你的目标也不能对你执行密谋行动。

# 新条女士的速度——奇麟rank2

激活：作为一个休整行动，如果你有可用的战马，你可以执行一次难度为2的求生（空）检定来以无以伦比的迅捷达到一个目的地。

效果：如果你成功，你可以花费平时的一半时间到达一个目的地。如果你想的话，一个角色可以与你一并骑行。

新机会效果：

空【机会】：你可以带领一群有自己坐骑的角色自愿的一起前行，角色数量至多等于你的学派等级。

# 阿古渡大人的怒吼——雄狮rank2

激活：每次游戏流程一次作为一个行动，你可以执行一次难度为1的命令（空）检定，目标为等于学派等级的范围内的每个敌对角色。

效果：如果你成功，每个目标都会承受眩目效果。

新机会效果：

空【机会】【机会】+：你每以此法花费两个机会，可以有一个距离执行此行动0-3的友方角色可以立刻对一个眩目角色执行一次打击行动。

# 间由使大人的低语——毒蝎rank2

激活：每次游戏流程一次作为一个行动，在一个叙事场景中或者作为一个休整行动，你可以执行一次难度为2的诡计（空）检定来发现一个能为你提供某个物体信息的线人。

效果：如果你成功，你发现一名可以为你提供你选择的主题信息的线人，如果该线人的简介需要被提供，使用一个合适的npc模版

新机会效果：

空【机会】：如果你成功，选择一个技能。线人拥有该技能，技能等级等于你的学派等级，并且可以执行该技能的检定来协助你，直到本场景结束。

# 富坚大人的洞察——潜龙rank2

激活：每次游戏流程作为一个行动，你可以执行一次难度为2的冥想（空）检定来对于你面前的困境寻求宇宙的智慧。

效果：如果你成功，你获得一个简短的视觉或者听到富坚的声音，其中提供了你可能可以进行的行动的一点提示（gm应当提供）。提示不是完整答案，但它应当帮助你接近解决方案或者至少制定行动计划。

新机会效果：

空【机会】：降低你面对此困难的第一个检定的难度，降低难度数为你的学派等级，难度至少为1

# 唤醒灵魂——rank5

激活：当你执行社交技能检定（空）或宗教（空）检定，目标为一个或更多角色时，你可以以如下方式使用机会：

空【机会】+：你每以此法花费一个机会，可以移除一个目标的一个如下状态：折磨、暴怒、精疲力尽、醉酒或者无意识。

# 天国敬畏——帝国rank4

激活：每场景一次，作为一次攻击和密谋行动，你可以执行一次命令【空】检定，目标场景内的任意数量角色。检定难度为你目标中最高的地位值。

效果： 如果你成功，每个目标遭受无声状态。
