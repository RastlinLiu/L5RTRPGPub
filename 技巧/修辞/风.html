# 风修辞

风修辞是间接的、试探性的，有时候是挑逗的。比起其他的说辞，这种谈话风格往往围绕着谈话中设置微妙的陷阱，让别人陷入其中，然后再利用他们的价值。因此风淑仪倾向于以细节为中心而非以大局为重；他们控制着谈话范围，以避开相反的证据或者逻辑论证。

# core

# 工匠的评价——rank2

激活：当你通过工匠技能（风）、表演（风）或者游戏（风）来接触一个目标或者表演检定时，你可以以如下方法使用【机会】：

风【机会】：选择一个轮，知晓制造者或者表演者的该轮值以及所有与此轮相关的不是诅咒类型的优势和劣势

风【机会】：如果制造者或者表演者是一名npc，那么了解他的风度

# 暴风而折——rank5

激活：当你执行一次目标为其他角色的社交技能（风）的检定时，你可以以如下方法使用【机会】：

风【机会】：选择一个优势或劣势，目标会相信你拥有此优势或劣势，而你不需要必须花费明显的努力或者直接说明（拥有此优势或劣势）。进一步的，目标认为你不知道你已经泄漏了自己拥有此优势或劣势的情况。如果你做了任何可能会引起这种错误观念被质疑的行为，任何角色可以通过难度为5的情绪检定（土6，火3）来抵抗，察觉到此优势或劣势不是真的。这个效果持续到目标获得了无可辩驳的证据，或者在你表现出瑕疵的情况下成功的进行了抵抗检定为止。

# 节奏——rank1

激活：当你执行一次社交技能鉴定（风）时，你可以以如下方法使用【机会】：

风【机会】：向一个观察你的拥有此技巧的角色传达一条秘密的信息，该信息改变了你对外说的话的含义（或者根本与你说的话无关）

风【机会】【机会】：向一个观察你的没有拥有此技巧的角色传达一条秘密的信息，该信息改变了你对外说的话的含义（或者根本与你说的话无关）

# 伪造开局——rank2

激活：作为一个移动和密谋行动，你可以执行一次表演检定（风），目标为一个在场景内观察你的角色。检定难度为目标的警戒值。该技巧只能被用于遭遇战和大规模战斗冲突中。

效果：如果你成功，你可以降低下一次对目标的攻击检定难度（在大规模战斗中为目标所在的大队），其数值为1+额外成功数。这个效果持续到你的下个回合结束。

新机会效果：

风【机会】+：你每以此法花费一个【机会】，就可以选择一名警戒值等于或低于原有目标的角色作为目标

风【机会】+：在大规模战斗中，增加目标为你所在的大队的攻击检定的难度，其数值为1+每以此法花费两个【机会】额外1

# 捕食弱者——rank2

激活：作为一个密谋行动，你可以执行一次难度为1的礼节检定（风），目标为一名妥协的角色。

效果：如果你成功，你的目标必须选择如下一项：

基于现有的情况，以他选择的举止方式立刻揭面

丧失等于你的风轮+额外成功数的荣耀值来离开场景

新机会效果：

风【机会】+：如果你的目标选择揭面，你每以此法花费一个机会，就可以了解一个他的劣势

风【机会】【机会】：如果目标选择离开场景，你会从他们离开的方向或者听到人群的耳语来获得他离开的目的地。

# 瑟瑟之叶——rank1

激活：当你执行一次社交技能鉴定（风）来散播谣言时，你可以以如下方式使用【机会】：

风【机会】+：你每以此法花费一个机会，增加任何追踪谣言源头至你的检定难度2点

# 双侧风吹——rank3

激活：当你执行一次礼节或者表演（风）检定，目标为另一个角色时，你可以以如下方法使用【机会】：

风【机会】+：目标角色下一次获得荣耀奖励时，你每以此法花费一个机会，该角色获得的荣耀值+1

风【机会】+：目标角色下一次丧失荣耀时，你每以此法花费一个机会，该角色丧失的荣耀值+1

# 朝堂低语——rank1

激活：作为一个密谋行动，你可以执行一次礼节（风）检定，目标为场景内的全部其他角色。检定难度等于目标中最低的警戒值

效果：如果你成功，你创造了一个如同风暴一般席卷场景的谣言。每个在场景内的角色都听到了，所有警戒值比你的额外成功数低的角色甚至于开始有些相信这个谣言了。如果在一个阴谋场景内，你的社交目标是散播谣言，你获得的修辞分为你的风轮+你的额外成功数。

# 狼之提议——rank4

激活：作为一个密谋行动，你可以执行一次礼节（风）或表演（风）检定，目标为此场景内的任意数量角色。检定难度为目标之中最高的警戒值。

效果：如果你成功，每个目标都会视作你的载誉值比实际值高10点或者低10点，以及你每有一个额外成功都会视为载誉值额外高或低10点（最高为100，最低为0）。如果你做了任何事情导致你看起来的载誉值被怀疑，任何目标都必须通过难度为4的情绪检定（土5，火2）抵抗来决定你的实际载誉属性。这个效果持续到此场景结束。

新机会效果：

风【机会】：选择一名场景内的角色，了解他的载誉值比你高还是低（以你的实际值为准）

# 困于蛛网——问者rank3

激活：每游戏流程一次，在一次叙事场景或者休整场景中，你可以执行一次难度为2的诡计（风）检定来招募一个NPC加入问者阴谋

效果：如果你成功，你可以招募一个爪牙或者之前没有涉及的NPC（在GM裁量下）作为问者的资产，把他记录在你的关系上。从现在开始，每个游戏流程你可以用合理的理由联系他一次。该NPC会在单一场景内帮助你，尽最大可能来完成安排给他的任务，或者进行一个休整行动。如果人物数据需要使用，则由GM决定使用一个合适的NPC数据。

新机会效果：

风【机会】：如果你成功，选择一个技能。NPC具有等同于你的学派等级的技能等级，并可以用执行该技能检定来协助你。
