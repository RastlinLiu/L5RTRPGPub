# 土修辞

土修辞的焦点是通过支持、期待和必要时压力的组合来建立他人的印象。使用这种修辞的人物可能会以一种有分寸的速度说话，仔细权衡语言并从基础发展论点

# 最重要的礼貌——rank2

激活：作为一个密谋行动，你可以执行一次命令或者礼节（土）检定，目标为一个角色，以荣誉之名来保护某角色个体。检定难度等于目标的警戒值

效果：如果你成功，目标必须丧失等于你土轮数值+额外成功数的载誉值并承受该数值的冲突值来进行一次目标为被保护个体的攻击或者密谋行动。这个效果持续到此场景结束，或者目标进行一次攻击或密谋行动为止。

新机会效果：

土【机会】+：你每以此法花费一个机会，你可以选择一个低于首个目标警戒值的角色作为额外目标

# 诚实评价——rank1

激活：作为一个支援行动，你可以执行一次难度为2的礼节检定（土）来评估一个角色的弱点

效果：如果你成功，选择一个目标已知的劣势。目标在本场景结束前，检定中忽略此劣势。

新机会效果：

土【机会】+：你每以此法花费一个机会，就可以额外选择一个劣势

土【机会】【机会】：降低目标下一次使用指定劣势所相关的轮的检定难度2点

# 平静之柱——rank4

激活：作为一次密谋和支援行动，你可以执行一次命令（土）检定，目标为至多等于你土轮乘以你的学派等级数量的角色。难度等于目标中最高的冲突值

效果：如果你成功目标全部参与冲突的领袖，你可以降低冲突等级一级（例如从大规模战斗降低为小规模战斗，小规模战斗降低为决斗，决斗降低为阴谋）

新机会效果：

土【机会】【机会】+：你每以此法花费两个机会，可以额外降低冲突等级一级

# 勇气的试金石——rank3

激活：作为一个支援行动，你可以执行一次命令检定（土），目标为任意数量角色（或者在大型战斗中你所在的大队）

效果：如果你在阴谋、决斗或者小规模冲突中成功，每个目标的妥协值增加等于你土轮数值。这个效果持续到本场景结束。

如果你在大规模战斗中成功，为你的军队纪律性增加等于你土轮数值+额外成功数值。这个效果持续到本场景结束。

新机会效果：

土【机会】【机会】：在阴谋、决斗或者小规模冲突中，每个目标移除等于你土轮的冲突

土【机会】【机会】：在大规模冲突中，你的军队移除等于你土轮数值的恐慌

# 不可动摇的和平之手——rank5

激活：每次游戏流程中一次，作为一个在决斗冲突或者小规模冲突中的密谋行动，你可以执行一次难度为5的命令检定（土），目标为场景中每个角色。

效果：如果你成功，此场景变为你和其他主要角色的阴谋冲突。这个效果持续一轮，以及每个额外成功持续额外一轮。如果你在时间完成前未达成一个和平的解决方案，那么场景回到之前的情况

新机会效果：

土【机会】：你也可以在大规模战斗或者其他形式冲突中使用此行动（除了阴谋冲突）
