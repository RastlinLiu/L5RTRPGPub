# 物品样式

# 垣田样式

垣田剑匠以对武器的精雕细琢闻名于绿岩大陆。尽管在一代人的时间中，也不一定有一柄武士刀值得被冠以“垣田之剑”的称号，不过他们打造的所有武器都在不损害武器坚固性的前提下，拥有更加精致锋利的剑刃。

拥有垣田样式的刃类武器获得+1致命度。

经验消耗：6XP 稀有度修正：+4

# 贤三之刃

贤三铁匠铺是仙鹤氏族土地上首屈一指的铁匠铺之一。这家铁匠铺由坂中贤三创立，为垣田家打造奇妙的决斗武器。事实上，贤三之刃——几乎每一件从贤三铁匠铺打造出来的这样或者那样的剑——被认为是绿岩大陆最好的决斗武器之一。贤三之刃既美观又极度致命。

生产自贤三铁匠铺的武器+1致命度，-1伤害，并获得华丽特质。

经验消耗：8XP 稀有度修正：+5

# 白银玉镶嵌

国白银是一名以他美观又致命的武器闻名的铁匠。在他的漫长职业生涯中，白银发现了一种方式来制作玉灌注的混合体，这种技术可以应用于精心制作的武器和护甲，可以用来保护个人免受阴影之地的恐惧。使用这种玉混合体打造的武器通常都拥有大量华丽的雕刻，从而将混合体注入其中。如果物品暴露在阴影之地的污染下，那么玉将会慢慢变成劣质品，需要再次填充混合物。对于某些人来说这个效果不是很方便，但白银玉镶嵌保护了许多巨蟹氏族战士的生命和灵魂。

使用白银玉镶嵌工艺制作的武器和护甲具有神圣特质。如此改动的护甲失去1点物理抵抗，而武器失去一点致命度。当装备获得损坏特质，装备上的玉也会逐渐脱落，必须重新填充混合物。当此发生时，物品失去神圣特质，直到一名铁匠将在修正阶段执行一次难度为3的铁匠（土）检定来重新填充。这个过程每次尝试都会丧失一枚玉戒指。

经验消耗：3XP 稀有度修正：+2

# 乌切马的技巧

乌切马的技巧是一种护甲制造技巧，用于为探子和围城工程师们打造特种护甲。乌切马的技巧可以是任何护甲，从简单的足轻甲到华丽而沉重的大铠。通过精心的裁剪和巧妙的使用系带和带子，铠甲匠可以使用该技巧让护甲更加灵活，更易于穿着。穿着乌切马技巧制成护甲的战士可以跑、跳、弯腰、攀爬，就好像穿着简单的衣服而非笨重的铠甲一样。

一套使用乌切马技巧的护甲失去本种特质并降低1点物理抵抗。

经验消耗：4XP 稀有度修正：+3

# 康纪钢

几十年前，巨蟹氏族铸剑师肯如康纪在改进他的熔炉时发明了一种新的钢合金。他偶然的发现矿石和精炼金属的混合物可以铸造出轻便且坚固的钢铁，不过却缺乏传统绿岩钢的美感。很快他们开始铸造巨大的双手剑和双手斧，用于保卫长城。康纪钢制作的武器在巨蟹氏族武士间十分流行，他们欣赏坚固且制作精良的利刃，对于美学则在乎的少一些。

一把使用康纪钢铸造的武器失去仪式、笨重与华丽特质（而且永远无法获得这些特质）。它获得耐用特质，如果该武器已经有耐用特质，那么它的致命度+1。

经验消耗：5XP 稀有度修正：+4
