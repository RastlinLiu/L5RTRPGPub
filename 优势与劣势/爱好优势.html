# 爱好优势

# core

# 动物连接（土）

类型：人际，心智

效果：以下效果应用于拥有动物连接爱好优势的角色

  • 作为你的兴趣回报，你能拥有一只平凡的小动物。选择一个体型0或1的动物，不管你前往何处它都会随你左右，并像伴侣一样行动。它可以被说服去做你想让他做的事情，但可能需要奖励（通常是食物）来按要求完成困难的任务。如果你的起始装备中包含一只动物，那只动物可以成为你的伴侣

  • 在你执行一个与动物互动的检定后（例如冥想【土】检定来治疗他的伤口，或者求生【土】检定来平抚它），你移除3点冲突

# 军备（土）

类型：心智

效果：以下效果应用于拥有军备爱好优势的角色

  • 除非一件装备来自于翡翠帝国之外，否则你知道任何你遇到的武器或者护甲的名字，了解它们的名字，并且对如何使用它有着基本认知。

  • 在你执行一个制作或者研究武器或护甲的检定后（例如铁匠【土】检定来抛光护甲，或者战术【土】检定来安排合适的长矛阵型），你移除3点冲突

# 书法（风）

类型：心智，体质

效果：以下效果应用于拥有书法爱好优势的角色

  • 你可以识别出众多不同流派和独立艺术家的标志性风格，也包括他们常规的手法和技巧

  • 在你执行一个学习或者练习书法的检定后（例如美学【风】检定来评价作品和作者的质量，或者创作【风】检定来为一幅作品增加画龙点睛的一笔），你移除3点冲突

# 好奇心（火）

类型：心智

效果：以下效果应用于拥有好奇心爱好优势的角色

  • 你第一次尝试某种新任务并失败时，你总是可以想到你可以尝试的其他路线来试图克服此困难。

  • 在你执行一个寻求有趣问题答案的检定后（例如医疗【火】检定来设计一种新的化学混合物，或者宗教【火】检定来试图确定特定神灵行为的形而上学的基础），你移除3点冲突

# 蛮勇（火）

类型：心智，体质

效果：以下效果应用于拥有蛮勇爱好优势的角色

  • 你可以不做考虑不用犹豫的拿自己生命冒险，即便潜在的后果是明显的或者可怕的。

  • 在你执行一个冒生命危险的检定后（例如体能【火】检定来抵抗致命一击，或者游戏【火】检定来浮夸的赢得比赛），你移除3点冲突

# 启迪（空）

类型：灵性

效果：以下效果应用于拥有启迪爱好优势的角色

  • 你对世俗的物品和尘世间的头衔没有欲望；你不会被物质财富或权力的承诺所诱惑

  • 在你执行一个让自己从物质问题中解脱的检定后（例如冥想【空】检定来思考自己在宇宙中的位置，或者求生【空】检定来与环境和谐共处），你移除3点冲突

# 时尚（火）

类型：人际，心智

效果：以下效果应用于拥有时尚爱好优势的角色

  • 你总是知道你家乡的流行趋势，并可以在新地方花费一个或更多场景后检定出什么是流行的（且可以从中获得优势）。

  • 在你执行一个和衣服、珠宝以及其他穿着的艺术品的检定后（例如文化【火】检定来推理下届冬季朝堂上其他人都会穿什么，或者设计【火】检定来创造一个令人震惊的作品），你移除3点冲突

# 算命（空）

类型：心智，灵性

效果：以下效果应用于拥有算命爱好优势的角色

  • 你可以鉴别任何常规的预兆（例如星座或者其他的星象，甲骨文的裂痕，某些特定动物的出现，以及其他要发生事情的征兆），以及他们对未来的影响

  • 在你执行一个和觉察未来相关的检定后（例如求生【空】检定来寻找尘世间的小征兆，或者宗教【空】检定来观星占卜），你移除3点冲突

# 慷慨（水）

类型：人际

效果：以下效果应用于拥有慷慨爱好优势的角色

  • 你总是可以判断给特定某人的合适的礼物来避免公开冒犯他或者冷落他

  • 在你执行一个获得或者给予礼物的检定后（例如设计【水】检定来选择与某人的衣着相配的礼物，或者商业【水】检定来从市场上找寻并选择一个礼物），你移除3点冲突

# 八卦（风）

类型：人际

效果：以下效果应用于拥有八卦爱好优势的角色

  • 在每个场景开始时，你总是能定位到哪个角色当下知道最为诱人的谣言（当然除了你以外）

  • 在你执行一个散播或者接收谣言的检定后（例如礼节【风】检定来讲述某些人的故事时，将他们放在聚光灯下，或者文化【风】检定来从人们遵守或者打破礼节的微妙行为中判断他们当前的焦虑），你移除3点冲突

# 历史（土）

类型：心智

效果：以下效果应用于拥有历史爱好优势的角色

  • 经过短时间的研究，你可以分辨出你遇到的任何物体、建筑或者文字的时代和背景，如果你不能，那么你可以确定翡翠帝国的编年史家们以前从未遇到过这个东西。

  • 在你执行一个回想历史轶事的检定后（例如管理【土】检定来与过去的事件进行对比，或者铁匠【土】检定来了解过去的工具和制品的用途和创造趋势），你移除3点冲突

# 插花（水）

类型：心智

效果：以下效果应用于拥有插花爱好优势的角色

  • 你总是可以从某种植物的一部分了解它的信息，你知道它的物理特性以及花语含义

  • 在你执行一个和花卉安排相关的检定后（例如美学【水】检定来重新安排或者提升，或者【求生】水检定来从市场上买到花卉），你移除3点冲突

# 嬉闹（风）

类型：人际

效果：以下效果应用于拥有嬉闹爱好优势的角色

  • 当你对某个地位低于或者等于你的人说一些不愉快的事情时，他们会把它当作你的嬉闹本质作祟，而不会被激怒或因此悲伤

  • 在你执行一个嘲笑或者斥骂某人的检定后（例如创作【风】检定来从对手的诗歌中找到你的下一部作品可以模仿或呼应的部分，或者礼节【风】检定来用一个巧妙的双关语戏弄某人的名字），你移除3点冲突

# 挑衅（火）

类型：人际

效果：以下效果应用于拥有挑衅爱好优势的角色

  • 你总是可以确定一个你可以利用来对付其他人的敏感点——尽管你不知道他们是会把这点当作略带攻击性的戏虐还是严重的侮辱

  • 在你执行一个对某人兴风作浪的检定后（例如礼节【火】来用令他人愤怒的昵称呼叫某人，或者命令【火】检定来让一群士兵热血沸腾并足够愤怒以猛烈攻击敌人的位置），你移除3点冲突

# 清酒（水）

类型：心智

效果：以下效果应用于拥有清酒爱好优势的角色

  • 你可以识别出各种清酒及其他酒精饮料的地区差异，你对制作清酒的过程了解甚多。你需要摄入别人两倍的酒类才会获得醉酒状态

  • 在你执行一个与发酵饮品的精细艺术互动的检定后（例如礼节【水】检定来以合适的流程在社交场合中招待饮酒，或者劳工【水】检定来发酵大米），你移除3点冲突

# 秘密（空）

类型：心智，人际

效果：以下效果应用于拥有秘密爱好优势的角色

  • 你永远不会忘记你听过的秘密，而且你本能的知晓你听到的内容是常见知识还是某些人想要隐瞒的内容（即便你不知道是谁想隐瞒或者为什么想隐瞒）

  • 在你执行一个获取或者维护秘密的检定后（例如冥想【空】检定来抵抗从你身上挖掘秘密的努力，或者共情【空】检定来预感是否有人在隐藏一个可怕的秘密），你移除3点冲突

# 故事（土）

类型：心智，人际

效果：以下效果应用于拥有故事爱好优势的角色

  • 你总是可以回忆你被讲述的故事，包含其他人可能会掩饰或者忽略的故事细节

  • 在你执行一个讲述或者回忆的检定后（例如表演【土】检定来侃侃而谈振奋人心，或者宗教【土】检定来从新生之道或者阿古渡的领导中背诵一段适合你当时情况的内容），你移除3点冲突

# 茶（空）

类型：体质，灵性

效果：以下效果应用于拥有茶爱好优势的角色

  • 你可以识别出各种茶的地区差异，你对制茶的工艺了解甚多，你可以容易的获得制作质朴茶的各种自然原料

  • 在你执行一个与茶互动的检定后（例如医疗【空】检定来使用可以帮助身体从疾病中恢复的原料煮茶，或者表演【空】检定来举办一场茶会，帮助受众思考他们的生活状态），你移除3点冲突

# 旅行（水）

类型：心智、体质

效果：以下效果应用于拥有旅行爱好优势的角色

  • 你不会因为外国的习俗感到不安，即便在陌生的环境中你也很舒适的融入其中

  • 在你执行一个到达一个新地方或者遭遇一件新鲜事的检定后（例如求生【水】检定开试图确定新植物、动物或者现象，或者航海【水】检定来用船旅行），你移除3点冲突

# 文字游戏

类型：人际，心智

效果：以下效果应用于拥有文字游戏爱好优势的角色

  • 在阅读或聆听他人的作品时，你总是可以从他们的作品中找到修辞手法和文学模式。

  • 在你执行一个解释或者创造文字游戏的检定后（例如创作【风】检定来书写一个拥有完美意境的双关语的诗歌时，或者表演【风】检定来巧妙的通过双关向观众暗示些什么），你移除3点冲突
