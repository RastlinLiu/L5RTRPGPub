# 概述

一个角色可以获得任意数量的卓越优势

一个角色在gm认可的情况下可以花3xp获得一个爱好优势，每个角色最多可以获得三个爱好优势

一个角色可以获得任意数量的逆境劣势

一个角色至多获得三个焦虑劣势

# 转变优势和劣势

反转你的优势：如果一个角色的优势在特定情况的检定下形成逻辑上的漏洞，那么gm可以反转该优势，将其视为相对应的劣势（卓越优势反转为逆境劣势，爱好优势反转为焦虑劣势）。若如此做，该角色在该检定完成后获得1个虚空点

利用目标优势：其他角色的优势也可能被用于对付他们自己。如果一个角色进行目标一个或多个的检定，那些目标的玩家可以选择一个他们知道的优势，且该优势在检定中可以形成逻辑上的漏洞，如果gm同意，该玩家可以花费1个虚空点来利用该优势，在该检定中将其视作一个你的优势

利用你的劣势：如果一个你的劣势在特定情况下的检定中可以制造一个逻辑上的优势，并且gm同意的情况下，该玩家可以花费1个虚空点将其视为相对应的优势（逆境劣势视为卓越优势，焦虑劣势视为爱好优势）

利用目标劣势：如果检定目标一个或多个角色，玩家可以选择一个可以在逻辑上对抗目标的已知劣势。如果gm同意，你可以花费1个虚空点来利用该优势，将其在此检定中转变为一个你对应优势（逆境劣势视为卓越优势，焦虑劣势视为爱好优势）
