# 盟友【姓名】（水）

类型：人际

效果：以下效果应用于拥有盟友【姓名】卓越优势的角色

  • 你已经向有的人证明你，让他能愿意帮助你（在特定原因下）。他们愿意和你分享绝大部分了解的话题，无论是在面对面的会议还是通过书写信函。他们也愿意为你们安排与他们认识的人见面。

  • 当你执行一个能从你与某个角色的亲密关系获得好处的检定时（例如礼节【水】来通过他们征用某个物件或者共情【水】来阅读他们的情绪），你可以至多重置两个骰子。

# 双巧手（风）

类型：体格

效果：以下效果应用于拥有双巧手卓越优势的角色

  • 当你执行一个能从双手相当的能力获得好处的检定时（例如武术【近战】【风】检定来用你的左手做出出乎意料的一击，或者诡计【风】检定来隐秘的从他人口袋里面偷窃），你可以至多重置两个骰子。

  • 你对超自然生物有着本能的感觉，你可以感受到灵魂或者相似的实体的存在，即便他们没有选择展示自身。在有邪恶意图的生物面前，你能感到不详，而如果这些超自然生灵只是正常行事，则这种感觉是中立的

  • 当你执行一个你需要跟随你的直觉的检定时（例如冥想（空）检定来在决斗中决定先攻，或者宗教【空】来感受宇宙中看不见的事物的咆哮），你可以至多重置两个骰子。

# 身材矮小（风）

类型：体质

效果：以下效果应用于拥有身材矮小卓越优势的角色：

  • 你适合于小的空间，可以躲在别人身后，并且在人群中难以被看到，有时候需要别人帮助来获得高处架子上的东西。

  • 当你执行一个身材矮小是优势的检定时（例如体能【风】检定来通过躲闪抵抗致命一击，或者诡计【风】检定来偷别人的口袋），你可以至多重置两个骰子。

# 敏锐的观察者（风）

类型：人际，心智

效果：以下效果应用于拥有敏锐的观察者卓越优势的角色：

  • 你可以懂得人们说话时的暗示，你可以读懂别人的面部表情和唇语来了解他们说了什么，即便你听不到他们的谈话。

  • 当你执行一个你需要来观察或使用周围其他人的小细节的检定时（例如：礼节【风】检定来把不愉快的真相愉快的展示出来，或者共情【风】来探知他人的弱点），你可以至多重置两个骰子。

# 【某组织】的支持者（水）

类型：人际

效果：以下效果应用于拥有【某组织】的支持者卓越优势的角色：

  • 你已经对某组织证明了你的价值，他们愿意在不损害自己声誉的情况下帮助你。它的成员会招待你，为你提供饭食，休息的住所，以及如果他们有治疗者的话一些医疗照看。他们也会给你前往他们其他设施的权限，例如图书馆、铁匠铺或者马厩。当选择此优势时，请与GM商讨以选择一个合适的组织。

  • 当你执行一个与该组织的成员互动并依赖你的亲和力的检定时（例如命令【水】检定来对他们发号施令，或者表演【水】检定来让组织内的成员对你感到钦佩），你可以至多重置两个骰子。
