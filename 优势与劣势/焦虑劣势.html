# 焦虑劣势

# core

# 成瘾（土）

类型：心智，体质

效果：以下效果应用于成瘾焦虑劣势的角色

  • 选择一个成瘾的物质或者体验；剥夺这种物质或者体验超过三天，就会引起颤抖、发冷、失眠，甚至更剧烈的身体状况，这种现象越来越难以掩饰。

  • 在你执行一个抵抗这种物质或者体验的检定后（例如体能【土】检定来在长时间无法接触这种物质/体验的情况下压制症状，或者冥想【土】检定来抵抗追逐这种物质或体验），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 战斗创伤（火）

类型：心智，体质

效果：以下效果应用于战斗创伤焦虑劣势的角色

  • 在决斗、小规模冲突或者大规模战斗开始时，你的双手在颤抖，身体紧绷，甚至于你的心智出现宕机。你正常的进行反应，但是你的不安状态对于任何周围人来说十分明显，其他人可能会利用它

  • 在你执行一个与战斗相关的检定后（例如武术【近战】【火】检定来击倒一名敌手，或者战术【火】检定来生成主动性），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 怀疑主义（火）

类型：人际，心智

效果：以下效果应用于怀疑主义焦虑劣势的角色

  • 当你面对一个真正新颖或者非正规的想法时，你总是在寻找它可能失败的方式，而且其他人必须在你会尝试前说服你这可能会成功（即便这个点子来自于你自己）

  • 在你执行一个想法异常的检定后（例如表演【火】检定来在表演中夹杂即兴，或者战术【火】检定来临时制定新的战略），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 黑暗秘密（空）

类型：人际

效果：以下效果应用于黑暗秘密焦虑劣势的角色

  • 你拥有一个黑暗秘密，如果被曝光的话，将会为你的家族带来毁灭，让你的祖先和后辈蒙羞。如果这个秘密的细节公布于众，你的荣耀值会急剧下降，就好像你失去了荣耀上的赌注一样（10点到20点之间）。依据这个秘密，你的地位值也可能像是失去了地位上的赌注一样（5到10点左右）

  • 在你执行一个让自己或他人为自己坚持意志的检定后（例如命令【空】检定命令部队誓死奋战，或者冥想【空】检定来寻求自身的平静），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 妄自尊大（空）

类型：心智，人际

效果：以下效果应用于妄自尊大焦虑劣势的角色

  • 你认为你自己命中注定要成就大事，没什么能阻止你对这个目标的执着。当你有机会提升你的荣耀值时，你必须试图追寻它

  • 在你执行一个要求你面对自己在这个冷漠宇宙中的低下地位的检定后（例如劳工【空】检定来在田野中讨生活，或者冥想【空】检定来盘算自己在这个宇宙中的地位），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 恐惧死亡（土）

类型：心智，体质

效果：以下效果应用于恐惧死亡焦虑劣势的角色

  • 你无法自愿的选择死亡或者牺牲你的生命（包括荣誉要求你切腹自尽），如果有机会你能活下来，你一定会去追寻这个机会

  • 在你执行一个拿自己的性命冒险或者接受自己的死亡的检定后（例如冥想【土】检定来在战斗前寻找内心的平静，或者战术【土】检定来为了极其偶然的机会坚守阵地），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 残忍（风）

类型：体质，心智

效果：以下效果应用于残忍焦虑劣势的角色

  • 当你打击别人时，你对于控制自己的力道存在困难，你倾向于全力猛击，即便其他更精细的行为更加合适。当你击败一个爪牙敌人时，那个敌人总是会死亡，即便你希望让他陷入无法行动状态。

  • 在你执行一个意图收力或者需要巧妙用力的检定后（例如游戏【风】检定来巧妙的操纵对手，或者武术【近战】【风】检定来挥出巧妙地一击），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 缺乏耐心（土）

类型：心智

效果：以下效果应用于缺乏耐心焦虑劣势的角色

  • 你发现非常难以克制自己，当你被迫等待的时候，会造成你明显的不适。如果给你机会在抓住机会和等待更好的时机之间选择，你将努力抓住机会，即便以后采取行动可能会降低风险

  • 在你执行一个战略性的权衡你的选择的检定后（例如管理【土】检定来回想与你现在情况相关的历史轶闻，或者战术【土】检定来确定你出击的合适时机），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 偏狭（水）

类型：人际

效果：以下效果应用于偏狭焦虑劣势的角色

  • 针对特定一群人（或者超自然生物）的特殊仇恨在你体内膨胀，当你面对这群人中的代表时，仇恨会压倒你更好的判断能力。选择一群人或者超自然生物作为你仇恨的敌人，当你在这些敌人面前（即便是有人提到他们），你的敌意对于所有能观察到你的人来说都是十分明显的

  • 在你执行一个与选择类型组织交互的检定后（例如商业【水】检定来与一个曾经冤枉过你的家族的其他成员讨价还价，或者创作【水】来翻译一些你曾经在战场上对战过的外国人的语言的内容），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 难抑调情（土）

类型：人际

效果：以下效果应用于难抑调情焦虑劣势的角色

  • 你发现放过与你认为有吸引力的角色调情的机会十分困难，当有人找到机会与你调情时，你会情不自已并会以你的调情进行回应。你的兴趣对于围观者来说十分明显

  • 在你执行一个与你感兴趣的某人（或者忽略某人）的没有超过适当程度的互动的检定后（例如礼节【土】检定来进行合适的短暂谈话，或者共情【土】检定来在社交场合谨慎行事），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 嫉妒（风）

类型：体质，心智

效果：以下效果应用于嫉妒焦虑劣势的角色

  • 你不能自愿放弃与竞争对手竞争的机会，也不会放弃在别人眼中展示你对某人优越性的机会

  • 在你执行一个与他人竞争来你的杰出的检定后（例如表演【风】检定在敌手前说服某位听众，或者铁匠【风】检定来在将武器送给自己的领主作为礼物前翻新它），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 唯物论（空）

类型：心智

效果：以下效果应用于唯物论焦虑劣势的角色

  • 你痴迷于物质财富和物体，无视《新生之语》对这种尘世上的纽带的看法。你不能拒绝稀有度3或以上的礼物

  • 在你执行一个放弃你拥有的物体或者其他有价值的东西的检定后（例如管理【空】检定来牺牲一栋建筑从而找到对敌手的优势），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 懦弱（火）

类型：人际

效果：以下效果应用于懦弱焦虑劣势的角色

  • 你极端害羞，为自己反对总是让你挣扎，你无法打断比你地位高的角色，你发现你很难在别人眼中证明自己，这包括比你地位低的角色

  • 在你执行一个反对或者获取你需要的东西的检定后（例如命令【火】检定来激励你指挥下的部队，以及管理【火】检定来在事务中获取你的权利），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 痛苦的诚实（风）

类型：人际

效果：以下效果应用于痛苦的诚实焦虑劣势的角色

  • 当你说谎或者认可谎言的时候，你会被内部冲突所折磨。你不能提出你知道是假的主张，尽管在礼仪或者情况需要的时候，你可以说在技术上是真实却暗示是假的事情

  • 在你执行一个误导他人的检定后（例如礼节【风】检定来利用虚假消息污蔑某人的名誉，或者命令【风】检定来操纵下属），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 偏执（水）

类型：体质，心智

效果：以下效果应用于偏执焦虑劣势的角色

  • 在每片阴影中，你都能看到刺客，在每个微笑中，你都能看到一个追求你灭亡的敌手。这种恐惧让你很难依赖他人，而且这会让你分散对真正威胁的注意力，除非你自己确认了重要的信息（例如目前的安全情况），否则你无法放松

  • 在你执行一个在自己环境内确认危险的检定后（例如管理【水】检定来了解某个朝堂内的政治情况，或者共情【水】检定来感觉某人对你的意图是否敌对），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 完美主义（水）

类型：体质，心智

效果：以下效果应用于完美主义焦虑劣势的角色

  • 你工作中任何不完美的地方都是不可接受的，你已经知道自己创造了美丽的作品，但却因为一些缺陷把它们丢在一边。你无法不努力改进你发现有缺陷的任何作品，即便这意味着向地位更高的人明示他们所犯的错误

  • 在你执行一个你必须与不完美或者未完成的作品互动的检定后（例如美学【水】检定来改进一件已经存在的艺术品，或者创作【水】检定来翻译一个作品但无法不丢失原文中的细节），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 恐惧（水）

类型：体质，心智

效果：以下效果应用于恐惧焦虑劣势的角色

  • 选择一件平凡的东西。你对这个东西感受到一种致命的恐惧。你没法说出它的名字。在它的面前，你无法隐藏你的恐惧，如果可能的话你试图逃离

  • 在你执行一个与你恐惧之物互动的检定后（例如礼节【水】检定来愉快的谈论此物，或者求生【水】检定来检定此物），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 心慈（火）

类型：人际，心智

效果：以下效果应用于心慈焦虑劣势的角色

  • 如果给你选择来杀掉某人或者放他一马，你总是选择饶恕他的性命，即便让他活下去不符合你的利益

  • 在你执行一个处理或涉及杀害他人或者让他人死亡的检定后（例如管理【火】检定来估计一场战争给普通人带来的负担，或者武术【远程】【火】检定来选择目标），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点

# 迷信（空）

类型：心智，灵性

效果：以下效果应用于迷信焦虑劣势的角色

  • 当你面对超自然迹象时，你会明显变得激动，并试图离开现场，安抚有问题的存在或者现象，或者取得某种类型的超自然庇护

  • 在你执行一个与超自然存在或者现象互动或者了解的检定后（例如冥想【空】检定来察觉到超自然的存在，或者宗教【空】检定来了解一个超自然存在的意图或者愿望），你受到3点冲突。如果这是你在本场景内第一次遇到此情况，获得1点虚空点
