参照学派中文翻译

# 问题4：你的角色如何在学派中脱颖而出

选择如下一项，提升你的轮数值：

+1火轮：创造性、热情或者驱动力

+1风轮：优雅、雄辩或者共情

+1水轮：适应性、亲密或者具有认知

+1土轮：彻底、耐心或者平静

1+空轮：自我认知、洞察力或者神秘主义

# 问题5：你的领主是谁以及你对他的责任是什么？

与gm讨论确定领主的名称、地位、历史，以及你们之间的关系，并确定你的义理

义理指的是你的角色在他们的领主眼中获得尊重的方式，关系到你的角色在帝国内的声誉，以及他们的荣耀

义理为你的角色提供了主要的驱动力之一（另一个是人情）。你应该在（和人情一起）考虑义理时对以下问题持肯定回答，否则需要和gm一起重新选择义理：

这个义理是你希望你的角色在故事里追逐或者对抗的吗？这并不是说你的角色总是会享受沉重的责任，但作为玩家你应该对于看到你的角色与之（义理）斗争而感到兴奋。

无论战役将你的角色带到何处，你的角色总是能在完成义理的路上取得进展吗？一个好的义理即便在与主线故事无关时也应该有着清晰的探索路径，并且应该是无论在何地或者面对何种困难都能影响你的角色的选择。

这个义理是否会至少偶尔与你的人情发生冲突？在责任和渴望中徘徊是许多武士主题故事的关键部分，即便这两者不完全相反，你的角色也应该会有时作出艰难的抉择，来决定哪个更加的重要

# 问题6：你的角色渴望什么？它会如何影响你的责任？

人情是你为你的角色设下的挑战，迫使他们去面对理想中的武士道，以及追逐渴望过程中维护武士道的代价——因此，这关系到他们的载誉。另一个方式来看待人情是自问：什么样的目标可能会让你的角色不服从他的领主或者做一些不光彩的事情？对他来说有什么值得冒这样的风险？

人情是你的角色的另一个主要驱动力，一些典型的人情元素包括：野心、艺术、美貌、归属、启迪、嫉妒、家族荣耀、贪婪、爱、正义、知识、和平、安全以及复仇。

你应该在（和义理一起）考虑人情时对以下问题持肯定回答，否则需要和gm一起重新选择人情：

这个人情是你希望你的角色在故事里追逐或者对抗的吗？这并不是说你的角色总是觉得追求渴望是快乐、明智或者健康的，但作为玩家你应该对于看到你的角色与之（人情）挣扎而感到兴奋。

无论战役将你的角色带到何处，这个人情总是会出现？一个好的人情可以让角色无论在哪里徘徊都会被其牵动，并且应该是无论在何地或者面对何种困难都能影响你的角色的选择。

这个人情能让你的角色可能作出玷污自己的载誉或者违背领主的行为吗？一个好的人情对角色来说是一个重大的诱惑，忽视它会产生严重的风险，这会导致武士心情低落或者不再有为他的领主服务的动力

# 问题7：你的角色和你的氏族是什么关系？

选择如下一项：

如果你的角色坚信他所在氏族的戒律以及所维护的价值观，那么由于作为氏族内坚定的成员赢得的声誉+5荣耀

如果你的角色对氏族的做法、政策以及信仰有着根本性的分歧，那么你可以从你等级0的技能中选择一项获得1个等级。考虑一下这项技能为什么能代表你的角色与氏族的训练或者价值观产生分歧

# 问题8：你的角色如何看待武士道？

选择如下一项：

如果你的角色坚信武士道正统诠释的生活，那么获得10点载誉值

如果你的角色对于武士如何表现的荣誉有着一些或者全部分歧，那么从以下技能中选择一项获得一个等级，来反映角色过去的行为不适合武士或者违反规范：商业、劳工、医疗、航海、诡计或者求生

# 问题9：你的角色目前为止所完成的最伟大成就是什么？

除去决定角色完成的最伟大成就外，选择一个卓越优势，来作为你的角色达成这一伟大成就的主要原因或者达成这一伟大成就的结果

# 问题10：你的角色生活中最大的阻力是什么？

为你的角色选择一个逆境劣势

# 问题11：什么活动能让你的角色感到最为平静？

为你的角色选择一个基于你的角色爱好或者兴趣的爱好优势

# 问题12：你的角色最关心、恐惧或者最为困扰的弱点是什么？

为你的角色选择一个焦虑劣势

# 问题13：你的角色一生中，从哪个角色那里学到的最多？

在关系一栏中，记录下你的角色学到最多的人物名，以及他们之间关系的本质，然后选择如下一项：

一项与你的角色的启发者和他们之间关系相关的优势

一项与你的角色的启发者和他们之间关系相关的劣势，以及一项技能的一个等级，作为你的角色因此而发展出的结果

# 问题14：当人们第一次遇到你的角色，最先被注意到的是什么？

回答人格、爱好及怪癖部分的问题

此外，为你的角色选择一个绝大部分时间都会佩戴或者携带的独特审美物品（例如围巾、发饰、雕刻的剑鞘、眼罩等），这样一点点的个人特征可能会突出他们的特点，或者与之相反

# 问题15：你的角色如何应对压力场景？

描述你的角色在面对压力时的显著物理特征，它总是可以作为揭面的一部分。

描述一些你的角色在面对压力时的生理上、语言上及心理上的行为，

# 问题16：你的角色现存的和其他氏族、家族、组织和传统之间的关系如何？

描述你的角色与其他组织之间的历史、争端、联盟以及抱怨

此外，选择一件稀有度在7及以下的物品，作为你的角色接收其他组织的礼物，与其他组织战斗的战利品，或者其他相关或标志了角色与其他组织的历史以及现状。将此物品加入起始装备中。

# 问题17：你角色的父母如何描述你的角色？

描述你的角色和父母以及家庭其他成员的关系和本质

此外，为一个技能等级为0的技能增加一个等级，并决定角色的父母是否支持这种额外的兴趣或者这是一种令人遗憾的偏差，以及为什么

# 问题18：你的角色的名字荣耀了谁？

扔d10两次，对照武士遗产表来确定两个潜在的家族荣耀。你可以选择其中之一（或者由gm指定选择第一项、第二项或都选择）。然后为每一项扔一个d10来确定具体结果

# 问题19：你的角色名字是什么？

确定你的角色的名字，以及他的含义

# 问题20：你的角色会如何死亡？

确定你的角色未来的死因，供gm使用
