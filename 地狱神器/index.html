# 地狱神器

# 力量的处罚

携带或者使用被诅咒的物品扭矩了角色的人性。他们的镇定值上限减少2。此外，他们只能使用被诅咒的物品标明的脱面形式。

# 绝望铁砧

绿岩大陆上最肮脏的神器中，绝望铁砧是最为臭名昭著的。阿贺娑龙园（Agasha Ryuden），使用从恶鬼体中提取的铁，并献祭了整个野猪氏族来创造了它。铁砧上锻造的任何物品都会被唤醒诅咒和恶意，最终腐蚀它的持有者。它是一只无形的手，塑造者绿岩大陆的历史，没人能说出有多少恶毒的利刃在这个恶毒的铁砧上被打造出来。

效果：每次你希望使用铁砧进行一次锻造检定，你都需要为邪术支付合理的牺牲。任何使用绝望铁砧打造的武器+2致命度，并获得不洁特质。

脱面形式：荣誉的决斗

# 符冷的黑珍珠

当一颗特别纯净的珍珠或者宝石暴露在污染之中，它很容易沾染到超自然杂质。在受到严重的腐化影响，特别是直接接触到阴影之地的力量后，这样的宝藏会变得扭曲且危险。邪术使用者可以供奉符冷的黑珍珠来吸引数英里内的恶毒邪灵。

效果：你持有符冷的黑珍珠的时，你可以使用邪术而无需支付合适的牺牲。当你执行邪术技巧的检定而没有支付合适的牺牲时，增加两个【机会】【冲突】的轮骰作为保留骰。

脱面形式：扭曲原则

# 变化赠物

被邪术师创造的被诅咒的变形礼物，看起来都是平凡的家用物品，每个都代表了不同的动物：鹰的发卡，银色的折纸猫，狼和服，野猪凉鞋以及狗扇。这些饰品可以将携带者变形为代表的动物，并可以将其变回人类。不过使用者每次使用此物品，他都会承受更多的污染，直到最后被困在该动物的怪物版本而无法变回人类。

效果：每个变化赠物都代表了一种动物。每场游戏一次作为一次支援行动，你也使用变化赠物来诅咒和强化自己的身体，将自己转化为变化赠物代表的动物。在动物形态下，你使用动物的轮数值，衍生属性，能力，以及所有体格和社交优势。不过你使用自己的技能等级，所有的心智和灵性优势。这个效果持续到你选择结束它或者获得无意识状态时，在那时你转变回原来的形态，并承受折磨状态。你变回原形态时，不要移除任何你在动物形态下获得的疲乏和冲突；如果这造成你的疲乏超过了你的耐力（或者你的冲突超过了你的镇定），那么你承受一次程度为5的致命一击（如果是冲突值情况的话，那么将进入妥协状态）。

脱面形式：慌张的撤退

# 符冷的瓷面具

由于只在渚寿路的遗言中以第三手的状态被记录，因此还不确定符冷的瓷面具是否存在。据描述当瓷面具被带上时，它会自动在脸上封死，只有死后才会脱落。它可以将使用者的面容改变为任意使用者想象的样子，并且可以通过使用者的双眼，辨识出隐藏在凡人身上的惊惧兽。如果这个神器真的存在，那么它那愚蠢的主人肯定会迷失在面具揭示的噩梦中。

效果：一旦你佩戴了面具，直到死亡前它无法被移除（或者其他稀有的事件），你在gm的选择下获得了一个阴影之地的污染劣势。

你带着面具时，以一个支援行动，你可以强化自身，将面具转化为你想要呈现的面孔；其他角色必须使用难度为4的共情检定来发现你伪装的缺陷。此效果持续到你进入无意识状态，或者按意愿结束。

你带面具时，可以占卜超自然存在，让他们身上散发着只有你能看到的微弱的紫光。这个效果持续到本场景结束。

脱面形式：暴露缺陷。
