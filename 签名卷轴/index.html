# 签名卷轴

每个学派教给学生的卷轴都包含着神圣的知识，其中大部分是学派专属的。此外，每个修验者都会扩展他们的神秘实践并保留自己的记录。因此，一名新手没有办法直接前往图书馆去获得过去世代最伟大的修验者的学识。不过他们可能被交付（或者“释放”）一个古代大师的卷轴来揭开新的宇宙秘密，获得祈愿新的效果。

如果一名修验者满足了获取一个签名卷轴的前提，他们可以花费标注的经验来习得它的秘密。

“释放”知识：在GM的许可下，一名获取了卷轴但不满足其前提的角色也可能学习它的知识——如果他们不能很好的隐藏这个知识的话，如此做会为他们树立强大的敌手。

# 英一的重获光明

数年前，有一名巫术猎人名为国英一（Kuni Ichi），他那发现邪术师、隐藏的恶魔以及被污染的个体的能力闻名于世。渐渐的随着他年龄的增加，他的眼睛逐渐花了，而他的眼光也逐渐消失。英一并不是一个甘愿于此的人，他开发了一种增强能力——在占卜中请求神灵来让他日渐浑浊的眼神也能洞悉那些地狱般的存在。

前提：巫术猎人，巨蟹氏族

获取此卷轴的经验：3XP

新效果：修验者成功使用祈愿来占卜后，任何可以被认为是这次占卜一部分的超自然存在或者被污染存在都会散发着修验者可见的微弱的光芒，即便此光芒需要穿过烟尘、雾、模糊视线等也是一样。

# 长谷川的否认

优替长谷川（Ugawari Hasegawa）是一名著名的巨蟹氏族修验者，他发现了强化自己进攻用卷轴及控制战场上恶魔移动的方法。他以在战斗中驱赶恶魔，将它们引到预先埋伏好爆炸魔法的陷阱和杀伤区。在他漫长的一生结束时，长谷川为后人记录下了他的秘密，以便未来的修验者可以使用他的技巧来守卫长城以及巨蟹氏族。

前提：巨蟹氏族修验者，学派等级3

获取此卷轴的经验：3XP

新效果：修验者成功使用祈愿打击目标后，目标距离0-3区域对于被污染的存在成为危险地形。这个效果持续到你的下个回合开始。

# 青枪的斥责

安城青枪是一名流浪修验者，以她的无畏、她的热情以及她奉献的力量。她经常消失几个星期来进入阴影之地寻找特定危险的鬼。她有时候与一名叫做慎太郎的浪人伙伴一起旅行，不过对于她来说，她大部分时间都自己一个人旅行，只凭她的热情来保护自己。一次她和恶鬼的长时间交锋中，青枪发现了一种强化呼唤神灵的方式，即便在邪灵感染的区域也可以召唤出神灵。这种增强效果在地狱而来的生物看来是突如其然且爆炸性的效果，她也确保自己在120岁永久的消失在阴影之地之前尽可能广泛的传播她的技巧。

前提：修验者

获取此卷轴的经验：3XP

新效果：修验者成功使用火祈愿打击目标后，目标接下来面对的一次攻击将被视为具有神圣特质。这个效果持续到此场景结束。
