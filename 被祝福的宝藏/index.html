# 与平的玉杯

大多数由名誉扫地的浪人与平创造的物件都遗失并被遗忘了，不过他制作的玉杯则还被人们记住。这些玉杯看起来就像是普通的清酒杯，但如果仔细的观察，它的内部镶嵌着几乎不可见的白色玉石。没人知道与平制作了多少玉杯，不过时不时的就会有一个出现在大名的财产中或者市场上。

苏醒的物品：与平的杯子是未眠。因此它拥有耐用和神圣特质。

怪癖：如果有异界存在在附近，杯子会发光（如果需要计算距离的话，距离为0-3）

封印的祈愿：每场景一次，你可以使用你的医疗技能执行寿老人的香膏。

# 玉瓣茶

玉瓣茶是唯一一种对抗污染的治疗手段，它不是真正的茶，而是由米酒和一种被成为玉莲的特殊花制成的酊剂，这些花朵生长在巨蟹氏族的土地，使用灌注了玉粉的水浇灌，从而产生了具有玉的超自然特效的可食用花瓣。种植它所需要的大量时间和费用让玉瓣茶成为了一种罕见的药物。服用它时，角色可以获得对阴影之地污染的抵抗能力，并可以抑制其效果。

国氏守望者们小心的控制着玉瓣茶的生产和去向，并保留了全部翡翠帝国内提供了玉瓣茶的目标。这样做既可以监控玉瓣茶对那些遭受了污染的人的效果，也可以确保没有被污染的个体没有被他们记录在案。经常出现偷窃玉瓣茶的情况，国氏则会无情的追踪和惩罚任何持有非法数量玉瓣茶的人。

效果：当你食用玉瓣茶后，移除折磨状态。

此外，你忽略所有你的阴影之地污染劣势。这个效果持续到你再次承受折磨状态，或者叙事上经过了一周（以先到的为准）。

# 鬼咒剑（Kishu）：斩鬼之剑

为了证明存在的价值，樋田给了昼马、堺雨和国一个任务：摧毁羽生野忠惠之鬼。为此堺雨打造了鬼咒剑，俗称斩鬼之剑。专为左撇子设计，这把古代的铁质太刀混合着玉粉。它可以切开恶鬼的护甲，无论这护甲有多厚。鬼咒剑遗失在阴影之地，有人相信这把剑还在失落的白昼堡中沉睡。

苏醒的武器：鬼咒剑是一把直剑（chokotō）。作为未眠，它具有+2致命度，以及神圣和耐用特质。

怪癖：鬼咒剑只在被左手持有的时候可以斩击。

封印的祈愿：捆绑阴影被封印其中，持有者每场景一次，可以使用武术【近战】检定执行此祈愿。

# 黑曜石墨水

黑曜石墨在通常墨的成分外，还混有黑曜石粉末，从视觉上无法与普通墨区分开。当黑曜石墨与水混合在一起时，产生的黑曜石墨水会暂时在任何卷轴、画布或者其他表面上赋予黑曜石的超自然特效。不过当使用血液代替水的时候，产生的墨水会吸引邪灵，其灌注的邪术仪式会产生更大的力量。

效果：当你准备一个包含祈愿的守卫时，如果你使用黑曜石墨水，那么你可以至多承受3冲突来为其增加如此多的保留【机会】【轮骰】。

此外，你可以使用此墨水与血混合，来准备包含邪术的守卫。如此做时，你可以至多承受3冲突来为其增加如此多的保留【机会】【冲突】【轮骰】
