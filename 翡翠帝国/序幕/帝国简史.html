# 帝国简史

# 天与地的时代

由宫廷书记宫千夏编纂

我们翡翠帝国的悠久而辉煌的历史直接追溯到其由神圣的众神创立之时。多亏了他们赐予人类的许多奇妙礼物——包括艺术、学问和武士道——绿岩得以作为文明的闪耀典范而长存。正是由于天界的恩赐，绿岩才有力量抵御在历史黎明时期侵袭它的巨大威胁。

那时还没有人类，大地由五大远古种族居住：天狗、狐族、人鱼、巨魔和地精。我们对那个时代知之甚少，因为他们的帝国兴衰，没有留下任何记录。最后的狐族与阿古渡大人的追随者结婚，并创立了狐族家族，但如果他们将过去的故事传给了他们的后代，狮族的历史学家并未分享这些。凤凰族的创始神柴大人与一位人鱼女子津丸结婚，她的孙女后来出版了她祖母的故事。

独角兽族的历史学家井出高氏声称，在五大远古种族之后，另一个蛇形的种族居住在后来成为绿岩的土地上。他的依据是在森林中发现的一些废墟，那里有许多蛇形雕像和雕刻壁画，具有类似人类的特征。

# 世界的创造

起初，空无一物。这不是虚空的无，因为虚空是不变的、不可改变的。这种“无”是万物的源头，但最初它是空的、孤独的。经过无数岁月，无意识到它是孤独的，于是它感到恐惧。这创造了世界的三分之一。然后，无渴望一个伴侣，这种渴望创造了另外三分之一的世界。接着，无意识到它所做的一切，并后悔恐惧和渴望，于是最后三分之一的世界被创造出来。然后，无不再存在，取而代之的是世界。

那时的世界并没有我们今天所知的形式。那时它是无形的、流动的，就像在水中旋转的黑色油。最终，较轻的部分分离出来，成为天界，较重的部分下沉，成为大地。然后，三位无名之神出现了，他们看到虽然天与地已经形成，但其中的一切仍然是狂野而无形的。经过商议，他们创造了两位神，并派遣他们去塑造世界。这两位存在思考如何做到这一点，然后他们俯身亲吻大地，并为其命名。

现在，大地被命名，充满了生命，新的神和野兽出现了：海洋之主水神、风神和四方风、元素龙，以及许多其他存在。那两位为大地命名的神也获得了名字：男人成为月神，女人成为日神。在天界中侍奉他们的是神族，而地下的魔族则在地底劳作，监督死者的灵魂（尽管在那些日子里，死者很少）。

那时还没有人类，大地由五大远古种族居住：天狗、狐族、人鱼、巨魔和地精。我们对那个时代知之甚少，因为他们的帝国兴衰，没有留下任何记录。最后的狐族与阿古渡大人的追随者结婚，并创立了狐族家族，但如果他们将过去的故事传给了他们的后代，狮族的历史学家并未分享这些。凤凰族的创始神柴大人与一位人鱼女子津丸结婚，她的孙女后来出版了她祖母的故事。

独角兽族的历史学家井出高氏声称，在五大远古种族之后，另一个蛇形的种族居住在后来成为绿岩的土地上。他的依据是在森林中发现的一些废墟，那里有许多蛇形雕像和雕刻壁画，具有类似人类的特征。

# 众神的降临

经过漫长的岁月，天照大神与月读命被赐予了九个孩子：九位众神。有一段时间，天界充满了欢乐，但随着孩子们的成长，月读命注意到他们多么强大和有天赋，既继承了他的血脉，也继承了太阳女神的血脉，他开始担心其中一个孩子会篡夺他的权力。嫉妒和恐惧在他心中滋长，直到最终，他下定决心确保这一切不会发生——通过吞噬他的孩子们。天照大神一想到这一点就痛苦地哭泣，她的眼泪落在大地上，形成了巨大的水池。由于不愿通过直接与丈夫开战来撕裂天界，她选择了另一条道路。每次月读命吞噬一个孩子时，她都会给他一杯酒，里面有一滴毒药。当他来到最小的孩子梵帝时，月读命已经被酒弄得晕头转向，天照大神得以用包裹在梵帝衣服里的石头替换了真正的孩子，她的丈夫吞下了石头。然后他睡着了。

天照大神偷偷将梵帝带出宫殿并藏了起来。随着他的成长，她教导他荣誉与战争。当他学会一切后，她用星光制成的剑武装他，并派他去救他的兄弟姐妹。当梵帝大步走向宫殿时，月读命正从睡梦中醒来。看到他那闪耀的儿子，月读命愤怒地咆哮，抓起他的剑，冲出去战斗。他们的决斗持续了无数年，梵帝巧妙地躲避了父亲的攻击，但自己也击中了对方。当月读命的血从天界落下时，它落入了天照大神的泪池中。从每个池中，两个人类站起来，惊奇地环顾四周——因为人类是由太阳女神的眼泪和月读命的血所创造的。

经过漫长的时光，梵帝终于找到了机会。他凝聚了全部力量，深深地砍向父亲的腹部，他的兄弟姐妹们纷纷从父亲的肚子里滚落，坠向大地。符冷是最后一个，月读命在他坠落时抓住了他。梵帝再次挥剑，砍断了父亲的手，符冷也随之与其他兄弟姐妹一同坠落。绝望中，符冷嚎叫着伸手去抓梵帝，两兄弟一同坠落。随着天界的荣耀逐渐远去，兄弟俩分散开来，梵帝不知道符冷落在了何处。梵帝、阿古渡、道寺、樋田、富坚、新条，以及双胞胎柴和间由史，都落在了相近的地方。

当他们抵达大地时，众神们感到双重震惊：首先，他们惊讶地发现自己现在变成了凡人，会死亡。其次，他们惊讶于人类，尽管与太阳女神和月读命有着联系，却处于一种可怜的状态。

# 未开化的时代（史前-5IC）

在人类时代的黎明，人类像北方的夷蕃人一样，分散在世界各地，以部落的形式生活。他们崇拜福神，即人类努力的神明，他们的祭司在人类、灵兽和野兽之间穿梭。这些人生活在小村庄里，尽管在少数地方也有城镇。他们用陶土制作陶器，但没有陶轮，用青铜锻造粗糙的工具和武器。他们用兽皮和编织的草料做衣服，从森林中狩猎和采集食物。有些人尝试种植他们发现的野生谷物和豆类的小片土地。然而，这些作物像风中的花瓣一样脆弱，风暴和海洋的年轻神明的愤怒可以在瞬间摧毁它们。

尽管如此，人类的心尚未被激发去创造真正的艺术。没有文字记录他们的过去，人类依靠代代相传的故事。没有染料或水稻的种植，他们的服装简陋，收成微薄。部落随着鼓点跳舞和唱歌，但他们这样做是狂野的，既不懂旋律也不懂和声。战争，最常见的掠夺，几乎没有战略，也没有荣誉。

众神在他们之间商议，决定各自周游世界，评判其价值。由于他们美丽、强壮且智慧，他们所到之处吸引了部落的极大关注。很快，他们每个人都有一群渴望从他们那里学习天界艺术的人类追随者。写作和书法、锻造铁器和钢制工具、弦乐器、获得荣誉和忠诚带来的力量，以及许多其他实践和思想随着众神的旅行在土地上传播开来。

# 新黎明的曙光（5IC-39IC）

最终，众神在他们降落的地方重聚。他们再次商议，决定尽管人类有巨大的价值，但必须用智慧组织和治理，以便其价值得以绽放。决定后，他们举行了一场比武，以确定谁将成为这片新土地的领袖。富坚大人没有参加比武，因为据说他以他的大智慧预见到梵帝将成为胜利者。

确定了谁将成为天皇之后，众神开始创建一个帝国。除了天皇梵帝，所有众神都建立了由他们的追随者组成的氏族。天皇梵帝为每个氏族分配了土地，允许他们耕种，只要他们向他纳税。梵帝的兄弟姐妹们组织了他们土地上的部落，并开始建立学校，教授武士道。他们都开设了武士学校，因为这片土地仍然危险，旅行不安全。许多人不理解众神带给大地的伟大祝福，并抵制他们的统治。但也有一些其他学校。道寺大人的学校最为著名，因为她是艺术的第一位赞助人，她的学生们在新帝国中传播关于众神和天界的故事和其他艺术作品。很快，城镇发展成城市，道路连接了各个领地。天皇选择了众神最初降落的山丘作为他的首都，御父居城由此建成。

# 对抗符冷的战争（39IC-42IC）

许多年来，新帝国一切安好。随着帝国各地的人们意识到众神的伟大智慧和崇高，氏族不断发展壮大，天皇梵帝制定了反映天界完美的法律。

有一天，符冷来到御父居城，与他的兄弟姐妹重聚。其他众神起初为得知他们的兄弟还活着而高兴，但担忧笼罩了他们的喜悦，因为樋田氏族深知南方（符冷来自的地方）正在酝酿的邪恶。符冷对他的兄弟姐妹们大发雷霆，指责他们将他排除在决定谁将统治绿岩的比武之外，尽管他们向他保证他们不知道他还活着。他指责他们撒谎，抛弃他让他死去，并选择不去寻找他。其他众神很快明智地意识到，符冷已被地狱的邪恶腐化，地狱是真正地下世界的扭曲对应物。

符冷要求获得挑战梵帝成为绿岩领袖的权利。悲哀的是，梵帝接受了决斗，但选择了富坚作为他的冠军。当符冷让富坚选择武器时，这位智慧但孤独的龙族创始人选择了整个绿岩和所有居住在其中的人。符冷更加愤怒，撤退到影地，发誓要带着自己的军队回来进行决斗。

不久，南方传来消息，城镇和村庄被噩梦般的生物大军淹没：哥布林、恶鬼和其他既可怕又未知的生物。宣誓保卫新帝国的战士们聚集起来与这支可怕的军队作战，但所有人都被打败了。符冷正在使用邪恶的巫术召唤他的军队并带领他们取得胜利。

慢慢地，梵帝和他的兄弟姐妹们组建的军队被逼退。

# 一位众神的跪拜

正是在这段时间，伊泽部落加入了帝国。此前，伊泽是一位能力非凡的精神领袖，深受福神和神明的信任，他认为自己和部落没有必要置于太阳和月亮之子的保护之下。随着战争的进行，柴大人来到部落，请求他们帮助作战。伊泽拒绝了，因为他不想让他的部落失去传统。柴随后跪在伊泽面前，发誓如果伊泽同意成为柴氏族的成员，他和他的后代将服务和保护伊泽及其部落。伊泽对这一真诚的表现印象深刻，接受了提议。因此，直到今天，虽然柴氏族人始终是凤凰族的冠军，但凤凰族由伊泽的后代统治。

# 新生

有一天，一位穿着僧袍的老人来到梵帝的营地。他说他的名字是新生，他知道天皇如何能打败符冷的军队。梵帝起初不愿听，但在新生徒手击败了被派去驱逐他的守卫后，天皇变得好奇。两人彻夜长谈，柴忠实地记录下了所有的对话。这些卷轴成为了新生的《道》，其中包含了新生关于世界、五行和觉悟的全部智慧。

新生告诉天皇，命运眷顾凡人，因此他将召集七位人类战士来击败黑暗之神。梵帝同意了，于是从每个氏族中选出了一位战士。新生带领这些战士——七雷——进入南方。许多星期过去了，没有任何消息。然后有一天，在一场大战中，符冷的军队突然陷入混乱和溃散。帝国的战士们鼓起勇气，更加激烈地战斗，将敌人赶出了战场。所有人都清楚，七雷赢了——雷之日已经到来。梵帝下令准备一场盛大的宴会来表彰英雄们，但只有新生和蝎族的雷——渚寿路——回来了。渚寿路带着十二卷卷轴，她说这些卷轴束缚了被击败的符冷。梵帝下令永远不要打开这些卷轴，并将它们交给蝎族保管。

# 伊泽历法

由伊泽所制定的历法以众神降临为元年，标志着第一年的开始。这一历法在绿岩广泛采用，因为它比官方记录中使用的皇权纪年系统更为直观，后者以天皇在位年份记录事件。一些人质疑为什么伊泽会使用这样的系统，考虑到他对众神统治的反对；然而，关于该历法最初标记的是众神从天而降那一年中的其他事件的说法，几乎没有得到认真的考虑。

# 千年的和平

现在我们来到了一个被公正地称为“千年和平”的时代。在梵帝天皇的仁慈统治下，并在众神后裔的引导下，绿岩的人民享受了数百年的和平与繁荣。如果那些异族旅行者被允许进入我们辉煌的帝国，他们将以羡慕和钦佩的目光注视着我们土地上随处可见的优雅、智慧和富足。

# 天皇仁慈的传播

# 麒麟的启程

随着符冷军队的威胁消失，绿岩的人民重新投入到帝国的建设中。然而，梵帝并未忘记来自绿岩之外的敌人所造成的伤害，他召见了他的妹妹新条。新条大人一直是家族中的流浪者，她总是以兴趣和好奇心迎接新的人民。天皇赋予她和她的追随者——当时被称为麒麟氏族——探索帝国之外的任务，看看边境之外可能存在的威胁。道寺大人对新条的离开感到非常悲伤，因为姐妹俩一直关系密切，她送给新条一把自己亲手绘制的精美扇子，作为她们之间纽带的纪念。

# 光辉王子的统治

新条离开后不久，梵帝超越了人界，离开了凡世。一些记录称他死于对抗符冷战争中所受的旧伤，另一些则说他只是厌倦了凡世，在一道光芒中返回了天界。这是祭司们争论的问题，而非历史学家，因此我们继续前进。

梵帝由他的儿子梵帝源氏继位，被称为光辉王子。源氏是一位优雅而多才多艺的人，年轻时追求了许多冒险和逸事，为当时的说书人提供了丰富的素材。作为天皇，他赞助了许多寺庙和修道院的建设，以便五行知识和新生的《道》能够传播到整个帝国。他还继续建设了帝国城市之间的庞大道路网络。在他去世后，他将皇位传给了他的长女紫，她的统治长久而繁荣。

# 帝国法律与翡翠法官

在源氏统治期间，道寺初雄和宗志裁判官改革并编纂了帝国法律。这些法律由众神梵帝制定，因此是完美公正且无遗漏的，但帝国各地的法官作为凡人，在应用这些法律时能力参差不齐。通过发布原始法律的注释版本，初雄和裁判官使得帝国各地的法官能够做出一致的裁决。他们还设立了翡翠法官，这些官员拥有天皇的授权，可以调查犯罪并对罪犯进行审判。

# 大黑的祝福

和平恢复后，帝国的农民可以致力于改善农业。第一个重大进步是铁制工具的普及，它们比青铜工具更便宜、更坚固。有了这些工具，农民可以开垦和耕种更多的土地。通过这些工具建造的沟渠和运河，带来了可靠的灌溉和湿地水稻的引入。稻田在相同面积的土地上产量更高。拥有平坦且水源充足土地的氏族很快在人口和财富上都有了增长。

最后一个伟大的创新是发现可以用小麦粉制作面条。小麦是一种耐寒的谷物，在过于寒冷或干燥的地区也能生长，但作为整粒谷物烹饪需要很长时间，而磨碎的小麦粥作为食物并不吸引人。然而，面条美味可口，很快在各地流行起来。越来越多的土地被开垦和种植，为帝国提供了更大、更稳定的粮食供应。

随着人口的增长，贸易也随之发展。然而，商人发现以物易物的系统有限制。在许多地区，小块或小条的金或玉被用来代替实际商品，但它们存在尺寸差异和伪造的问题。随着采矿和铸造技术的普及，硬币促进了商人之间的贸易，天皇宣布，一石代表一个人一年所需的稻米量。有了石和更小的硬币，商人可以轻松买卖他们的商品，很快，食物、织物、染料和许多其他物品在帝国的道路上流动。

# 福禄寿的祝福

在武士阶层中，识字率迅速增长。鹤族、凤凰族、龙族和蝎族一直珍视智慧并传播思想。他们从一开始就接受了众神赐予的书写，认为它在履行职责中不可或缺。现在，樋田和狮族的领主们也开始看到它对所有战士的价值。注释地图使策划战役更加容易。书面命令不依赖于信使准确记住指挥官所说的话的能力。战斗后写的报告为未来的指挥官保存了知识以供学习。

随着识字能力的提高，书面作品的数量也随之增加。在帝国早期，阿古渡写了《领导力》，这是一篇关于战争和将军的论文。道寺大人的丈夫——第一位翡翠冠军——写了《剑》，解释了他的剑术技巧和哲学。

或许作为回应，他的对手见元芳胜写了《二天》，描述了他和他的养父见元开发的双剑技术。新生的《道》无需进一步解释，伊泽的论文《元素》也是在这个时代写的。在第一世纪末，出现了《谎言》；它的真正作者有争议，但传统上将其归功于众神间由史。

在第二世纪，文学作品出现了。濑分名香写了《光辉王子》，这是一系列关于梵帝源氏的松散关联的故事。当时写了许多关于梵帝源氏的书，但名香的作品被认为是最好的，至今仍在阅读。

绿岩的人民还扩展了众神和新生赐予他们的对天界的理解。龙族和濑分家族在占星术上取得了巨大进展。通过研究太阳女神、月读命和其他天体的运动，占星家可以解释神明的愿望和行动，甚至预测它们。通过这种方式，聪明的人可以预测重大事件，或确定他们的领主采取某些行动的最吉祥时间。

# 寿老人的祝福

修验者阿贺娑是富坚的追随者之一，她是最早清楚地看到新生的教导与人民长期持有的古老福神信仰之间没有矛盾的人之一。借助新生的新理解，并继续她对神明和自然世界的研究，阿贺娑发现了许多至今仍是医学基础的治愈方法。

在帝国早期，阿贺娑的门徒和绿岩各地的学者在医学和炼金术方面取得了巨大进展。他们发现了草药和其他植物的恢复特性，单独使用或作为炼金混合物的一部分。他们还开发了利用人的气来帮助愈合过程的方法，例如通过针刺身体来纠正能量失衡。最具影响力的学者之一是源次郎，一位盲僧，他游历绿岩，通过气味识别草药并传播新生的教导。

# 茶的祝福

也是在这个时代，由道寺大人发明的茶道开始传播到鹤族之外。它首先在凤凰族和龙族中流行起来，因为他们都欣赏它所培养的平静、冥想状态。茶道在朝廷中也变得非常流行，狮族和蝎族的朝臣学习了它，并将其带回他们的家乡省份。

樋田族花了几百年时间才接受它，但最终他们不再将其视为无脑朝臣炫耀陶器收藏的无聊活动，而是将其视为一种清除思想并为战斗做好准备的心理训练。

# 贪婪的时代（390IC-435IC）

# 菊花的黯淡

在第四世纪末期，鹤族与樋田族因谁有权控制他们共同边界上的某些地区而发生争执。冲突升级的原因是，安城家族出于自身原因，宣布放弃对鹤族的忠诚，转而宣誓效忠樋田族的冠军。一道天皇的诏书本可以轻易解决整个问题，但当时，虽然梵帝坐在皇位上，他们并未真正统治帝国。

三位武士——鹤族、凤凰族和蝎族的冠军——因密谋反对天皇而给自己的名字带来了永久的耻辱。他们自称御族，联合起来绑架了天皇的继承人，迫使天皇给予他们某些政治让步。天皇去世后，御族准备将那位继承人作为傀儡天皇扶上皇位。

# 帝国的复兴

虽然天皇的儿子们都被寄养在鹤族、凤凰族和蝎族的领主家中，被教导成为御族的顺从者，但他的幼女丰后姬却由狮族抚养长大。阿古渡氏族教会了她战争的艺术，并向她灌输了武士道和对她天皇祖先荣誉的崇敬。当天皇去世时，丰后姬向她的长兄发起决斗，杀死了他，并宣布自己继承皇位。御族对这一变故完全措手不及，新天皇迅速打破了他们对帝国政府的控制。太阳女神的光芒再次照耀在她的后代身上，巩固了他们统治的权利。

# 城市的崛起

在御族统治期间，御父居城经历了一段巨大的发展期。在此之前，它主要由皇宫和为各氏族的朝臣及其服务的平民所维护的各种房屋组成。御族官员安排改善了城市的港口，并扩展了通往城市的道路网络。结果，更多的武士、工匠和商人来到这座城市——甚至有一些来自远方的土地。

# 莲花的绽放（435IC-442IC）

# 兄弟会的方式

到了第五世纪，被称为新生兄弟会的僧侣组织在治愈和医学方面取得了显著进展。因此，平民和贵族都增加了对绿岩各地僧侣的支持。由于兄弟会资源的迅速增长和它所获得的尊重，它能够大大扩展其成员，并教导越来越多的平民关于《道》和觉悟之路的知识。尽管即使是最偏远的农民也知道关于新生的各种民间故事，但在帝国的许多地区，对新生《道》的扎实知识仍然缺乏。

为了尽可能广泛地传播正确的教义，兄弟会在帝国各城镇赞助了节日。这些节日以新生生平事件的戏剧化表演为特色，穿插着《道》的讲道和诵读。这些表演很快催生了一种被称为歌舞伎的戏剧形式。歌舞伎使用精致的服装和高戏剧性的情节，迅速成为富裕平民和低级武士的最爱。随着更熟练的作家对这种形式产生兴趣，它逐渐被高级武士所接受。

除了传播《道》，僧侣们还向任何表现出兴趣的人——无论是武士还是平民——教授茶道。艺妓也学习了这门艺术，以便与客户一起表演茶道。一些武士对仅仅商人表演茶道的想法感到震惊，但许多祭司认为这并无害处，甚至可能帮助他们净化与金钱打交道的影响。如今，艺妓和城市中富裕的工匠和商人是平民中最有可能进行茶道的人；很少发现农民对这类事情感兴趣。多年来，几位平民因他们的茶道技艺而闻名。在第七世纪后期，茶道大师莉莎——一位富裕染匠的女儿——有一位鹤族领主作为她的赞助人。

# 友谊之花

在第五世纪，外国人来到御父居城，请求与天皇丰后姬会面。这在朝廷中引起了很大的轰动。在此之前，帝国中唯一的外族人是来自西方的沙漠部落和来自象牙王国及更远地方的商船队。这些请求与梵帝会面的外族人来自帕瓦尔，一个位于太阳女神海对面的王国，他们希望与帝国进行贸易。天皇丰后姬以她的伟大慈悲，允许他们在御父居城停留两年，以便他们展示是否能够接受文明的行为。

这引发了极大的政治不安，因为各氏族的领袖和朝臣们争论如何应对这些陌生人。作为一个小氏族，螳螂族在这场辩论中尤为活跃，因为他们看到了增加自己财富和影响力的机会。许多人被来访者的奇怪习俗或他们对天界秩序的无知所排斥。

# 狮之大师

每个氏族都有自己的制度和传统，其中许多在几个世纪中几乎没有改变。然而，即使在绿岩的保守束缚中，非凡的个体也能够打破常规。

在狮氏族的领地上，一个农民的孩子出生时便拥有与元素之灵对话的天赋。他在年幼时被木津家族发现，并立即被该家族收养，取名木津太鼓。太鼓对火元素有着极强的亲和力，并在召唤和操纵这一元素方面取得了巨大进展。他的才华如此卓越，以至于当他从狮氏族退休时，凤凰氏族邀请他担任元素大师议会中的火之大师一职。他接受了这一职位，成为第一位非凤凰氏族（也是少数非伊泽家族）的议会成员。

# 内外的背叛（442IC-610IC）

在第六世纪，翡翠帝国的边界内发生了钢与血的冲突，尽管这些冲突从未能打破千年和平。事实上，这些事件几乎不能被称为战争，因为战争需要人类以荣誉面对他们的同胞。武士砍杀外族人和影地的腐化居民，与法官处决罪犯并无区别。

# 外族的背叛

经过两年的争议，天皇厌倦了外族人引起的骚动。以她的智慧，并作为她的权利，她突然命令外族人停止与绿岩人民的贸易，并立即离开她的土地。由于天皇的诏书是法律，外族人的恳求和抗议被无视。那时，这些忘恩负义的客人揭示了他们野蛮本性的深度，拿起武器对抗帝国。在随后的冲突中，天皇本人被杀。此后，对外族人不再有任何怜悯，任何在绿岩土地上被发现的外族人立即被处死。这场冲突被称为白鹿之战，以复仇武士发动反击的悬崖命名。那些设法逃到船上的人遭遇了鹤族和螳螂族的联合舰队。如果有人设法逃回他们的家园，他们带回了关于翡翠帝国战士勇气和技艺的故事。

# 血法师的崛起

或许由于外族人给绿岩带来的不平衡，一位邪恶的禁忌血魔法——魔法的实践者聚集了一群不洁的追随者，并对帝国进行了无荣誉的行为。血法师尤奇班在第六世纪初曝光，当时武士工匠朝比奈弥真将他锻造的剑献给了樋田族、鹤族、狮族和蝎族的冠军。不久之后，狮族冠军对龙族发动了一场灾难性的冬季战役，樋田族冠军谋杀了他的孩子，鹤族冠军在他的整个宫廷面前承认了一段恋情。三人用这些剑结束了自己的生命。只有蝎族冠军逃脱了这样的命运，很快揭示了铁匠的腐化以及血语者教派的存在。

这个教派由尤奇班领导，他的家族和氏族仍然未知。这位邪恶的法师召集了一支不死军队向御父居城进军——当时在帝国中，尸体被埋葬，而不是火化。多亏了蝎族冠军的警告，帝国军团和各氏族的军队击败了不死力量。

尤奇班被处决后，被埋葬在一个专门设下结界以防止他的灵魂逃脱的坟墓中。这使他成为帝国中最后一个被埋葬的人；此后，天皇颁布法令，所有尸体必须火化，以防止它们被邪恶魔法亵渎。

# 重燃的灯

随后的和平年代见证了能剧的复兴，它在歌舞伎面前开始衰落。剧作家垣田岩根创作了一系列关于众神生死的惊人戏剧，重新点燃了人们对这种形式的兴趣。对于许多武士来说，在白鹿之战和血语者对御父居城的袭击之后，能剧提供了一种与绿岩辉煌过去的联系，满足了人们对那些传奇日子的渴望。岩根的戏剧专注于演员通过最少的动作传达广泛情感的能力，并将其与更年轻、更炫目的对手区分开来。

随着武士们拥抱能剧的传统，歌舞伎在贵族阶层中的受欢迎程度下降，尽管它在平民中上升。城市平民几乎像崇拜他们扮演的神话角色一样崇拜流行的歌舞伎演员。最近开发的木版印刷技术在这一时期首次广泛使用，印刷品流传着描绘流行演员和他们最著名角色的图像。对于武士来说，这些大规模生产的图像是对真正艺术的扭曲，缺乏灵魂和身份。它们以如此庆祝的方式描绘贱民，只会增加这种侮辱。

# 严酷的教训

在533年，在一个高度乐观和战略逻辑可疑的时刻，蝎族对生驹城发动了攻击。攻击失败后，作为报复，松族攻击了蝎族的荣誉城。在夺取堡垒后，他们杀死了其中的每一个蝎族，并为了强调这一点，将其更名为谦逊的教训。

# 天界的审判

这个世纪的最后几年见证了天皇十六世的加冕、统治和死亡，他是唯一失去天命的天皇。尽管他统治的头几年很有希望，但这位被后人称为钢菊的天皇陷入了偏执的暴力之中。看到到处都是背叛，天皇处决了数千名武士，罪名是莫须有的罪行。朝臣、修验者和僧侣们恳求他展示仁慈，却遭遇了更多的流血事件。最终，在天皇十六世命令他的母亲在朝廷面前被勒死后，他的儿子率领帝国卫队对他发动了叛乱。这次成功清楚地表明，天照大神已经收回了对他的青睐。

在天皇十六世的儿子对父亲举起手后，他剃了头，退隐到新生兄弟会，而参与此事的每一位帝国卫队成员都切腹自尽。钢菊的最年轻的弟弟成为天皇十七世，他长久而和平的统治证明梵帝家族仍然拥有天命。

# 光明与黑暗的潮汐（610IC-815IC）

经过几个世纪未被影地生物侵扰的时光，绿岩的大多数人几乎忘记了这一威胁。对于樋田领地之外的大多数人来说，影地是绿岩起源的一部分，具有历史重要性，但比每天听到他们祈祷的众神更远离他们时代的帝国。许多人认为，第八世纪影地和血法师的重新出现是天界的惩罚——尽管是对钢菊的罪行还是背叛他的卫队的亵渎，意见不一。

# 艺术的进步

在天皇十七世统治的末期，一位名叫阿贺娑冰太郎的龙族炼金术士开发了多种新的陶瓷釉料，具有前所未有的色彩深度。与此同时，堺雨内善开发了一种新型窑炉烟道，使陶艺家对烧制过程有了更多的控制。这两项进步相结合，创造了各种新的陶瓷装饰效果。茶道大师们对新风格在茶道中的适用性产生了尖锐的分歧，并为此进行了多次决斗。

# 南方的黑暗

在第八世纪，影地的邪恶两次侵入了帝国。在后来被称为潮汐陆桥之战的战斗中，一次大规模的影地攻击分散了樋田族的主力，而由鬼金次郎领导的第二支部队则袭击了地震鱼湾附近的瞭望塔。瞭望塔的守卫几乎被压倒，直到大道寺领主大道寺正重和他的私人卫队作为援军抵达。他们在低潮时穿过海湾，最终将影地部队逼回海湾，涨潮时海水吞没了他们。

没有一位鹤族在这场战斗中幸存，当正重的破损头盔被冲上岸时，樋田族建造了一座神社来安置它。樋田族的幸存者开始称大道寺为“铁鹤”，这一绰号最终传遍了整个帝国。

正如后来所显现的，在潮汐陆桥之战中被击败的入侵者只是即将到来的威胁的前兆。第二年，被称为“巨口”的鬼率领的部队横扫樋田领地，将氏族的战线推得如此之北，以至于昼马家族的祖传堡垒日光城完全失守。最终，昼马和樋田的联合部队，在国净化者和猎巫者的增援下，阻止了邪恶的推进，堺雨建造了今天我们所知的木匠墙。昼马领地从未被收复，标志着自帝国建立以来首次领土的丧失。

# 血法师的回归

如果不是750年的事件，影地的入侵可能会被铭记为本世纪最大的不幸（即使在樋田领地之外）。当时，法师尤奇班的灵魂逃脱了他的坟墓，占据了一个身体作为自己的容器。他召集了一支由魔法教派信徒和复活动物组成的军队，再次攻击了帝国。大氏族的军队在沉睡河之战中阻止了尤奇班部队的推进，龙族富坚派的一位神秘僧侣成功地将尤奇班的灵魂困在了他自己的体内，时间足够将两者都封印在一个坟墓中。

事后，法官们发现，在尤奇班首次被击败后的几年里，许多村民、村长甚至祭司，主要是在小型或偏远的定居点，违抗了天皇关于完整埋葬尸体的法令。由于将他们的传统置于梵帝的意志之上，从而危及了帝国，这些人面临着严厉的惩罚。

# 新风吹拂（815IC-1123IC）

麒麟族的回归，带来了翡翠帝国内部的巨大变化和社会动荡。这是自帝国建立以来，大氏族之间首次几乎爆发真正的战争。外族商品和人员以白鹿之战以来未见之规模进入翡翠帝国。一些学者怀疑这些巨大变化是否是修验者所观察到的当前元素失衡的原因，并在1120年的灾难性地震和海啸中得到了体现。

# 马与武士的氏族

在815年，新条的孩子们以一种极具戏剧性的方式结束了他们的流浪。由于一系列涉及外族魔法的奇特情况，他们发现自己位于影地的北部边缘。为了迅速逃离这片被污染的土地，他们无视外交礼仪，直接利用他们强大的骑兵冲破了樋田族的防线。帝国陷入了骚动，因为起初人们并没有认出麒麟族是新条的氏族。在他们的旅途中，他们将名字改为独角兽，并采用了奇异的服饰和习俗。他们的语言也从纯正的绿岩语退化，而帝国内的语言在几个世纪中已经发展。然而，他们的马匹令人惊叹：高大、健壮且迅捷。当蝎族军队攻击他们时，独角兽骑兵采用了帝国前所未见的战术，粉碎了对手。

当独角兽部落缓慢穿越世界脊山脉时，狮族动员了他们的军队。然而，由于冬雪阻止了即将发生的军队冲突，独角兽的代表得以与鹤族接触。他们携带的珍宝之一是一把古老的扇子，据说是新条的。鹤族宣布这是道寺大人送给她姐姐的扇子，入侵者实际上是新条的氏族。天皇接受了这一点，并禁止狮族攻击他们。他还将麒麟族的旧地赐予独角兽族定居。这在狮族中引起了一些敌意，因为他们曾被授予麒麟领地的管理权，这代表着肥沃农田的严重损失。然而，他们服从天皇的命令，允许独角兽族无阻碍地前往他们的土地。

尽管有鹤族的证言，起初，许多绿岩人将独角兽族视为几乎等同于外族人。他们有奇怪的名字、味道不佳的食物、粗鲁的举止和花哨的时尚。更糟糕的是，在他们的旅途中，他们获得了一个新的家族——元族，他们是西方荒原乌吉克部落的亲属。一些朝臣建议按照禁止外族人的法律驱逐元族，但独角兽族冠军代表他们向天皇上诉。他说，元族的祖先是由新条本人命令收养入族的，这使他们成为绿岩人。天皇同意，众神的权威不可否认，元族得以留下。

鹤族承担了使独角兽族更加文明的任务，结果好坏参半。新条的孩子们热情地接受了戏剧（尤其是能剧）、茶道、当前的《道》和元素教义、绿岩美食和香道。他们拒绝放弃他们的外族名字、自己的食物、服装风格或握手问候的习俗。当他们在领地外旅行时，他们的朝臣和使者开始表现得像真正的绿岩人，但即使在今天，访问独角兽领地也像是进入了一个异国他乡。

尽管如此，独角兽族进入了帝国的生活。虽然起初他们的政治影响力几乎为零，但他们对经济产生了强大的影响。他们带来了新的金属加工、皮革加工和织物染色技术。除了绿岩前所未见的快速、强壮的马匹外，独角兽族还带来了马镫和先进的骑术。很快，独角兽的马匹在整个帝国中需求旺盛，尽管该族只出售阉马和偶尔的母马，这确保了他们对供应的控制。他们的旅行使他们与帝国外的国家建立了联系，他们进一步发展这些联系以建立有利可图的贸易路线。不久，象牙、稀有木材、宝石以及昂贵的草药和香料通过独角兽族的手流入帝国。

也许最重要的是，独角兽族带来了名称道的实践。使用这种名称魔法，独角兽族的修验者将灵魂束缚在护身符中，实现了其他氏族通过向神明献祭和祈求所获得的效果。对于许多修验者和其他祭司来说，名称道是一种亵渎的外族实践。

# 睡眠与梦之花

其中一种昂贵的草药是罂粟，鸦片的来源。独角兽族已经使用它几个世纪来缓解疼痛，帝国的治疗师们也采用了它。然而，随着它在医学中的使用传播，一些人开始将其作为娱乐吸食。很快，人们注意到以这种方式使用鸦片的武士对领主的忠诚度下降，很快就有反对它的呼声。余吴家族的草药师发现罂粟在绿冈尾张城周围的土地上生长得非常好，因此蝎族冠军向天皇请愿，要求将罂粟的种植和使用纳入帝国法规。天皇批准了这一请求，并授予蝎族独家种植和制造鸦片的权利。

在接下来的十年里，谎言城的规模翻了四倍，并继续增长，因为鸦片贸易为这座城市带来了无尽的商人和金钱。虽然蝎族对医用鸦片的控制为他们带来了巨大的财富，但一些法官抱怨说，这并没有阻止鸦片的滥用。木津纪家族的法官一度指出，城市周围用于种植罂粟的土地远远超过了合法鸦片供应所需的数量。该省的总督解释说，额外的种植是为了确保高质量供应，低质量材料被销毁，没有人对此提出质疑。直到今天，绿冈尾张周围的土地上仍然布满了大片的罂粟田。

# 城市的崛起

第十和第十一世纪见证了自帝国早期以来城市地区的最大扩张。许多因素促成了这一增长，包括独角兽族带来的许多创新。马匹的重新流行增加了各省和城市之间的陆路旅行，这让负责维护帝国道路的领主们感到非常不安。

农业技术和工具的许多进步提高了作物产量，使得村庄能够满足税收评估员的要求，即使越来越多的年轻人离开村庄前往城市。独角兽族带来的马具和轭具使牛和其他牲畜变得更加有用，而手动泵的改进使得田地灌溉更加高效。一些农民开始在邻近的城镇和城市交易自己的商品，成为商人。

对于一些传统的武士来说，农民商人的向上流动仍然是对天界秩序的冒犯，然而，甚至谈论这些事情通常被认为是不合适的。狮族的武士强烈反对这些变化，严厉惩罚那些试图放弃祖传村庄去城市寻找新生活的农民。交易自己农产品的农民被指控偷窃领主的财产，并受到相应的对待。

在帝国与外族隔离四个世纪后，独角兽族的汗布拉克城的重要性表明，外来影响在绿岩仍然在上升。受到独角兽族贸易成功的鼓舞，螳螂族增加了与绿岩大陆的外族商品贸易。独角兽族正式回归帝国的接受在某种程度上被视为对他们带来的外族习俗和血统的宽容。结果，外族商品、习俗甚至旅行者在帝国的贸易港口甚至内陆城市中逐渐获得了更大的接受——或至少是更弱的抵制。曾经的小型沿海社区，如清水村和孤岸城，在这一时期得到了巨大的发展。

# 完美之地的召唤

在第九世纪末，七雷神社的一位年轻僧侣提出了引发完美之地教派的有争议的教义。悠季相信，新生与梵帝之间的对话开启了一个持续八百年的天界美德时代——每个听到新生教诲的众神各占一个世纪——而第九世纪开始了美德衰退的时代，以腐败和难以遵循《道》为标志。为了祈求新生回归，这位僧侣不断念诵咒语“Shoshi ni kie”（“对新生的奉献”或“对新生的绝对信任”）。悠季开始相信，如果有足够的人真诚地念诵这个咒语，新生将会回归，开启一个新的天界美德时代。

悠季的学生雅太在悠季去世后创立了完美之地教派，基于她在临终前写下的一部经文。这部卷轴声称，新生在离开人界时并没有回归虚空，而是居住在天界中的一个完美之地。完美之地教派相信，那些念诵“kie”（悠季的咒语）的人可以在死后加入新生的完美之地，而不是在冥界接受审判并根据业力重生。在完美之地，在新生本人的指导下，信徒可以在不经历轮回的痛苦下实现觉悟。

对于新生兄弟会的许多人来说，这些信仰是异端，违背了新生自己的《道》和天界秩序。完美之地教派在农民中迅速获得了流行，因为它提供了以前未知的东西：摆脱今世和来世考验的自由。许多平民相信，美德衰退的时代指的是武士阶层的腐败。这种信仰导致许多武士进一步谴责该教派。到了第十世纪中叶，完美之地教派在凤凰领地被取缔，许多信徒前往龙族山区寻求安全和隔离。

# 新的荣耀

独角兽族的到来也对帝国的艺术产生了深远的影响，为其带来了新的技术和材料。从那时起，一些绿岩人谴责了“外族污染”的影响，一些工匠在融合新技术时过于热情，但理性的批评者必须承认，整体效果是积极的。

事实上，到了第八世纪，帝国的大多数艺术家已经停止复制经典作品，开始互相模仿，结果令人遗憾。那个时代只有《盐与太阳的日子》，决斗者木津纪本藏记录的武士旅程，显示出真正的价值。本藏的笔法在几个技术领域有所欠缺，但他的构图感无可挑剔，他有一种捕捉运动中的主题的天赋。他讽刺的幽默与艺术完美匹配。一个才华稍逊的人可能会将两个农民、酒壶和鳟鱼的故事变成闹剧，但在本藏的手中，这是一段细腻而闪耀的幽默。只有垣田绫野的诗集，充满了对逝去季节的渴望和敏感描述，在质量上接近。没有第三个。

这一切在第九世纪发生了变化。无论艺术家选择接受独角兽族带来的新思想还是拒绝它们，他们都被迫以全新的视角看待世界。垣田梅为藤御殿创作的屏风画将一些新的透视技术融入古典主题，展示了新事物不必摧毁旧事物。许多画家追随她的脚步。柴寒子的《暮色中的梅花》和道寺素子的《红钢》激发了人们对小说的新兴趣，最近出版的垣田亮子的杰作《冬天》显示，艺术的质量尚未衰退。同时，间由纪子和垣田麻子开始了他们漫长的诗歌比赛，为这一形式带来了细腻的洞察力和尖锐的机智。一个卷轴可以填满更多的例子。

# 辉煌承诺的展开

这便是我们帝国的第一个千年，一个充满荣誉与勇气的辉煌土地。我们已进入太阳女神眷顾梵帝的时代，我们的土地沐浴在天皇统治下的和平、繁荣与正义之中。
