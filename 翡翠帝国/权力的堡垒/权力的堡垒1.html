# 权力的堡垒1

武士勒住马缰，仰起头，眯起眼睛抵挡着上午晚些时候强烈的阳光。阳光将城堡的灰泥墙壁照得雪白。城门敞开，迎接前来参加道寺·佐藤大人宫廷的宾客。

此时，阅兵场空无一人，午后的微风从坚硬的地面上卷起一层白垩色的尘土。更多的守卫站在那里，当他走近时，他们纷纷鞠躬。其中一名守卫在他下马时牵住了马缰。“佐藤大人正在宫廷内等候您，大道寺大人。”

阳光透过高层的窗户射入宫廷大厅，细小的尘埃在光线中漂浮，悬浮在朝臣们的头顶上方。十几把扇子轻轻摇动，展示着五个不同家族的颜色和家纹，宾客们打量着这位新来者。宫廷的气息——香水与丝绸的味道——弥漫在空气中。

终于，他走到了大厅尽头的台阶前，微微迟疑了一下，随即在领主面前俯身行礼。聚集的朝臣们交换了心照不宣的点头：这个人离开宫廷文明太久了。

绿岩是一个由武士阶级统治的帝国，从最早的历史开始，这些武士就建造城堡来保护和控制他们的领地。尽管这些建筑首先是军事要塞，但它们也作为政府、行政、外交和文明的中心。虽然一些武士在乡村管理小片领地，但被邀请居住在城堡城镇，甚至更好的是在城堡内，几乎总是被视为一种晋升。武士与他们的领主越接近，他们的影响力——以及他们被认可的地位——就越大。

# 领主与治理

在绿岩，从天皇到最底层的地区大名，封建治理的核心都是城堡。城堡既是领主权力的象征，也是他们统治领地的行政中心。税收被运送到城堡进行征收，士兵和治安官从城堡出发巡逻以维护秩序，而希望向领主请愿的平民也必须前往当地的城堡。

每个大氏族的领地被划分为若干个省份，每个省份都有一位大名，其权力中心通常位于该省最坚固的城堡中。氏族冠军和各家族领袖可能居住在这些最显赫的城堡中，也可能根据传统拥有自己的独立城堡；这些最高级别的领主名义上直接统治一个省份，但通常他们会将这一任务委托给下属，例如执事或旗本。

大氏族领地内的主要城市有自己的总督。这些总督通常也居住在城堡中，尽管有时他们可能住在城市内的非防御性住所中。无论如何，城市总督和省份总督都对其领地拥有统治权，负责维护法律、维持秩序并征收税收。

这一体系并未止步于总督。在他们之下是被称为守护的低级领主，他们以领主的名义统治更小的领地。这些地区领主是能够自称大名的最低级别武士。他们往往野心勃勃且难以驾驭，容易与邻居发生争执——尤其是当这些邻居恰好位于家族或氏族领土边界对面时。自然，他们每个人都居住在城堡中，因为不这样做将意味着失去面子。因此，绿岩封建制度的性质确保了数百座城堡遍布帝国。

封建制度进一步延伸到服务于地区领主的本地庄园领主。这些武士统治着非常小的区域，通常是几个村庄，甚至可能只有一个。这是一项卑微的职责，但这仍然使他们能够将自己视为地主阶级的一部分，并从他们的土地上征收税收。

统治一个偏远村庄的庄园领主在社会等级上高于守卫边境或在氏族军队中作战的武士。然而，这些武士的地位太低，无法拥有城堡，而是居住在各种防御级别的庄园住宅中。

# 绿岩的城堡

自从帝国的最早时期起，绿岩人就开始建造城堡。如果某些古代文献可信的话，伊泽部落在众神从天而降之前就已经这样做了。与帝国文化和文明的许多方面一样，绿岩的城堡设计和建造也受到可以追溯到最早时期的传统所支配。它的方法和风格始于梵帝一世建立帝都御父居城之时，因此至今仍被认为是唯一“正确”的建造方式。当时，梵帝和其他众神为构成绿岩的土地带来了建筑和工程学的新理解。他们教导他们的人民建造类似于天界宫殿的城堡。这些设计元素——覆盖墙壁和屋顶的倾斜瓦片、塔楼和主楼垂直的宝塔式结构、光滑且略微倾斜的外墙——几乎出现在绿岩的所有城堡中。这种模仿习俗还有其他方面。例如，由于梵帝宫殿的主楼有十层高，因此其他绿岩城堡的主楼不得超过九层，以免其大名被指责将自己置于与天皇同等的地位。

在一个外表与现实同样重要的社会中，城堡扮演着双重角色：它们既是军事力量、文化活动和行政管理的中心，也是每个氏族权力、财富和理想的象征。虽然所有城堡都遵循某些基本的设计原则，但每个氏族都会修改这些原则以反映其需求、价值观和美学。蟹氏族的城堡极其实用，狮氏族的城堡朴素而传统，鹤氏族的城堡轻盈而美丽，蝎氏族的城堡充满了隐秘的通道，等等。此外，特定的家族和个别领主根据他们自己的职责和价值观建造城堡，这些可能与氏族的整体风格有所不同。垣田家族的城堡与大道寺家族的城堡不同，尽管两者都属于鹤氏族。

独角兽氏族的城堡，与该氏族的许多其他事务一样，与绿岩的传统设计有显著的不同。独角兽氏族在帝国缺席了八个世纪，在漫长的旅程中受到了许多不同的外来影响。因此，他们的建筑将那些外来概念与帝国最早时期的绿岩设计相结合，并且直到最近才开始逐渐采用现代帝国标准的美学。远行者城堡是一座巨大的三面主楼，而战女城堡则在多座塔楼上加上了钟形圆顶，元家族在汗布拉克的宫殿甚至不是一座坚固的建筑，而是一系列悬挂在巨大柱子之间的华丽挂毯。唯一符合帝国期望的主要独角兽要塞是大日城堡，该氏族建造它的具体目标是与帝国的其他部分建立良好的外交关系。

# 领主的职责

任何领主的主要职责包括维持秩序、保护其领地和资产免受外部入侵以及内部威胁（如土匪、海盗和叛乱）的侵害，以及征收税收。为此，每位领主都维持一支地侍（jizamurai）力量，负责驻守领地、巡逻道路和边境，并保护领主的城堡。领主越富有、权力越大，能够维持的地侍数量就越多，其领地的安全也越有保障。领主还会任命低级别官员，如税务评估员、治安官和地方土地管理者，以维护法律并管理领地。在战争时期，领主负责征召和训练足轻（ashigaru），即农民士兵，以参战。

如果某个地区防御薄弱或治安混乱，领主被视为应对此失败负责，并必须迅速纠正。未能做到这一点将受到惩罚，可能仅仅是公开羞辱的斥责，但也可能是降级甚至切腹自尽的风险。

领主对其封臣的福祉和命运负有责任；然而，不同氏族和家族对此的重视程度各不相同。一些领主会竭尽全力确保其封臣生活优渥，甚至亲自确保他们获得美满的婚姻，或在每个重要场合赠送礼物；而在另一个极端，有些领主认为他们唯一的责任就是支付封臣的月俸。大多数统治者则介于两者之间。

由于领主的城堡既是其领地中最具防御性的地方，也是行政管理的中心，因此它是进行最重要的治理活动的首选地点。每年秋天，税收被运送到城堡，经过清点并妥善保管，直到将适当的份额上交给上级领主和天皇。被抓获的土匪和罪犯通常被带到城堡处决，如果他们在别处被杀，他们的头颅可能会被展示在城堡外以警示他人。最重要的是，城堡既是外交活动的场所，也是战争计划的制定地。

领主还有款待宾客的职责。领主应为任何作为客人访问城堡的武士提供安全且文明的住所，无论他们是低级别的过路者还是高级贵族。不同氏族对款待的标准有不同的看法：蟹氏族或狮氏族的领主认为，只要客人有食物可吃、有房间可睡，他们的职责就已尽到；而鹤氏族的领主则确保客人拥有布置精美的房间、精致的餐食、艺术和音乐娱乐，以及愉快的交谈来消磨时间。各氏族对于领主是否应将城堡打造成艺术和文化中心也存在分歧。再次，那些更具军事性和禁欲性的氏族倾向于最小化或完全拒绝此类事务，而其他氏族则将其视为与其余职责同等重要的责任。

所有领地的领主都负责协调和主持全年举行的各种宗教节日和庆典。这需要与当地寺庙及其氏族的修验者（shugenja）合作，确保仪式得以正确举行，且没有任何不祥之事破坏这些活动。绿岩人是一个虔诚的民族，平民尤其迷信。未能正确敬奉神灵和福神（Fortunes）的领主，或在其统治期间出现不祥预兆的领主，将很快面临日益增长的不满情绪。

领主主要职责的最后一个方面是维持平民（heimin）和贱民（hinin）之间的良好秩序，确保这些卑微的农民完成简单但至关重要的任务，以保持帝国的平稳运行。允许下层阶级不敬或公然违法，或在危险时期冷酷地未能保护他们的领主，是失职的，并可能面临农民起义的羞辱。在这种情况下，领主的城堡将成为一座监狱，统治者及其封臣被平民的愤怒所困。

# 宫廷旗帜

即使没有任何其他装饰，城堡的宫廷大厅也总会在高台之上的墙壁上展示领主的家族和氏族旗帜；如果领主有个人的家纹，也可能一并展示。当有身份的客人来到城堡时，他们的旗帜会被悬挂在宫廷中，与领主的旗帜正对，以示尊重。性情乖戾或意图挑起战争的领主有时会以微妙的方式展示这些旗帜，以侮辱客人，例如将客人的旗帜悬挂得略低于领主的旗帜，或将它们悬挂在侧墙而非正对的位置。

当城堡举办帝国冬之宫廷时，一套复杂的礼仪体系规定了家族和氏族旗帜的悬挂方式。传统上，帝国旗帜——印有梵帝在帝国初年选择的菊花标志——是最大的旗帜，直接悬挂在高台上方，左右两侧分别是濑分和尾朝的旗帜。大氏族的旗帜悬挂在左右墙壁上，其悬挂顺序揭示了哪些氏族受到天皇的青睐。（传统上，作为天皇左膀右臂的鹤氏族和狮氏族的旗帜最靠近帝国旗帜，但如果梵帝对它们确实不满，这一点可能会改变。）龙氏族的旗帜通常直接悬挂在天皇旗帜的对面，反映了富坚拒绝参加古代神之锦标赛的态度。

# 宫廷及其居民

任何不仅仅是军事据点的城堡都包含一个宫廷大厅，即使它非常小，更多地象征着理想而非现实。在最基本的层面上，宫廷是领主召集当地武士进行公开讨论的地方，包括愉快的交谈和严肃的外交。与绿岩中的一切一样，长期以来的惯例通常决定了宫廷大厅的外观及其所扮演的角色。

传统上，宫廷大厅是一个两层的宽敞房间，二楼有一个环绕的阳台。主楼层大多空置，唯一的显著特征是一端的高台，与主入口相对，主持的领主或其代表坐在那里。上层通常在其周围放置几张桌子。朝臣和其他外交官在两层之间穿梭，形成对话群体，同时观察彼此。艺术表演和正式展示也在宫廷中举行，通常是在领主的高台前。

理想的宫廷大厅应足够大，至少可容纳两百名武士在主楼层聚集参加重大活动。然而，只有少数城堡能够拥有如此宽敞的宫廷。更常见的是，即使上下两层加起来也无法容纳这么多人，领主会提供额外的房间或将大型活动移至室外，如花园或阅兵场。

宫廷大厅全天对访客开放，但其正式角色传统上发生在早晨，当领主主持房间，外交官们聚集进行正式讨论时。客人可以请求领主允许向整个宫廷发表讲话，尤其是宣布重要事件，如婚姻和正式联盟，或宣战。这些活动通常在高台前进行，发言者向主持的领主致辞，同时期望被所有人听到。

艺术表演等特别活动可以在白天或晚上的任何时间举行。在正式宫廷时间之外，它们更可能安排在城堡的其他地点，如花园、餐厅、道场或私人房间。当宫廷不举行会议且大厅需要用于其他用途时，障子（shōji）屏风可以将空间分隔以供其他用途。在战争时期，这可能用于为有声望的武士提供额外的营房，甚至让房间作为伤员的医务室。

不同城堡的宫廷展示的艺术和娱乐活动各不相同。一些领主安排音乐家在阳台上演奏。其他领主则在房间周围放置盆栽、障子屏风上的绘画或花道（ikebana）插花，以提供话题。小丑可能会在场，以幽默缓解紧张气氛，或巧妙地刺激来自敌对领主宫廷的访客。富有（或野心勃勃）的领主可能会安排更精致的艺术展示，如室内锦鲤池或戏剧表演。相反，贫穷或性情禁欲的领主可能会故意减少宫廷大厅的装饰，或许偶尔会提到新生（Shinsei）关于世俗分心危险的教诲。

# 绿岩的宫廷小丑

在绿岩，小丑是一种非常特殊的艺人，因为他们的艺术本质上是在宫廷中进行的表演。为了从事这一职业，小丑们学习并掌握多种艺术形式，包括诗歌、舞蹈、歌唱和歌舞伎表演。他们还研究政治和人性，因为他们在宫廷中的角色是戳穿周围的虚伪和自大。他们是绿岩社会中“无视闹剧”和“以沉默杀人”原则的特例：他们公开指出那些其他人必须默默忍受的事情。因此，他们的功能与狮氏族的生驹吟游诗人（Ikoma Bards）相似：表达那些社会不允许他人揭示的人类情感。

与绿岩文化的许多方面一样，小丑的概念似乎是由鹤氏族首创的，可能是在第二或第三世纪。然而，武士并不需要进入特定的学校接受训练才能成为小丑：他们只需要展示出从事这一工作的天赋，并获得领主的任命。（试图在没有领主任命及其隐含保护的情况下扮演小丑，很可能会以耻辱和死亡告终。）

绿岩的小丑并不欢乐或疯癫。他们是讽刺的，常常尖酸刻薄，他们的幽默尖锐而严厉。他们用双关语、谜语和隐晦的语言说话，揭露他人的不光彩行为，并讽刺宫廷世界的礼貌惯例。只要他们依靠这种嘲讽，他们就可以公开指责他人的不良行为，而不会面临决斗的风险。当然，这种社会保护也有其限度——一个严厉的狮氏族成员或脾气暴躁的蟹氏族成员不太可能永远忍受小丑的挑衅——而明智的小丑知道何时该退让，不再纠缠他们的目标。

# 宫廷的隐形面：仆人

每座城堡都是一个几乎不被武士所见的完整社区的家园：仆人。他们从黎明前劳作到深夜，以确保城堡的生活保持清洁、有序和可靠。仆人甚至通过特殊的门进出城堡，这些门几乎不被所有武士注意到。事实上，在某些方面，他们比他们的社会上层拥有更多的自由。武士在城堡中的行动受到限制：只有最高级别的人，如领主及其家人，才能随意走动。而仆人则无处不在，日夜穿梭，几乎不被主人察觉……他们看到并听到了一切。武士擅长彼此保守秘密，即使在城堡的狭小空间内，但没有人能对仆人保守秘密。

讽刺的是，正是这种社会盲点使武士忽视了仆人，也使他们很少利用仆人对城堡事务的深入了解。毕竟，除了下达命令之外，与仆人交谈对武士来说是贬低身份的。然而，少数愿意降低身份的武士通过这种方式获得了对竞争对手的显著优势。贿赂、威胁和勒索，或简单的魅力，都可以从仆人那里获取有用的信息。同样，仆人的入口为那些希望避免注意的人提供了潜在的进入点，因为很少有人会想到武士会以这种方式进入城堡。不出所料，蝎氏族最擅长这种隐秘的手段。

# 宫廷中的重要人物

任何宫廷都被一群被称为“朝臣”的武士所主导，这些武士经过专门训练，专注于城堡安全围墙内的独特世界。朝臣的数量相对较少，主要集中在政治、外交和阴谋的封闭环境中。在他们的行列中，有许多专门的角色。以下是其中一些较为常见或重要的角色，但朝臣的队伍中还包括大量专门维护帝国权力机器的官僚。

# 大使

尽管大多数氏族都试图在其他氏族的宫廷中保持外交存在，但大使这一特定职位还承载着额外的声望和责任。大使被授权代表其领主发言，并达成对领主具有约束力的协议，仿佛领主本人在场一样。大使职位既显赫又要求极高，因为这样的武士被赋予了氏族的未来和声誉。如果大使失败、丢脸或表现出软弱，他们将为领主和整个氏族带来损害。这是朝臣使用隐晦、间接语言的主要原因之一。在武士社会中，直言不讳不仅被视为粗鲁，而且说话过于明确的大使可能会不明智地作出承诺。武士的言辞即是他们的承诺，因此大使的明确声明可能像书面条约一样具有约束力。模棱两可或间接的声明则为灵活应对留有余地，并在必要时避免失礼或丢脸。

# 执事（家老）

执事，有时被称为家老（karō），是被指派为领主担任高级助手的武士。他们可以作为顾问，但他们最重要的角色是管理领主的事务，监督城堡、保存记录、向仆人主管传达命令，并处理类似事务。此外，当领主不在时，家老需要担任领主的代理人，管理城堡和领地，主持宫廷，并以领主的名义发布命令。被任命为家老是一项重大的荣誉和信任的标志；通常，这一职位会代代相传。在可能的情况下，这一职位会委托给旗本（hatamoto），以确保最大程度的忠诚。

# 私人封臣（旗本）

旗本是高级领主（如家族大名或氏族冠军）的私人封臣。只有如此高贵的领主才能任命旗本，他们的忠诚完全且直接归属于该领主，不受任何干扰。在社会地位上，氏族冠军的旗本被认为高于省级领主，使他们能够作为解决问题的专家被派往问题或不忠诚省份的城堡，并执行他们领主的意志。

# 工匠

许多武士从事艺术追求，这些文明活动使他们与平民的粗俗世界区分开来。然而，只有少数武士专门接受艺术训练，而不是传统的武士生活核心——军事训练。工匠和艺术家主要在宫廷世界中活动，因为宫廷为展示他们的作品和寻找赞助人提供了最佳机会。新艺术作品的首次亮相——戏剧、绘画、雕塑等——是任何宫廷的亮点。大多数努力在其要塞中维持文明宫廷的领主都会尝试在城堡中至少保留一两名工匠，并努力吸引其他地方的著名工匠来访并展示他们的作品。

# 媒人（仲人）

在绿岩，婚姻几乎总是由他人安排，武士家族传统上依赖被称为仲人（nakōdo）的媒人来完成这些安排。熟练的仲人服务需求很大，尤其是因为婚姻通常是外交协议的一部分。任何有地位的领主都会尝试在其宫廷中拥有一名才华横溢的仲人，而这些媒人的服务本身也是一种有用的外交筹码。当仲人不履行其媒人职责时，他们会执行许多与朝臣相同的任务。

# 导师（先生）

城堡是武士为其领主服务的训练中心。因此，每座城堡至少包含一个道场，而大型城堡则拥有多个道场。最负盛名的学校和最受尊敬的先生（sensei）几乎总是位于其氏族的主要城堡内，使这些城堡成为学习和军事权力的中心。即使是小型城堡也至少有一个小型道场和一名常驻先生，以训练其武士。先生通常还担任领主的顾问，尤其是如果他们曾亲自教导过领主。他们的意见总是受到重视，因为他们负责训练氏族的武士并传承其学校的古老秘密。
